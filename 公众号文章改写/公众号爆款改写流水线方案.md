## 7步标准流程

### Step 1：搜索文章

**工具：**
- `wechat-search-redfox` — 10w+爆款榜单，按评分排序
- `wechat-article-search` — 关键词搜索，返回真实URL

**执行命令：**
```bash
# 场景一：关键词搜索
cd /root/.openclaw/workspace/skills/wechat-article-search/scripts
node search_wechat.js "退休老人 健康" -n 5 -r

# 或场景二：爆款榜单
cd /root/.openclaw/workspace/skills/wechat-search-redfox/scripts
REDFOX_API_KEY="***" python3 fetch_gzh_trends.py --keyword "养老" --start-date 2026-05-27

# 备用：gzh-10w-hot
cd /root/.openclaw/workspace/skills/gzh-10w-hot/scripts
python3 fetch_hot_articles.py --category 健康养生 --start_date yesterday --end_date yesterday
```

**产出：** 文章标题 + URL + 摘要 + 阅读数（场景二）

---

### Step 2：抓取正文

**工具：** `wechat-article-spider`

**执行命令：**
```bash
cd /root/.openclaw/workspace/skills/wechat-article-spider/scripts
python3 main.py "<文章URL>" /root/.openclaw/workspace/改写素材/
```

**产出：**
- 正文章节文件（.md）
- 图片目录（`images/`）

**注意：**
- 云服务器IP访问微信图片会被拦截，返回0张图片
- 如需图片，跳到Step 4从公开图库下载

---

### Step 3：改写标题+正文+摘要,并去AI味

**执行者：** 阿呆直接执行

**改写要求：**
- 标题：吸引目标读者（50岁退休老人）
- 正文：换角度/换结构/换表达，保持核心内容
- 摘要：100字内，突出文章核心价值
- 去AI味：输出内容流畅自然，无AI典型句式

**产出：** 改写后的标题 + 正文 + 摘要

---

### Step 4：搜索配图并下载

**工具：** 网络图片搜索 + curl/wget 下载

**图片来源：**
- Unsplash：`https://images.unsplash.com/photo-xxx?w=600`
- Pexels：`https://images.pexels.com/photos/xxx?auto=compress&cs=tinysrgb&w=600`

**执行命令：**
```bash
mkdir -p /root/.openclaw/workspace/图片素材

# 通过代理下载（云服务器需代理）
proxychains4 curl -s --max-time 15 -L "图片URL" -o /root/.openclaw/workspace/图片素材/图片名.jpg

# 或直接下载
curl -s --max-time 15 -L "图片URL" -o /root/.openclaw/workspace/图片素材/图片名.jpg
```

**图片要求：**
- 主题契合文章内容
- 4-6张（文章开头 + 段落配图）
- 优先选 Unsplash/Pexels 免费图

**图片识别：** 可用 `image` 工具分析图片内容确认是否适合

---

### Step 5：标题+正文+配图+摘要上传飞书

**工具：** `lark-cli`

**执行步骤：**

#### 5.1 创建飞书文档（含摘要）

```bash
cd /root/.openclaw/workspace

lark-cli docs +create \
  --api-version v2 \
  --title "【改写】文章标题" \
  --content "# 文章标题

## 摘要
文章摘要内容...

## 正文
文章正文内容..." \
  --doc-format markdown
```

#### 5.2 上传图片到云空间

```bash
cd /root/.openclaw/workspace/图片素材

lark-cli drive +upload --file 图片1.jpg
lark-cli drive +upload --file 图片2.jpg
# ... 每次间隔2-3秒
```

#### 5.3 在文档中插入图片

```bash
cd /root/.openclaw/workspace

lark-cli docs +media-insert \
  --doc <文档ID> \
  --file "图片1.jpg" \
  --selection-with-ellipsis "[图片1]"
```

**注意：**
- 图片需在 `/root/.openclaw/workspace/` 目录下
- 文档中用 `[图片1]` 等占位符标记位置
- `--api-version v2` 必须加，v1 已废弃

---

### Step 6：用户审核

**用户动作：**
- 打开飞书文档链接
- 阅读改写内容
- 审核图片是否合适
- 确认是否需要调整

---

### Step 7：调整优化

**根据用户反馈：**
- 标题不够吸引 → 重新拟标题
- 正文太长/太短 → 调整长度
- 图片不对 → 重新下载替换
- 整体不满意 → 重新改写

**执行命令：**
```bash
# 替换文档图片
lark-cli docs +media-insert \
  --doc <文档ID> \
  --file "新图片.jpg" \
  --selection-with-ellipsis "[原图片标记]"
```

## 关键限制说明

| 问题 | 说明 |
|------|------|
| 微信图片被拦截 | 云服务器IP访问微信图片返回400，需从公开图库下载 |
| 飞书API权限 | `application:self_manage` 缺失时报错，用 lark-cli 绕过 |
| 图片文字环绕 | 飞书文档API不支持图文混排，只能手动调整 |
| 去AI味 | humanize-chinese-2-0-0 仅对AI生成的文本有效，人类原文改写需阿呆直接处理 |
