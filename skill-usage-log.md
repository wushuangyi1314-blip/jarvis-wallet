> 创建时间：2026-04-20
> 用途：记录阿呆（主Agent）每次加载/触发技能的日志，供定期统计和报告生成

---

## 使用规则

### 记录时机
1. 阿呆直接加载某个 SKILL.md 时，顺手追加一行
2. 阿呆 spawn SubAgent 执行某技能任务时，顺手追加一行
3. SubAgent 内部技能使用**不记录**（run 模式无状态，由阿呆层面记录）

### 记录格式
```
| 时间 | 技能 | 来源 | 说明 |
| 2026-04-20 08:50 | copywriting | 阿呆 | 写推广文案 |
| 2026-04-20 09:00 | git-workflows | 阿呆 | rebase解决冲突 |
```

### 字段说明
| 字段 | 说明 |
|------|------|
| 时间 | ISO 8601 格式，精确到分钟 |
| 技能 | skill 目录名（如 copywriting、git-workflows） |
| 来源 | 阿呆 / rd-agent / operations-agent 等 |
| 说明 | 简短描述本次触发原因（可选） |
