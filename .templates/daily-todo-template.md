## 模板使用说明

### 字段说明

| 字段 | 说明 |
|------|------|
| 🔴 未完成 | 验证后确实未完成的任务 |
| 🟡 待确认 | 需要用户提供确认或信息 |
| ✅ 今日可完成 | 当前可以立即执行的行动项 |
| 📌 今日行动建议 | 根据任务优先级给出的建议 |

### 验证要求

- 每个任务必须说明验证情况
- 不能只说"未完成"，要说明"验证了什么"
- 下一步必须具体可执行

### 更新规则

- 完成时更新状态为 ✅，记录完成时间
- 禁止删除待办条目
- 保留所有历史变更记录
