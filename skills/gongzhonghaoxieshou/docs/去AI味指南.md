# 去AI味指南

公众号内容创作必须去除AI痕迹，让文章有真人味。

---

## 核心问题

AI写作的典型问题：
- 用词精准但缺乏个性
- 结构完美但缺乏节奏
- 内容正确但缺乏温度
- 说理清晰但缺乏故事

---

## 十条去AI味规则

### 1. 删除填充短语

- 删除"可以"、"能够"、"能够帮助"
- 直接说"这样做"而不是"能够这样做"
- 删除"在这个时间点"、"在……的情况下"

### 2. 打破公式结构

- 避免"不仅仅……而且……"的二元对比
- 不要用三段式列举，改用两项或四项
- 避免戏剧性的分段和修辞性设置

### 3. 变化节奏

- 混合使用短句和长句
- 不要让连续三个句子长度相同
- 段落结尾方式要多样化

### 4. 删除AI高频词

**避免使用的词**：
- 此外、确保、强调、关键、体现
- 标志着、彰显了、凸显
- 培养、促进、提升

**替换建议**：

| AI词 | 自然表达 |
|------|---------|
| 此外 | 直接写下一句，不用连接词 |
| 确保 | 直接说结果 |
| 强调 | 用具体例子代替 |
| 培养 | 用"学会"、"掌握" |
| 提升 | 用具体数据说明 |

### 5. 去除机械强调

- 减少粗体使用，只强调真正重要的内容
- 不要用"必须"、"应该"、"务必"等命令式语气
- 避免括号中的重复解释

### 6. 增加个性

- 适当使用"我"等第一人称表达观点
- 可以有真实的反应和感受
- 允许一些不确定性："我倾向于认为……"

### 7. 具体而非模糊

| 模糊 | 具体 |
|------|------|
| 这个方法很好 | 这个方法让80%的人效率翻倍 |
| 很多人喜欢 | 2024年调查中，85%的人表示喜欢 |
| 效果显著 | 使用后投诉率下降60% |

### 8. 用口语表达

| 书面表达 | 口语表达 |
|---------|---------|
| 进行深入分析 | 仔细看了看 |
| 取得显著成效 | 效果很明显 |
| 具有重要意义 | 很重要 |

### 9. 避免空洞结论

- 不要用"激动人心的时代即将到来"等套话
- 用具体行动代替："下次我们试试……"
- 删除"这代表了重要的一步"等无意义表述

### 10. 自然表达

- 使用"是/有"等简单动词
- 避免过度使用分词短语"-ing"
- 不要用破折号来增加戏剧性效果

## 示例对比

### AI风格（修改前）

> 在这个部分，我们将会深入探讨如何提升写作效率。写作效率是内容创作中至关重要的主题，它强调了时间管理的重要性。此外，掌握高效的写作技巧能够在实际应用中发挥关键作用，帮助创作者在有限的时间内产出更多优质内容。

### 去AI味后（修改后）

> 怎么写得更快？这事儿很多人关心。

> 时间就是钱，写得慢等于在浪费钱。我见过不少人，一篇文章写一天，最后发出去没人看。

> 其实写快不难，掌握几个技巧就行。今天说说我常用的几个方法。

**改进点**：
- 删除了"将会深入探讨"、"至关重要的"等AI高频词
- 删除了"此外"、"强调"等冗余表达
- 用口语表达代替书面表达
- 增加了"我"的第一人称视角
- 读起来像真人写的
