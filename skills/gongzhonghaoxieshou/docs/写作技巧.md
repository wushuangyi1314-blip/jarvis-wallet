# 写作技巧库

公众号内容创作的实用技巧汇总。

---

## 标题创作技巧

### 数字标题公式

```
[数字] + [方法/工具/要点] + [效果/结果]
```

示例：
- "5个方法，让你的阅读效率翻倍"
- "3个工具，帮你摆脱拖延症"
- "10条建议，写给刚入职的你"

### 身份标题公式

```
写给[身份人群] + [痛点/需求/建议]
```

示例：
- "写给30岁还在迷茫的你"
- "宝妈必看：这样做孩子更聪明"
- "程序员自救指南"

### 反差标题公式

```
[常见认知] + 但是 + [真相/反转]
```

示例：
- "你以为的努力，其实是在浪费时间"
- "看似省钱的习惯，其实在花钱"

## 爆款结构模板

### 金字塔模式模板

```markdown
# [观点/结论]

[开篇：用痛点或故事引入观点]

## 核心观点
[详细阐述核心观点]

## 支撑论据
论据一：[具体案例或数据]
论据二：[具体案例或数据]

## 实践建议
[给出可操作的建议]

## 结语
[总结或引发思考]
```

### 故事模式模板

```markdown
# [故事主题]

## 起：开篇
[人物背景、情境设定]

## 承：发展
[故事推进、关键转折]

## 转：高潮
[核心冲突、解决过程]

## 合：收尾
[结局、感悟升华]
```

### 问题模式模板

```markdown
# [问题主题]

## 问题提出
[痛点或困惑描述]

## 原因分析
[问题产生的深层原因]

## 解决方案
方案一：[具体方法]
方案二：[具体方法]

## 实践案例
[实际应用案例]
```

## 开头模板

### 痛点开头

```
你是不是也遇到过这样的情况：
[描述痛点场景]
[描述读者可能的反应]
其实，问题的根源在于...
```

### 数据开头

```
根据[权威来源]的调查，
[呈现数据]
这个数据背后，折射出的问题是...
```

### 故事开头

```
[时间/地点]，[人物]遇到了一件事：
[描述事件]
这件事让我想到了...
```

## 金句创作公式

| 公式 | 示例 |
|------|------|
| **对比式** | "不是你不够努力，而是方法不对" |
| **递进式** | "从0到1很难，但从1到100很简单" |
| **反问式** | "你还在等机会，机会却在等你吗" |
| **定义式** | "成长，就是不断打破认知的过程" |
| **因果式** | "因为相信，所以看见" |

## 结尾模板

### 总结式结尾

```
总结一下，[核心观点]。
记住这几点：[要点罗列]。
从今天开始，[行动建议]。
```

### 反问式结尾

```
你是否也认同[观点]？
如果答案是肯定的，不妨试试[建议]。
```

### 金句式结尾

```
最后送你一句话：
[金句]
```
