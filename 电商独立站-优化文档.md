## 一、WordPress网站搭建学习总结

### 1.1 WordPress核心架构

| 组件 | 说明 |
|------|------|
| **WordPress Core** | 核心程序，处理内容管理、URL路由、用户认证 |
| **Theme (主题)** | 控制网站外观和布局，Astra是轻量级高性能主题 |
| **Plugin (插件)** | 扩展功能，WooCommerce是最流行的电商插件 |
| **Database** | MySQL存储所有内容、配置和产品数据 |
| **Media Library** | 存储上传的图片/文件 |

### 1.2 Astra主题关键知识

**Astra特点：**
- 轻量级（<50KB）、加载快、对SEO友好
- 与WooCommerce深度集成
- 支持古腾堡编辑器和Elementor页面构建器
- 有大量预制模板可一键导入

**Astra主题结构：**
```
wp-content/themes/astra/
  assets/css/          # 样式文件
  inc/                 # 核心功能
  inc/builder/         # 页面构建器模块
  template-parts/       # 模板部件
```

**Astra + WooCommerce集成：**
- WooCommerce会自动应用Astra的WC样式（`woocommerce-grid.min.css`）
- 产品页自动应用Astra的容器宽度（1240px）
- 购物车和结账页面自动适配移动端

**Astra主题定制入口：**
1. `wp-admin/customize.php` - 主题自定义面板
2. `wp-admin/admin.php?page=astra` - Astra高级选项
3. Elementor（需安装）- 可视化页面编辑

### 1.3 WooCommerce核心知识

**WooCommerce设置页面结构：**
| Tab | 功能 |
|-----|------|
| General | 基础设置（货币、国家/地区、销售区域） |
| Products | 产品设置（标签、评论、库存管理） |
| Tax | 税务设置 |
| Shipping | 运费设置 |
| Payments | 支付网关（WooCommerce Payments、PayPal、Stripe等） |
| Account | 账户/隐私设置 |
| Email | 邮件通知模板 |
| Advanced | 高级（REST API、WEBHOOK、页面分配） |

**WooCommerce产品数据模型：**
```javascript
Product {
  id: number,
  title: string,
  description: string,  // 完整描述（富文本）
  short_description: string, // 简短描述（商城列表用）
  regular_price: string,   // 正常价格
  sale_price: string,      // 促销价格
  sku: string,             // 库存单位
  stock_quantity: number,  // 库存数量
  stock_status: 'instock' | 'outofstock', // 库存状态
  categories: [{ id, name, slug }],  // 产品分类
  images: [{ id, src, name, alt }],   // 产品图片
  reviews: [{ id, rating, comment }]  // 评论（需插件）
}
```

**WooCommerce REST API：**
- 默认禁用（需要在Settings > Advanced > REST API创建Key）
- 使用Consumer Key + Consumer Secret认证
- 支持CRUD操作产品、订单、客户

## 二、跨境独立站必备插件

### 2.1 已安装插件

| 插件 | 版本 | 功能 | 跨境电商重要性 |
|------|------|------|:------------:|
| **WooCommerce** | 9.9.5 | 电商核心：产品/订单/支付/配送/库存 | ⭐⭐⭐ 必须 |
| **WooCommerce PayPal Payments** | 内置 | PayPal支付网关 | ⭐⭐⭐ 推荐（跨境收款） |
| **Starter Templates** | - | 一键导入完整网站模板 | ⭐⭐ 可选 |

### 2.2 推荐安装插件（跨境必备）

| 插件 | 功能 | 用途 |
|------|------|------|
| **WP Mail SMTP** | 配置邮件发送（SMTP） | 交易邮件、订单确认邮件送达 |
| **Yoast SEO** | SEO优化 | 搜索引擎排名 |
| **Wordfence Security** | 安全防护 | 防黑客、防垃圾评论 |
| **WP Rocket** | 网站加速缓存 | 提升页面加载速度（对SEO重要） |
| **Product Reviews for WooCommerce** | 商品评论 | 增强用户信任感，提升转化率 |
| **Table Rate Shipping** | 复杂运费规则 | 按重量/地区/订单金额计算运费 |
| **MonsterInsights** | Google Analytics集成 | 流量分析 |
| **Smush** | 图片压缩优化 | 加速网站 |

### 2.3 支付网关选择（跨境电商）

| 支付方式 | 说明 | 适用场景 |
|---------|------|---------|
| **PayPal** | 全球最普及 | 跨境独立站必备 |
| **Stripe** | 支持国际信用卡 | 进阶支付方案 |
| **Alipay/WeChat Pay** | 支付宝/微信 | 如果目标客户是中国人 |
| **Klarna** | 先买后付 | 北美/欧洲市场流行 |

## 五、技术限制记录

### 5.1 已验证的自动化限制

| 操作 | 限制原因 | 解决方案 |
|------|---------|---------|
| 产品图片上传 | WordPress媒体库需要实际文件上传 | 需手动或通过REST API |
| Select2下拉框 | WooCommerce的Select2与Playwright不兼容 | 使用JS evaluate直接设置值 |
| 支付网关配置 | 需要外部账号授权 | 手动在后台配置 |
| 产品分类关联 | Gutenberg分类面板加载时机问题 | 需在产品编辑页面手动选择 |
| 优惠券创建 | Gutenberg界面加载超时 | 需手动创建 |

### 5.2 Playwright自动化发现

**成功自动化的操作：**
- 登录WordPress后台
- 填写产品标题、价格
- 保存产品
- 修改WooCommerce通用设置
- 创建产品分类
- 浏览产品列表

**无法自动化的操作：**
- 媒体库上传（需要文件上传API）
- Gutenberg富文本编辑
- Select2下拉框选择
- Thickbox媒体弹窗交互

## 六、主题评估

### 6.1 Astra主题评估

**优点：**
- ✅ 轻量、快速，对服务器友好
- ✅ 与WooCommerce完美集成
- ✅ 支持预制模板（可一键导入完整商城）
- ✅ 可通过Customizer深度定制

**缺点：**
- ⚠ 默认界面较朴素，需要深度定制
- ⚠ 部分高级功能需要付费版（Astra Pro）

**结论：** Astra适合当前的简单电商场景。如果想要更精美的商城界面，可以：
1. 购买Astra Pro解锁更多预制模板
2. 或安装Elementor页面构建器
3. 或导入Starter Templates中的完整商城模板

### 6.2 替代主题推荐

| 主题 | 特点 | 费用 |
|------|------|------|
| **Flatsome** | 最流行的WooCommerce主题，完整商城功能 | $59（有License） |
| **Shopkeeper** | 现代简洁的设计，适合高端产品 | $59 |
| **Astra** | 当前选择，轻量快速 | 免费（基础版） |
