## 一、概述

### 1.1 星空巨兽定义

星空巨兽是宇宙中最古老、最强大的生命形式之一。它们不是智慧文明，而是以本能驱动的宇宙生物。它们的体型从小行星大小到恒星级别不等，能力从基础的物理攻击到操控引力、时空、量子态等宇宙基本力量。

### 1.2 星空巨兽分类

| 类型 | 数量 | 科技等级 | 特点 |
|------|------|---------|------|
| 引力型巨兽 | 4种 | 3-4级 | 操控引力场 |
| 能量型巨兽 | 5种 | 2-3级 | 吸收和释放能量 |
| 时空型巨兽 | 3种 | 4级+ | 穿越时空 |
| 量子型巨兽 | 4种 | 3-4级 | 量子叠加态 |
| 物质型巨兽 | 4种 | 3级 | 改变物质形态 |
| 心灵型巨兽 | 3种 | 2-3级 | 跨星系心灵感应 |
| 吞噬型巨兽 | 4种 | 3级 | 吞噬天体 |
| 复合型巨兽 | 3种 | 4-5级 | 多种能力 |

### 1.3 命名规范

星空巨兽的命名遵循"形态特征+行为特征"的原则，采用宇宙中真实存在的星云、星系、恒星等命名灵感。

## 二、引力型巨兽（4种）

### 2.1 引力巨鲸（Gravitational Leviathan）

| 属性 | 详情 |
|------|------|
| 体型 | 直径约500-2000公里 |
| 形态 | 巨大的鲸鱼状生物，皮肤呈深黑色，边缘有引力波纹 |
| 能力 | 操控引力场，扭曲周围空间 |
| 分布 | 银河系旋臂，尤其喜欢在恒星附近活动 |
| 威胁等级 | 🔴极高 |
| 特殊材料 | 引力晶核（可制造引力武器）、时空纤维（可制造曲速引擎） |

**能力详解**：
- **引力扭曲**：在巨兽周围形成引力透镜，可弯曲光线
- **引力捕获**：释放引力井，捕获经过的飞船或小行星
- **引力冲击**：将引力波凝聚为定向冲击波

**行为特征**：
- 单独行动，领地意识极强
- 对恒星能量有本能的吸引力
- 被认为是宇宙中最古老的生命形式之一

### 2.2 深空螺牛（Deep Space Nautilus）

| 属性 | 详情 |
|------|------|
| 体型 | 直径约50-100公里 |
| 形态 | 螺旋壳状结构，壳由引力结晶构成 |
| 能力 | 引力场编织，形成防御护盾 |
| 分布 | 银河系边缘稀疏星域 |
| 威胁等级 | 🟡中高 |
| 特殊材料 | 引力结晶外壳（制造引力护盾核心材料） |

**能力详解**：
- **引力编织**：用引力线编织成防御护盾，可抵御激光和导弹攻击
- **引力脉冲**：向周围释放引力波，用于通讯或攻击
- **引力结晶**：能够将引力能量结晶化为固体物质

**行为特征**：
- 行动缓慢，但防御力极强
- 受到威胁时会缩入壳中
- 通常不主动攻击其他生命

**行为特征**：
- 穿越冰态星域寻找食物
- 对温暖的物体有好奇感
