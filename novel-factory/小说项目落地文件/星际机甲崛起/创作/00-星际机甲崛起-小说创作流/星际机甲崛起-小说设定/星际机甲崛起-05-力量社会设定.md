## 一、力量体系

### 力量类型总览

| 力量类型 | 层级 | 说明 |
|---------|------|------|
| 机甲战力 | 个人级 | 单台机甲的作战能力 |
| 舰队战力 | 集体级 | 多舰协同作战能力 |
| 明月子体 | 无人上限 | AI控制的无人机组 |
| 科技武器 | 星系级 | 轨道炮、战略导弹等 |
| 阿尔佐姆遗产 | 超越一切 | 领先现代200-500年的禁忌技术 |

### 力量来源

| 来源 | 机制 |
|------|------|
| 机甲 | 阿尔佐姆遗产技术（振金装甲+量子能核心） |
| 舰队 | 工业制造能力（造船规模） |
| 明月子体 | 量子AI生命（子体控制无上限） |
| 科技 | 联盟主流技术+阿尔佐姆遗产破解 |

### 高层战力特征

| 层级 | 特征 |
|------|------|
| 仲裁者级文明 | 可碾压恒星，利用零点能掌控整个星系 |
| 单体王牌机甲 | 可对抗整支常规舰队（使徒IV级） |
| 明月终极体 | 子体无上限，可控制整个舰队群 |
| 阿尔佐姆遗产 | 领先现代科技200-500年，无代差碾压 |

### 修炼/提升方式

| 方式 | 说明 |
|------|------|
| 机甲驾驶技术 | 操控能力的精进，熟能生巧 |
| 精神力同步 | 与明月/机甲的契合度，影响战斗力上限 |
| 科技知识 | 技术解读能力，决定装备利用率 |

## 二、社会体系

### 社会结构

```
银河议会（最高权力机构）
  └── 安理会（日常最高决策机构）
        └── 各成员种族政府
              └── 清道夫舰队（联盟执法）
                    └── 方舟（夏天势力）
                          └── 地球反抗军
```

### 阶层划分

| 等级 | 名称 | 代表势力 |
|------|------|---------|
| 一级 | 霸主 | 仲裁者文明 |
| 二级 | 强权 | 维拉图斯帝国 |
| 三级 | 领主 | 图萨联邦、银狮族、艾尔文明 |
| 四级 | 独立 | 人类、托尼克联邦 |
| 五级 | 附属 | 部分蛮荒小文明 |
| 六级 | 原始 | 边缘土著 |

### 主要组织

| 组织 | 性质 | 说明 |
|------|------|------|
| 银河议会 | 立法 | 名义最高权力，实际被高等种族操控 |
| 安理会 | 行政 | 日常最高决策机构 |
| 清道夫舰队 | 军事执法 | 镇压低等种族反抗 |
| 各成员种族政府 | 行政 | 各自管理内部事务 |
| 方舟 | 反抗势力 | 夏天建立的秘密势力 |
| 地球反抗军 | 地下组织 | 争取人类独立的武装力量 |

## 三、职业等级体系

### 机甲驾驶员等级

| 等级 | 名称 | 同步率要求 | 特征 |
|------|------|-----------|------|
| 1级 | 见飞行员 | 30-45% | 新手，只能执行简单任务 |
| 2级 | 士官级 | 45-60% | 老兵，正常战斗能力 |
| 3级 | 军官级 | 60-75% | 精锐，超常反应和判断 |
| 4级 | 将官级 | 75-90% | 王牌，接近本能反应 |
| 5级 | 阿尔佐姆级 | 90%+ | 传说，完美人机合一 |

**同步率效果：**

| 同步率 | 等级 | 效果 |
|--------|------|------|
| 0-30% | 基础 | 只能执行简单操作 |
| 30-60% | 熟练 | 正常战斗能力 |
| 60-80% | 精英 | 超常反应和判断 |
| 80-95% | 王牌 | 接近本能反应 |
| 95%+ | 阿尔佐姆级 | 完美人机合一，副作用风险极高 |

### 军队编制

#### 图萨联邦标准编制

| 编制 | 规模 | 说明 |
|------|------|------|
| 小队 | 4-6台机甲 | 基本战术单位 |
| 中队 | 24-32台机甲 | 战术主力 |
| 大队 | 64台机甲 | 战役基本单位 |
| 联队 | 256台机甲 | 战役主力 |
| 军团 | 1024+台机甲 | 战略单位 |

#### 方舟舰队编制

| 编制 | 规模 | 说明 |
|------|------|------|
| 明月小队 | 10-30台 | 精锐突击单位 |
| 混编队 | 50台 | 常规作战单位 |
| 旗舰编队 | 200台 | 核心决战单位 |
| 方舟编队 | 500+台 | 全面战争单位 |

## 五、种族设定

### 人类

| 项目 | 内容 |
|------|------|
| 特征 | 碳基类人，1.7-1.8米，寿命100年 |
| 天赋 | 应用科技转化快、学习能力强、坚韧 |
| 地位 | 四级独立级，联盟底层 |
| 文化 | 生存、发展、尊严 |

### 图萨联邦

| 项目 | 内容 |
|------|------|
| 特征 | 类人，2.2-2.5米，灰蓝皮肤，四眼，寿命300年 |
| 天赋 | 秩序管理、技术应用、军事 |
| 地位 | 三级领主，安理会成员 |

### 维拉图斯帝国

| 项目 | 内容 |
|------|------|
| 特征 | 硅基机械，60%+机械，寿命无限 |
| 天赋 | 逻辑计算、机械改造、量子技术 |
| 地位 | 二级强权，安理会常任 |

### 银狮族

| 项目 | 内容 |
|------|------|
| 特征 | 兽化种族，2.5-3米银色毛发，寿命150年 |
| 天赋 | 战斗、追踪、生物强化 |
| 地位 | 三级领主 |

### 艾尔文明

| 项目 | 内容 |
|------|------|
| 特征 | 能量体生命，可凝聚实体，寿命500年 |
| 天赋 | 量子通讯、能量存储、远程感知 |
| 地位 | 三级领主 |

### 仲裁者文明

| 项目 | 内容 |
|------|------|
| 特征 | 高度进化类人，3米以上，淡金色皮肤，寿命800年 |
| 天赋 | 星系级武器、空间折叠、绝对权威 |
| 地位 | 一级霸主，安理会常任+否决权 |
