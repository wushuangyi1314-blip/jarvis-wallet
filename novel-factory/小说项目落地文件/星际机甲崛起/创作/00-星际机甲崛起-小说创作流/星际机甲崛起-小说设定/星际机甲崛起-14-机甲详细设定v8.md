# 二、机甲详细设定

## 1.1 骑士II型 Knight-II（联盟第二代标准机）

| 项目 | 内容 |
|------|------|
| **代际** | TL-2（科技1级） |
| **研发时间** | 约公元2180年代 |
| **研发国家** | 图尔萨联邦 |
| **设计用途** | 全域通用型，主力制式装备 |
| **功能定位** | 平衡型 |
| **驾驶方式** | 神经链接（手动备份） |
| **装甲** | 复合装甲（内层合金钢+外层陶瓷复合材料），总厚度80-120mm |
| **自重** | 75吨 |
| **高度** | 7.2米 |
| **动力系统** | 小型核聚变反应堆，输出功率约2500千马力 |
| **制式武器** | 重联装30mm光束炮×1（功率500kW，射程8km）；四联装反载具导弹×1（射程10km，多目标攻击） |
| **特殊武器** | 速射近防炮×2（拦截来袭导弹/无人机）；战术烟雾发射器×6 |
| **防御武器** | 能量护盾（抵挡TL-1武器，5%/小时） |
| **产量** | 50万+台 |

**航速：**
- 大气层常规：120km/h
- 大气层极速：180km/h
- 越野模式：75km/h
- 悬浮模式：悬停高度5米，持续30秒
- 星际巡航：800km/s

**续航时间：**
- 常规巡逻：10-20小时
- 中等强度战斗：2-5小时
- 高烈度战斗：1-2小时
- 燃料补给：任何能源补充站

**曲速巡航：** 不可（TL-2以下不具备曲速能力）

**设计背景：** 骑士II是图尔萨联邦统一后的第一种标准化机甲，旨在替换各国原有的杂乱机甲体系。设计理念是"够用就好"——不在任何单一性能上追求极致，而是追求整体平衡和可维护性。联邦采用模块化设计，允许各盟国根据自身需求进行轻度改装，因此骑士II的实际装备型号多达数十种变型。作为人类机甲工业的基石，骑士II参与了人类与虫族战争的几乎每一场战役，是当之无愧的"战场老兵"。

**外观描述：** 骑士II是联盟工业美学标杆，整体身形修长匀称，高宽比约3:1。头部采用"单眼"设计，大型多光谱传感器居中嵌入黑色面罩式头盔，激活时传感器光源由暗红渐变为明亮红；躯干线条硬朗，胸腔部位内置核聚变反应堆，呼吸灯在激活时由蓝绿渐变为明黄；肩甲外侧刻有图尔萨联邦鹰徽纹章，膝部与手肘护甲呈斜切式装甲板；配色以深灰与铁黑为主，联邦钴蓝色作为细节点缀。辨识特征：背部两块矩形散热板呈"八"字形展开。

**外观描述：** 仲裁者III是"猎杀者"的代名词，整体造型充满压迫感。头部采用双传感器"双眼"布局，面甲正中有一道蓝色光源线，激活时表示锁定状态；躯干呈楔形结构前倾，主武器炮管从胸前中央延伸；肩甲宽大外张呈倒梯形，左肩黑色闪电徽章，右肩银色天平徽章；配色以哑光黑为底，暗金色勾边，传感器光源为深红色。辨识特征：胸前延伸的重型光束炮炮管。
