# 通用部分

## 世界规则设定

### 机甲改装框架

#### 1. 模块化改装体系

机甲改装采用**五大模块**架构，每台机甲可根据战术需求选择性加装或升级对应模块。

| 模块 | 改装内容 | 改装负担 | 代表改装件 |
|------|---------|---------|-----------|
| 武器模块 | 光束炮 / 导弹 / 近战武器升级 | 高 | 重型光束炮 / 智能导弹系统 |
| 装甲模块 | 外甲 / 内衬 / 护盾发生器 | 中 | 振金装甲 / 能量护盾 |
| 动力模块 | 引擎 / 能量核心 / 冷却系统 | 极高 | 幽能引擎 / 量子核心 |
| 推进模块 | 腿部 / 背部 / 关节强化 | 高 | 矢量推进器 / 空间折叠推进 |
| 电子战模块 | 雷达 / 干扰 / 通讯 / AI辅助 | 中 | 量子雷达 / 全谱干扰器 |

> **改装负担说明：** 负担越高，对驾驶员的体能消耗和同步率要求越大。部分高负担改装件需配合身份等级方可合法驾驶。

#### 改装科技等级要求

**核心原则：改装件必须与机甲的能源系统兼容，不同科技等级的改装件有严格的安装门槛。**

| 改装件科技等级 | 可安装机甲 | 说明 |
|--------------|---------|------|
| 0级改装件 | TL-1（科技0级）以上 | 化学能改装件，基础改装 |
| 1级改装件 | TL-2（科技1级）以上 | 核聚变系改装件 |
| 2级改装件 | TL-3（科技2级）以上 | 幽能/暗物质系改装件 |
| 3级改装件 | TL-4（科技3级）以上 | 零点能/空间系改装件 |
| 4级改装件 | TL-5（科技4级）以上 | 弦能/量子能改装件 |

**改装件与ML等级的关系：**
- ML-1 改装：通常使用1级改装件
- ML-2 改装：通常需要2级改装件，部分核心件需要3级
- ML-3 改装：需要2-3级改装件，高端改装件为3级
- ML-4 改装：需要3-4级改装件
- ML-X 改装：超越所有等级，使用遗产级4级以上技术

#### 2. 改装负担体系

##### 改装度（Modularity Level）

| 等级 | 名称 | 负担加成 | 说明 |
|------|------|---------|------|
| ML-0 | 制式机甲 | 无 | 出厂标配，无任何改装 |
| ML-1 | 轻度改装 | +10% | 标准化小幅升级 |
| ML-2 | 中度改装 | +25% | 核心组件替换，战斗力显著提升 |
| ML-3 | 重度改装 | +50% | 多模块联合改装，需高超驾驶技术 |
| ML-4 | 极限改装 | +100% | 逼近机甲物理极限，驾驶受严格限制 |

##### 操作负担

改装度提升后，驾驶员将面临以下操作压力：

- **反应速度要求提升**：系统延迟压缩，驾驶员神经同步频率需跟上改装后的响应节奏
- **同步率阈值提高**：人机合一的要求更高，意志薄弱者难以驾驭
- **驾驶员体能消耗增加**：高功率输出意味着更大的身体负荷，长期驾驶损耗显著

---

#### 3. 身份等级制度

机甲改装与驾驶权限实行**身份绑定**，不同等级对应不同的合法驾驶权限。

| 等级 | 可驾驶ML | 身份要求 | 代表人物 |
|------|---------|---------|---------|
| 尉官级 | ML-1 | 尉官以上 | 普通王牌驾驶员 |
| 校官级 | ML-2 | 校官以上 | 精锐王牌 |
| 将官级 | ML-3 | 将官以上 | 舰队指挥官 / 传奇驾驶员 |
| 贵族限定 | ML-4 | 贵族血统 + 皇室批准 | 皇室成员 / 传说级 |
| 禁忌改装 | ML-X | 绝对禁止（存在即违规） | 夏天的初火号 |

> **ML-X 说明：** 超越所有等级限制的禁忌级改装，不存在于合法注册体系中。一旦出现即为秩序挑战者。

---

#### 4. 改装流派

根据战术定位和改装偏好，主流改装分为五大流派：

| 流派 | 特点 | 典型改装 | 驾驶负担 |
|------|------|---------|---------|
| 重装型 | 高防护 / 高火力 / 低速 | 双层振金装甲 + 重型光束炮 | 极高 |
| 速攻型 | 高机动 / 高速度 / 轻装甲 | 矢量推进 + 轻型武器 | 高 |
| 均衡型 | 平衡攻防 / 通用性强 | 标准改装套件 | 中 |
| 电子战型 | 干扰 / 压制 / 信息战 | 全谱干扰器 + 量子雷达 | 中 |
| 特攻型 | 针对特定目标优化 | 反舰导弹 + 穿甲弹头 | 高 |

---

#### 5. 改装费用体系

改装不仅考验技术，更消耗大量资源。费用以**宇宙晶**为单位结算。

| 改装等级 | 改装费用（宇宙晶） | 年度维护成本 |
|---------|------------------|-------------|
| ML-1 | 5,000 B级 | 500 B级 |
| ML-2 | 50,000 B级 | 5,000 B级 |
| ML-3 | 500,000 B级 | 50,000 B级 |
| ML-4 | 5,000,000 B级（禁区市场） | 非卖品 |

> **B级宇宙晶：** 银河系通用标准结算单位，B级为中等价值单位，一次典型星际货运任务报酬约为 200～500 B级。

#### 6. 改装案例

##### 夏天初火号

- **原始型号：** 仲裁者III（定位为 ML-2 中度改装机甲）
- **改装等级：** ML-3+（超越原设计上限）
- **核心改装：**
  - 阿尔佐姆振金装甲——极限硬度外甲，重量却异常轻盈
  - 量子能核心——近乎无限的能量供给，代价是过热风险
  - 空间折叠刃——近战一击必杀兵器，可短距离空间斩击
- **身份限制：** 超越将官级，接近禁忌（ML-X 边缘）
- **特殊要求：** 需明月AI协同控制，普通驾驶员无法单独驾驭

> 夏天的初火号是少数被认为"选中了驾驶员"的机甲——不是人选择机甲，而是机甲选择了人。
