# 第一部分：核心概念

## 1.1 文明轮回定律

**核心定律：宇宙中的智慧文明存在周期性兴衰，平均周期约8-12亿年。**

| 阶段 | 周期 | 特征 |
|------|------|------|
| 萌芽期 | 1-10万年 | 行星文明诞生，核前时代 |
| 扩张期 | 10-1000万年 | 走出行星，建立星际殖民 |
| 鼎盛期 | 1000万-3亿年 | 银河系霸权，建立星际帝国 |
| 衰退期 | 100-1000万年 | 内部腐化+外部威胁 |
| 消亡期 | 10-100万年 | 文明崩溃，遗留遗迹 |
| 断层期 | 5000万-1亿年 | 无智慧文明，只有遗迹和虫族 |
| 重启期 | 10-100万年 | 新文明在遗迹上崛起 |

**轮回根因：**
- 资源枯竭
- 超级武器失控
- 虫族泛滥
- 维度侵蚀（高阶文明副作用）

## 1.3 当前宇宙纪年

**纪年方式：艾萨克纪年（Isaac Calendar）**

| 纪元 | 说明 |
|------|------|
| AE（After Empire） | 帝国后纪元，以阿尔佐姆帝国覆灭为元年 |
| BE（Before Empire） | 帝国前纪元，用于标注帝国前事件 |
| 当前时间 | AE 8,000,000年（约800万年） |

**时间基准：**
- 1艾萨克年 = 1.2地球年
- 使用银河标准时（GST）
