# 第一部分：虫族基础设定

## 1.1 虫族起源

| 项目 | 内容 |
|------|------|
| 起源时间 | 约10万年前（宇宙早期生命形式） |
| 起源地点 | 银河系边缘某未知星球 |
| 物种性质 | 非智慧种族，本能驱动，无文字/语言/文化 |
| 繁殖方式 | 母巢生产，无性繁殖为主，有性繁殖为辅，基因突变产生变异个体 |
| 社会结构 | 母巢是整个虫族的"大脑"，通过信息素控制所有个体，无个体意识 |
| 智慧等级 | 无智慧，本能行为，但领主级个体展现出初级战术能力 |
| 与阿尔佐姆帝国的关系 | 早于阿尔佐姆帝国存在，帝国覆灭后虫族大规模扩张 |
| 与联盟的关系 | 永恒的敌对，虫族对所有智慧种族都有攻击本能 |

**虫族的本质：** 虫族不是传统意义上的"敌人"，而是一种宇宙级别的生态灾害。没有善恶观，没有谈判可能，只有本能的繁殖和吞噬。

## 1.3 虫族信息素系统

虫族通过信息素进行通讯和控制，信息素是虫族唯一的通讯方式。

| 信息素类型 | 功能 | 作用范围 |
|-----------|------|---------|
| 恐惧信息素 | 传递危险信号，引导同类撤退 | 500米 |
| 攻击信息素 | 标记敌人位置，引导同类围攻 | 1公里 |
| 集结信息素 | 召集同类到指定位置 | 5公里 |
| 母巢指令 | 母巢发出的控制信号，优先级最高 | 10公里 |
| 领主指令 | 领主虫发出的战术指挥信号 | 3公里 |
| 领域标记 | 标记领地边界，警告外来者 | 1公里 |
| 繁殖信息素 | 吸引配偶或刺激产卵（仅特定时期） | 2公里 |

**信息素干扰：** 某些植物（如尤里西斯联邦的"沉默藤"）可分泌干扰信息素，是对抗虫族的天然手段之一。

**信息素抵抗：** 领主级以上的虫族对信息素干扰有一定抗性，高级领主甚至能分泌压制干扰的强效信息素。

## 1.4 虫族生理特征

### 通用特征

| 特征 | 描述 |
|------|------|
| 外骨骼 | 主要由几丁质+钙化层构成，强度极高，可防轻武器 |
| 甲壳颜色 | 根据种类不同从深褐到暗红不等，幼体颜色较浅 |
| 血液 | 蓝绿色，含有高浓度幽能（虫族储存能量的方式） |
| 眼睛 | 大多数虫族有复眼，视觉范围广但对静止物体感知差 |
| 肢体 | 大多数虫族为六足，少数特化型有四足或翼肢 |
| 幽能腺 | 所有虫族都有，用于储存和释放幽能 |
| 信息素腺 | 位于腹部，用于释放信息素 |
| 感知器官 | 触角（感知震动+信息素）、复眼（视觉）、听觉毛（声波） |

### 虫甲成分与利用

| 成分 | 占比 | 利用价值 |
|------|------|---------|
| 几丁质 | 40% | 可提炼几丁质用于工业材料 |
| 钙化层 | 30% | 可作为建筑材料 |
| 幽能 | 15% | 核心价值，可提炼幽能晶体 |
| 蛋白质 | 10% | 可作为饲料或特殊材料 |
| 其他 | 5% | 微量矿物质 |

**夏天商业模式：** 猎杀虫族→提取幽能晶体（主要收入）→甲壳卖给材料商→领主虫核心（极珍贵，单独出售）

## 1.5 虫族威胁等级

| 等级 | 描述 | 代表物种 |
|------|------|---------|
| 1级 | 无威胁，可忽略 | 虫族幼体、穴虫（无攻击性） |
| 2级 | 轻威胁，单体可击杀 | 拾荒虫（攻击型）、小型蝎刺 |
| 3级 | 中等威胁，需要机甲应对 | 标准蝎刺、螳母·蛛后（初级） |
| 4级 | 高威胁，需要战术配合 | 裂骨·重颚、母蛛·深渊织影、巨螯·凿石者（护卫型） |
| 5级 | 极高威胁，需要舰队级火力 | 幼巢·血卵、大型噬界者 |
| 6级 | 灾难级，区域灭绝风险 | 噬界者·深渊之蚀群落、母巢群落 |

# 第二部分：虫族分类体系

## 2.0 通用格式说明

每个虫种的数据表包含以下标准化区块：

### 基础信息
- 等级、拉丁学名、体型、体重、外观

### 攻击详解
- 攻击机制：近战/远程/特殊
- 攻击姿态：具体的攻击动作描写
- 攻击节奏：攻击频率和间隔
- 伤害数值：保持一致性（见伤害体系）

### 防御详解
- 护甲类型：外骨骼/复合骨甲/能量护盾
- 护甲厚度：具体数值
- 再生能力：是否能再生、再生时间
- 受伤反应：受伤后的行为变化

### 进化条件
- 进化前置条件
- 进化触发机制
- 进化周期

### 孕育信息
- 孕育方式
- 孕育周期
- 产出比例

## 4.1 基因突变机制

虫族的进化基于基因突变，但并非随机，而是有方向性的演化。

### 突变类型

| 类型 | 触发条件 | 进化方向 | 概率 |
|------|---------|---------|------|
| 环境适应突变 | 长期处于特定环境 | 适应该环境的特化 | 100%（必然发生） |
| 战术需求突变 | 遭遇特定敌人 | 产生克制敌人的能力 | 30% |
| 能量刺激突变 | 吸收大量幽能 | 强化战斗能力或体型 | 15% |
| 母巢指令突变 | 母巢发出进化指令 | 母巢指定进化方向 | 80% |

### 突变过程

1. **信息素触发**：母巢释放特定信息素激活目标个体的基因
2. **蜕变准备**：个体停止进食和战斗，寻找安全地点
3. **蜕皮/变态**：外骨骼脱落或身体重组
4. **基因表达**：新基因激活，身体结构重建
5. **能力适应**：新器官功能校准

## 4.3 战场进化（遭遇战适应）

- 标准虫种逐渐被替代
