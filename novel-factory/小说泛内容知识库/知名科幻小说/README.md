## 概述

本目录收集了10本经典/热门科幻小说的通用概念整理，其中：
- 前5本为国际经典科幻小说
- 后5本为星际科幻网文

每本小说抽取其中可以**独立使用、不绑定具体剧情**的概念，分为：
- 社会规则类概念
- 技术类概念
- 生物类概念

**收录原则：**
- 只抽取通用概念，不含具体剧情
- 概念可独立复用，不依赖原书世界观
- 每个概念包含：本质描述、关键属性、可复用场景
