# 星际航母 - 通用概念

> 来源：《星际航母》/ 三天故障 / 起点中文网
> 最后更新：2026-05-12

---

## 概述

《星际航母》是一部以星际战舰设计、建造和战术运用为核心的科幻小说。小说详细描述了星际战舰的设计原理、舰队战术、以及星际战争中的军事策略。

**核心价值：** 提供了星际战舰设计、舰队战术体系、战争博弈等可独立使用的技术和社会设定。

---

## 一，社会规则类概念

### 1. 星际军事战略

**本质：** 在星际尺度上进行战争的整体战略规划。

| 战略 | 描述 |
|------|------|
| 消耗战 | 通过持久战消耗敌方资源 |
| 闪电战 | 快速打击后立即撤退 |
| 封锁战 | 切断敌方补给线和通讯 |
| 斩首行动 | 摧毁敌方指挥系统 |

**可复用场景：** 星际战争、军事战略

---

### 2. 舰队指挥体系

**本质：** 星际舰队中的指挥和决策结构。

| 层级 | 职责 |
|------|------|
| 旗舰 | 舰队最高指挥舰 |
| 编队 | 若干战舰组成作战单元 |
| 战术组 | 具体战术执行单位 |
| 单舰 | 独立作战单元 |

**可复用场景：** 军事指挥、舰队管理

---

### 3. 星际后勤保障

**本质：** 维持舰队长期作战的后勤支持体系。

| 要素 | 描述 |
|------|------|
| 补给舰 | 提供弹药、能源、食物 |
| 维修舰 | 战场维修受损舰船 |
| 医疗舰 | 救治伤员 |
| 指挥舰 | 协调整个舰队后勤 |

**可复用场景：** 军事后勤、战争持续能力

---

## 二，技术类概念

### 4. 星际战舰设计

**本质：** 星际战舰的设计原理和功能分区。

| 系统 | 描述 |
|------|------|
| 推进系统 | 提供航行能力（核聚变/反物质） |
| 武器系统 | 攻击性武器（导弹/激光/粒子束） |
| 防护系统 | 护盾和装甲 |
| 动力系统 | 提供舰船运行能源 |
| 起降系统 | 舰载机/登陆艇的起降 |

**可复用场景：** 战舰设计、武器设定

---

### 5. 舰载机作战系统

**本质：** 航母搭载的舰载机和其作战方式。

| 类型 | 功能 |
|------|------|
| 战斗机 | 争夺制空权 |
| 轰炸机 | 对地面或舰船目标打击 |
| 侦察机 | 收集情报 |
| 电子战机 | 电子对抗 |

**可复用场景：** 空军战术、立体打击

---

### 6. 防御体系

**本质：** 战舰和舰队的多层防御系统。

| 层级 | 描述 |
|------|------|
| 外层防御 | 远距离拦截来袭武器 |
| 中层防御 | 护盾系统 |
| 内层防御 | 近程武器系统 |
| 装甲 | 最后一道防线 |

**可复用场景：** 防御体系、战争设定

| 方向 | 描述 |
|------|------|
| 基因优化 | 通过基因工程提升身体素质 |
| 机械增强 | 安装机械增强体 |
| 意识增强 | 提升智力和感知能力 |
| 环境适应 | 适应不同星球环境 |

**可复用场景：** 人类进化、人体改造

| 现象 | 描述 |
|------|------|
