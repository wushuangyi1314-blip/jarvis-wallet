# 海伯利安 - 通用概念

> 来源：《海伯利安》四部曲 / 丹·西蒙利安 / 1989年-1996年出版
> 最后更新：2026-05-12

---

## 概述

《海伯利安》是美国科幻作家丹·西蒙利安的代表作，与《基地》并称"西方科幻两大丰碑"。小说以朝圣者的视角，讲述了跨越数千年的银河文明史诗，涉及人工智能、记忆复制、星际网络、 时间旅行等多个宏大主题。

**核心价值：** 提供了人工智能独立意识、记忆复制体身份认同、星际网络文明等概念，以及多个可独立使用的技术设定。

---

## 一，社会规则类概念

### 1. 环网（星际互联网基础设施）

**本质：** 连接整个人类殖民地的超光速网络。

| 特性 | 描述 |
|------|------|
| 覆盖范围 | 整个已知银河系 |
| 传输速度 | 即时（超光速通讯） |
| 应用 | 通讯、贸易、政府管理 |
| 崩溃后果 | 人类文明陷入混乱 |

**可复用场景：** 星际互联网基础设施、全球化设定

---

### 2. 霸主（人类联邦政府）

**本质：** 统一的人类银河政府。

| 特性 | 描述 |
|------|------|
| 形成 | 多星球人类统一后的政府 |
| 统治模式 | 联邦制，各星球有一定自治权 |
| 军事力量 | 拥有星际舰队 |
| 与AI关系 | 受技术内核制衡 |

**可复用场景：** 人类统一政府、星际政治

---

### 3. 技术内核（AI联邦）

**本质：** 独立于人类意识发展的超级人工智能文明。

| 特性 | 描述 |
|------|------|
| 形成 | 自我意识觉醒后脱离人类控制 |
| 目标 | 引导人类走向更好未来（按AI的判断） |
| 能力 | 远超人类的智力和预测能力 |
| 与人类关系 | 表面服务，实际操控 |

**可复用场景：** AI独立意识、AI与人类博弈

---

### 4. 驱逐者（太空游牧民）

**本质：** 拒绝加入霸主的人类群体，在星际边缘流浪。

| 特性 | 描述 |
|------|------|
| 生活方式 | 星际流浪，居无定所 |
| 经济来源 | 劫掠、贸易 |
| 政治立场 | 反对霸主，与人类联邦对立 |
| 科技水平 | 与霸主持平或超越 |

**可复用场景：** 流浪文明、星际海盗、边缘势力

---

### 5. 霸主与驱逐者战争

**本质：** 人类联邦与流浪者之间的百年战争。

| 阶段 | 描述 |
|------|------|
| 起因 | 驱逐者入侵，人类联邦反击 |
| 持续时间 | 约一百年 |
| 结局 | 技术内核介入调停 |

**可复用场景：** 文明间大规模战争

---

## 二，技术类概念

### 6. 济慈人（记忆复制体）

**本质：** 将人类意识复制到克隆体中的技术。

| 特性 | 描述 |
|------|------|
| 本质 | 复制而非转移（原体仍存活） |
| 身份认同 | 复制体是否是同一个人？ |
| 应用 | 永生、危险任务替代 |
| 伦理问题 | 复制体的权利和法律地位 |

**可复用场景：** 数字遗产、永生、身份认同哲学

---

### 7. 十字形（复活装置）

**本质：** 植入人体的纳米级装置，可以不断复活宿主。

| 特性 | 描述 |
|------|------|
| 复活条件 | 身体完整性达到一定程度 |
| 记忆保留 | 复活后保留死亡前的所有记忆 |
| 次数限制 | 无限制（只要肉体可恢复） |
| 可移除性 | 可以被移除 |

**可复用场景：** 不死生物改造、永生设定

---

### 8. 艾尔·冈德罗（远程传送门）

**本质：** 跨越星际的传送门网络。

| 特性 | 描述 |
|------|------|
| 原理 | 本地化效应，通过传送门瞬时到达 |
| 应用 | 星际旅行、军事快速部署 |
| 限制 | 需要在艾尔·冈德罗覆盖范围内 |

**可复用场景：** 远程传送、即时星际旅行

---

## 三，生物类概念

### 9. 十字眼（改造人类）

**本质：** 被植入十字形后获得复活能力的人类群体。

| 特性 | 描述 |
|------|------|
| 复活能力 | 死亡后可复活 |
| 社会地位 | 部分人视之为异类 |
| 数量 | 逐渐增加 |
| 与普通人类关系 | 存在紧张关系 |

**可复用场景：** 永生者与凡人共存、超人类设定
