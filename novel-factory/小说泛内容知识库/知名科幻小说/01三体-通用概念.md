# 三体 - 通用概念

> 来源：《三体》三部曲 / 刘慈欣 / 2008年出版
> 最后更新：2026-05-12

---

## 概述

《三体》是中国科幻作家刘慈欣的代表作，讲述了地球文明与三体文明的首次接触与博弈。小说以文革为起点，构建了一套完整的宇宙文明社会学体系，其中的"黑暗森林法则"已成为科幻文学中最具影响力的文明博弈理论之一。

**核心价值：** 提供了宇宙文明之间"零和博弈"的社会学框架，以及多种可独立使用的技术设定。

---

## 一，社会规则类概念

### 1. 黑暗森林法则

**本质：** 宇宙是一座黑暗森林，每个文明都是带枪的猎人。发现其他文明后，唯一的选择是立即消灭。

**推导逻辑：**

```
前提1：宇宙中文明数量庞大（概率论证）
前提2：文明增长呈指数型，但宇宙资源总量固定
前提3：文明间地理距离远，通讯延迟导致猜疑链
前提4：文明可能随时发生"技术爆炸"（超越常理的发展速度）

结论：发现其他文明后，无法判断其善意或威胁
     技术爆炸可能使其瞬间超越自己
     唯一理性策略：立即消灭
```

**核心要素：**

| 要素 | 说明 |
|------|------|
| 生存本能 | 文明第一需求是生存，暴露=死亡 |
| 猜疑链 | 无法跨越的信息鸿沟 |
| 技术爆炸 | 落后不代表永远落后 |
| 清理成本 | 攻击成本远低于被超越的风险 |

**可复用场景：** 任何星际文明题材的文明博弈

---

### 2. 猜疑链

**本质：** 信息不对称导致的信任崩溃。

**形成过程：**

```
A文明发现B文明
→ A无法判断B是否善意
→ 即使B表示善意，A仍怀疑B在欺骗
→ B也无法确定A是否真的相信
→ 猜疑无限循环
→ 唯一理性选择：先动手
```

**可复用场景：** 任何涉及多势力博弈的题材

---

### 3. 宇宙社会学

**核心公理：**

```
公理1：生存是文明的第一需求
公理2：宇宙资源总量不变
公理3：文明不断增长扩张
```

**由此导出：** 宇宙是零和博弈，暴露即死亡。

## 二，技术类概念

### 4. 维度打击（二向箔）

**本质：** 将目标从高维降至低维的攻击。

| 属性 | 描述 |
|------|------|
| 原理 | 太阳系从三维降到二维，所有生命无法在低维中存活 |
| 特性 | 不可逆，一旦启动无法阻止 |
| 速度 | 缓慢扩散，但无法逃离（光速飞船也逃不过） |
| 防御 | 无 |

**可复用场景：** 超级文明对低级文明的碾压式打击

---

### 5. 智子（量子通讯+干扰）

**本质：** 利用量子特性实现的实时通讯+干扰装置。

| 功能 | 实现方式 |
|------|---------|
| 实时通讯 | 量子纠缠实现跨越光年的即时通讯 |
| 微观干扰 | 进入粒子加速器干扰实验结果 |
| 监控 | 高密度部署后覆盖整个星球 |
| 干扰核武器 | 智子可进入核弹引信使其失效 |

**可复用场景：** 超光年通讯、量子级干扰、监控网络

---

### 6. 光速飞船

**本质：** 接近或达到光速的航行技术。

| 特性 | 描述 |
|------|------|
| 能量需求 | 接近光速需要巨大能量 |
| 副作用 | 航迹暴露文明坐标 |
| 逃离能力 | 可以逃离二维化等灾难 |

**可复用场景：** 星际逃亡、太空竞赛

---

### 7. 技术锁死

**本质：** 通过限制基础科学认知，锁死对方发展。

**方式：**
- 干扰微观物理实验（让结果随机）
- 封锁关键技术资料
- 阻断技术交流

**可复用场景：** 星际战争、文明封锁

---

## 三，生物/社会类概念

### 8. 脱水休眠（极端生存适应）

**本质：** 生物体在恶劣环境下的生存策略。

| 阶段 | 状态 |
|------|------|
| 正常态 | 有机体形态，正常生活 |
| 脱水态 | 失去水分，变成纤维/干物质 |
| 复水态 | 遇水浸泡，恢复正常 |

**可复用场景：** 极端气候文明、末日生存、长途星际旅行

---

### 9. 三体运动（无规则恒星系统）

**本质：** 三个恒星在引力作用下做无规则运动。

| 状态 | 描述 |
|------|------|
| 恒纪元 | 三星远离，行星气候稳定 |
| 乱纪元 | 三星靠近，行星温度剧变 |
| 三星连珠 | 三星排成一线，行星被撕裂 |

**可复用场景：** 多恒星系统设定、不稳定环境文明

---

### 10. 技术爆炸

**本质：** 文明发展速度非线性。

**核心：** 一个落后文明可能在极短时间内实现技术飞跃，超越先进文明。"今天落后不代表永远落后。"

**可复用场景：** 星际文明竞争、追赶型文明叙事
