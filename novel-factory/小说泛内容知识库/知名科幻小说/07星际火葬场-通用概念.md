# 星际火葬场 - 通用概念

> 来源：《星际火葬场》/ 核动力烤鸡 / 起点中文网
> 最后更新：2026-05-12

---

## 概述

《星际火葬场》是一部星际科幻小说，讲述人类文明在宇宙中开拓生存空间、与外星文明竞争的故事。小说以星际开拓为核心，涉及文明竞争、物种进化、资源争夺等主题。

**核心价值：** 提供了星际开拓、文明的竞争与淘汰、物种进化等可独立使用的设定。

---

## 一，社会规则类概念

### 1. 星际开拓（文明扩张）

**本质：** 文明向宇宙扩展生存空间的行为模式。

| 阶段 | 描述 |
|------|------|
| 探测期 | 无人探测器探索目标星系 |
| 前哨期 | 建立小型科研或军事前哨 |
| 殖民期 | 大规模人口迁移和建设 |
| 同化期 | 将新领土融入文明体系 |

**可复用场景：** 星际殖民、文明建设叙事

---

### 2. 文明淘汰法则

**本质：** 宇宙中文明之间的弱肉强食规则。

| 法则 | 描述 |
|------|------|
| 资源有限 | 宇宙资源总量有限 |
| 竞争必然 | 文明之间必然产生竞争 |
| 弱肉强食 | 落后文明被先进文明淘汰 |
| 适应生存 | 不能适应的文明走向灭亡 |

**可复用场景：** 宇宙社会学、文明兴衰

---

### 3. 星际势力版图

**本质：** 宇宙中各文明势力范围的划分。

| 类型 | 描述 |
|------|------|
| 核心区 | 文明发源和主要居住区 |
| 边缘区 | 新开拓的领土 |
| 中立区 | 多方势力交界的缓冲地带 |
| 争议区 | 多方争夺的领土 |

**可复用场景：** 星际政治、地缘政治设定

---

## 二，技术类概念

### 4. 星际舰队战术

**本质：** 在星际空间进行战争的整体战术体系。

| 战术 | 描述 |
|------|------|
| 超视距打击 | 在视线范围外发起攻击 |
| 跃迁突袭 | 利用跃迁进行突然打击 |
| 联合打击 | 多舰种配合作战 |
| 防守反击 | 以防守消耗敌方后反击 |

**可复用场景：** 星际战争、舰队作战设定

---

### 5. 星际通讯网络

**本质：** 连接各星球和舰队的即时通讯系统。

| 类型 | 描述 |
|------|------|
| 即时通讯 | 各势力内部的实时通讯 |
| 情报网络 | 收集和分析敌方情报 |
| 通讯干扰 | 切断敌方通讯能力 |
| 加密传输 | 防止情报被窃取 |

**可复用场景：** 星际军事、情报战设定

---

### 6. 能源采集技术

**本质：** 在星际环境中获取能源的技术。

| 类型 | 描述 |
|------|------|
| 恒星能源 | 收集恒星释放的能量 |
| 戴森球 | 包裹恒星获取全部能量 |
| 反物质能源 | 正反物质湮灭释放的能量 |
| 真空零点能 | 从真空中提取的能量 |

**可复用场景：** 星际能源、超级工程设定

---

## 三，生物类概念

### 7. 物种进化逻辑

**本质：** 生物在宇宙环境中进化的规律。

| 规律 | 描述 |
|------|------|
| 环境选择 | 环境决定进化方向 |
| 基因突变 | 随机突变带来进化可能 |
| 适者生存 | 适应环境的个体存活 |
| 共生进化 | 不同物种相互影响进化 |

**可复用场景：** 外星生物进化、宇宙生物学

---

### 8. 改造生命体

**本质：** 通过技术手段改造的生物。

| 类型 | 描述 |
|------|------|
| 基因改造 | 改变生物基因组成 |
| 机械增强 | 在生物体上安装机械 |
| 半机械人 | 有机与机械结合的生命 |
| 完全机械 | 完全由机械构成 |

**可复用场景：** 人类进化、人体改造设定
