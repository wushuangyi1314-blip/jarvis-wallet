# 星际守望者 - 通用概念

> 来源：《星际守望者》/ 洛icos / 起点中文网
> 最后更新：2026-05-12

---

## 概述

《星际守望者》是一部以人类宇宙开拓史为主题的星际科幻小说，讲述了人类在宇宙中建立联邦、开拓殖民地、与外星文明接触的故事。

**核心价值：** 提供了星际联邦制度、人类宇宙开拓史、外星文明接触等可独立使用的设定。

---

## 一，社会规则类概念

### 1. 星际联邦制度

**本质：** 多个星球或文明联合形成的政治实体。

| 要素 | 描述 |
|------|------|
| 联邦宪法 | 联邦的根本大法 |
| 联邦议会 | 各星球代表组成的立法机构 |
| 联邦政府 | 联邦的执行机构 |
| 联邦法院 | 联邦的司法机构 |

**可复用场景：** 星际政府、政治制度

---

### 2. 人类宇宙开拓史

**本质：** 人类从地球走出、向宇宙扩展的历史进程。

| 阶段 | 描述 |
|------|------|
| 地球时代 | 人类在地球上的发展 |
| 太阳系时代 | 开发太阳系其他星球 |
| 星际时代 | 到达其他恒星系 |
| 宇宙时代 | 在整个宇宙中扩张 |

**可复用场景：** 文明发展史、时间跨度叙事

---

### 3. 守望者职责

**本质：** 在星际时代守护人类文明的特殊职责。

| 职责 | 描述 |
|------|------|
| 文明保护 | 保护人类免受外星威胁 |
| 边界巡逻 | 在星际边界巡逻 |
| 文明观察 | 观察和记录外星文明 |
| 危机应对 | 处理星际危机 |

**可复用场景：** 守护者设定、职责与使命

---

## 二，技术类概念

### 4. 星际航行技术

**本质：** 使人类能够在星际间航行的技术。

| 技术 | 描述 |
|------|------|
| 核聚变推进 | 利用核聚变产生推力 |
| 反物质推进 | 利用反物质湮灭产生能量 |
| 曲速引擎 | 理论上可以超越光速 |
| 冬眠技术 | 长途旅行中的休眠 |

**可复用场景：** 星际航行、探索设定

---

### 5. 星际通讯技术

**本质：** 在星际尺度上进行通讯的技术。

| 技术 | 描述 |
|------|------|
| 光速通讯 | 以光速传递信息 |
| 量子通讯 | 理论上可以即时通讯 |
| 中继站 | 在太空中建立信号中继 |
| 加密技术 | 防止信息被截获 |

**可复用场景：** 星际通讯、情报传递

---

### 6. 殖民地建设

**本质：** 在新星球上建立人类居住地的技术。

| 技术 | 描述 |
|------|------|
| 居留舱 | 最初的居住单元 |
| 生命支持 | 提供空气、水、食物 |
| 农业系统 | 在新星球上种植食物 |
| 工业系统 | 在新星球上建立工业 |

**可复用场景：** 殖民设定、文明建设

## 三，生物类概念

### 7. 外星文明接触

**本质：** 人类与外星文明相遇时的互动模式。

| 模式 | 描述 |
|------|------|
| 友好接触 | 与外星文明和平交流 |
| 敌对接触 | 与外星文明发生冲突 |
| 神秘接触 | 外星文明意图不明 |
| 文化碰撞 | 不同文明之间的文化冲突 |

**可复用场景：** 外星接触、文明碰撞

---

### 8. 人类多样性

**本质：** 星际时代人类在不同环境下的分化。

| 类型 | 描述 |
|------|------|
| 地球人 | 在地球上生活的人类 |
| 外星出生 | 在外星球出生的人类 |
| 宇宙游牧 | 在飞船上生活的人类 |
| 改造人 | 经过身体改造的人类 |

**可复用场景：** 人类分化、身份认同
