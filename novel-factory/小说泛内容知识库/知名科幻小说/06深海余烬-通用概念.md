# 深海余烬 - 通用概念

> 来源：《深海余烬》/ 远瞳 / 起点中文网
> 最后更新：2026-05-12

---

## 概述

《深海余烬》是远瞳的星际科幻代表作，讲述了在宇宙边缘的"深海"中，各星际文明与神秘虚空势力之间的生存博弈。小说构建了独特的"希灵神系"宇宙观，涉及多个星际种族的战争与共存。

**核心价值：** 提供了虚空文明设定、多神系并存宇宙、星际种族战争等可独立使用的社会和技术设定。

---

## 一，社会规则类概念

### 1. 希灵神系（多神宇宙体系）

**本质：** 宇宙中多个高等文明各自发展出独立的神系架构，多神并存。

| 特性 | 描述 |
|------|------|
| 神系独立 | 每个神系有独立的信仰体系和发展路径 |
| 神力来源 | 神力来源于信徒的信仰和文明本身的意志 |
| 神系冲突 | 不同神系之间存在竞争和冲突 |
| 凡人角色 | 凡人是神系存续的基础 |

**可复用场景：** 多神宇宙体系、神权政治设定

---

### 2. 虚空文明

**本质：** 来自宇宙边缘"深海"的未知文明，与已知星际文明接触时的生存策略。

| 特性 | 描述 |
|------|------|
| 未知性 | 文明形态和目的完全未知 |
| 威胁性 | 对已知文明构成潜在威胁 |
| 适应能力 | 可以适应各种宇宙环境 |
| 沟通方式 | 难以用常规方式沟通 |

**可复用场景：** 宇宙边缘设定、未知文明接触

---

### 3. 星际种族战争

**本质：** 多个星际文明之间的生存空间争夺战。

| 类型 | 描述 |
|------|------|
| 种族灭绝 | 消灭对方种族 |
| 领土扩张 | 占领对方星球 |
| 资源掠夺 | 抢夺稀缺资源 |
| 意识形态 | 文明理念冲突 |

**可复用场景：** 星际战争、文明存亡危机

---

## 二，技术类概念

### 4. 星际跃迁网络

**本质：** 连接各星际文明的超光速航行网络。

| 特性 | 描述 |
|------|------|
| 覆盖范围 | 连接整个已知宇宙 |
| 建造者 | 通常由高等文明建造 |
| 控制权 | 控制者拥有战略优势 |
| 维护成本 | 维护需要巨大资源 |

**可复用场景：** 星际航行基础设施、战略资源控制

---

### 5. 文明火种（末日保存技术）

**本质：** 文明面临毁灭时保存种族的技术手段。

| 方式 | 描述 |
|------|------|
| 星际飞船 | 携带基因库的移动家园 |
| 意识保存 | 将意识数字化保存 |
| 休眠技术 | 长时间休眠等待时机 |
| 分散部署 | 将人口分散到多个星球 |

**可复用场景：** 末日生存、文明延续设定

---

## 三，生物类概念

### 6. 星际种族多样性

**本质：** 宇宙中不同星际种族之间的生物学差异。

| 类型 | 特征 |
|------|------|
| 硅基生命 | 以硅为基础的肉体 |
| 能量生命 | 以能量形式存在的生命 |
| 机械生命 | 机械化后的有机生命 |
| 复合生命 | 有机与机械结合 |

**可复用场景：** 外星种族设计、多物种宇宙

---

### 7. 神族血统（高等文明遗传）

**本质：** 高等文明通过血统传承特殊能力。

| 特性 | 描述 |
|------|------|
| 能力遗传 | 战斗力、感知力等可遗传 |
| 血统纯度 | 纯度决定能力强度 |
| 稀释退化 | 血统稀释导致能力下降 |
| 纯化追求 | 种族追求血统纯化 |

**可复用场景：** 超人类设定、血统论设定
