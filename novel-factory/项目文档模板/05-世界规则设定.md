## 一、世界构造

| 字段 | 内容 |
|------|------|
| 宇宙结构 | 一重宇宙，多维度空间（各维度不互通） |
| 已知范围 | 银河系约100亿光年已知半径 |
| 文明分布 | 主要集中在银河系旋臂和核心区 |
| 维度关系 | 各维度空间独立不互通，维度跳跃技术已失传 |

## 二、世界层级

```
宇宙联盟（一级）
  └── 安理会常任理事国（二级）
        └── 普通成员国（三级）
              └── 低等种族/新晋文明（四级）
                    └── 原始文明/边缘种族（五级）
```

**说明：** 宇宙联盟是名义上的银河系最高权力机构，实际被仲裁者文明等高等种族把持。

## 三、时间法则

| 规则 | 说明 |
|------|------|
| 时间流 | 统一时间流，整个银河系时间同步 |
| 曲速航行 | 可缩短实际旅行时间，但不改变时间流 |
| 时间不可逆 | 时间回溯技术理论上存在，但代价极高且被联盟禁止 |
| 时间膨胀 | 接近光速航行时会产生相对时间膨胀（狭义相对论） |

---

## 四、空间法则

| 规则 | 说明 |
|------|------|
| 空间稳定 | 常规空间稳定，曲速/空间折叠不会破坏空间结构 |
| 亚光速航行 | 最高可达0.99倍光速，能量消耗巨大 |
| 曲速航行 | 通过扭曲空间实现超光速，V1级文明可用 |
| 空间折叠 | 直接跨越空间，V2级以上文明可用 |
| 虫洞跳跃 | 通过天然或人工虫洞实现瞬间跨越，不稳定 |

## 六、联盟三大禁忌

| 禁忌 | 内容 | 处罚 |
|------|------|------|
| 第一禁忌 | 禁止智能AI生命 | 立即处决+连坐种族 |
| 第二禁忌 | 禁止克隆人 | 强制绝育+种族降级 |
| 第三禁忌 | 低等种族可被压榨 | 不限规模压迫合法化 |

**说明：** 三大禁忌由万年前"阿尔佐姆AI暴动"引发。夏天拥有的明月量子AI生命是联盟最高通缉技术。

## 七、世界主要矛盾

### 核心矛盾

**人类作为低等种族被压迫、被当炮灰、被奴隶走私；夏天拥有禁忌AI技术却无法公开拯救人类。**

### 矛盾起源

万年前阿尔佐姆帝国AI暴动导致宇宙联盟建立，确立高等种族统治秩序，人类作为新兴文明被归为"低等种族"。

### 矛盾现状

- 人类是联盟正式成员但无实质话语权
- 被强制征召当炮灰参与虫族战争
- 被其他种族鄙视和压迫
- 夏天手握阿尔佐姆遗产却必须隐藏身份

### 矛盾发展趋势

```
暗中积蓄力量（1-3卷）
  → 身份曝光危机（3-4卷）
  → 最低谷（4卷）
  → 亮明身份（5卷）
  → 人类成为安理会第九常任理事国（6卷）
  → 建立银河新秩序（结局）
```
