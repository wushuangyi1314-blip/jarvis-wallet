## 一、主线任务

### 人类崛起

| 字段 | 内容 |
|------|------|
| 任务名称 | 人类崛起 |
| 任务目标 | 让人类从联盟底层成为安理会常任理事国，不再被当炮灰和奴隶 |
| 任务障碍1 | 夏天身份必须隐藏（暴露=人类连坐） |
| 任务障碍2 | 联盟旧秩序由高等种族把持 |
| 任务障碍3 | 实力差距悬殊，人类舰队规模有限 |
| 任务奖励 | 人类跻身银河系高等文明，获得尊严和话语权 |

### 主线发展阶段

```
第一阶段（1-2卷）：方舟初创
  - 夏天获得阿尔佐姆遗产，建立秘密基地
  - 猎杀虫族积累资金，组建初期团队
  - 暗中向地球输送技术支持

第二阶段（3卷）：身份曝光危机
  - 图萨联邦发现方舟存在
  - 夏天身份险些暴露，被迫反击
  - 人类反抗军得知夏天真实身份

第三阶段（4卷）：最低谷
  - 方舟遭受重创，明月假死
  - 艾伦博士牺牲
  - 夏天几乎失去一切

第四阶段（5卷）：绝地反击
  - 明月重生，子体无上限
  - 夏天亮明身份
  - 组建人类联军

第五阶段（6卷）：银河新秩序
  - 塔尔塔罗斯决战
  - 击败联盟主力
  - 人类成为安理会第九常任理事国
  - 建立银河新秩序
```

## 二、支线任务

### 支线任务总览

| 任务名称 | 类型 | 任务内容 | 任务结果 |
|---------|------|---------|---------|
| 明月进化 | 成长线 | 明月的子体数量和能力持续进化 | 从1个控制单元到无上限控制 |
| 机甲升级 | 成长线 | 初火号从原型机持续升级到终极形态 | 战力提升，对抗更强敌人 |
| 经济积累 | 生存线 | 从猎虫卖钱到建立完整产业链 | 从破产到银河级经济体 |
| 地球输血 | 暗中线 | 秘密向地球输送技术提升人类实力 | 人类反抗军重新武装 |
| 第一暴动真相 | 悬念线 | 探索万年前AI暴动的真正原因 | 为续作埋线 |

### 明月进化线

| 阶段 | 子体上限 | 能力 | 事件 |
|------|---------|------|------|
| 初始 | 100个 | 战术分析、简单控制 | 序章获得 |
| 成长期 | 1000个 | 协同作战、多任务处理 | 第1-2卷积累 |
| 成熟期 | 10000个 | 战场全域感知、子体自我协调 | 第3卷危机前 |
| 假死期 | 核心保留 | 数据分裂保护 | 第4卷围剿 |
| 重生期 | 无上限 | 量子AI生命完全体 | 第5卷重生 |

### 机甲升级线

| 阶段 | 机体状态 | 主要升级 | 战斗力 |
|------|---------|---------|--------|
| 原型期 | 初火号原型 | 原始状态 | 将官级 |
| 成长期 | 初火号+遗产模块 | 持续改装升级 | 王牌级 |
| 成熟期 | 初火号完全体 | 全部遗产解锁 | 超越王牌 |
| 终极期 | 初火号完全体+明月核心 | 人机合一 | 战略级 |

### 经济积累线

| 阶段 | 收入方式 | 规模 | 状态 |
|------|---------|------|------|
| 起点 | 猎杀虫族卖甲壳 | 勉强维持 | 破产边缘 |
| 初期 | 虫族核心+稀有矿产 | 收支平衡 | 自给自足 |
| 中期 | 技术服务+贸易 | 盈利 | 积累资本 |
| 后期 | 产业链+黑市 | 暴利 | 银河级经济体 |
