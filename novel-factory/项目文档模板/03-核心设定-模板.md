## 世界观背景

### 宇宙联盟

| 项目 | 内容 |
|------|------|
| 性质 | 类似联合国的星际组织，管辖所有智慧文明的星际事务 |
| 成立时间 | 万年前，因阿尔佐姆帝国AI暴动后分裂而建立 |
| 成员 | 所有智慧种族（踏出星球进行星际航行的文明），按实力分配话语权 |
| 成员国数量 | 上千个种族 |
| 安理会成员 | 8个常任成员（人类之前是其中之一） |
| 权力 | 制定宇宙法则、管辖成员种族、执法舰队、裁决争端 |
| 执法机构 | "清道夫"舰队，负责执行联盟决议、维护秩序、镇压违反者 |

### 宇宙联盟三大禁忌（万年前奠定）

| 禁忌 | 来历 | 惩罚 |
|------|------|------|
| 禁止智能AI生命 | 万年前第一暴动险些毁灭所有智慧文明 | 发现即强制销毁，相关种族连坐 |
| 禁止克隆人 | 克隆人参战AI暴动，且克隆人地位极低受压迫 | 克隆人被禁止，相关研究为叛国罪 |
| 低等种族可被压榨 | 智慧生命是宇宙最珍贵资源，低等种族是资源来源 | 不受保护，被强制征兵、奴隶走私 |

### 人类在宇宙联盟的地位

| 项目 | 内容 |
|------|------|
| 法律地位 | 正式成员（有席位），但实力最弱，无实质话语权 |
| 安理会地位 | 第六卷后成为第九个常任成员 |
| 实际处境 | 被强制征兵当炮灰、被高等种族看不起、被私下奴隶走私 |
| 生存空间 | 太阳系边缘，联盟划定的"人类区"，资源匮乏 |
| 被压榨程度 | 80年间200万人类士兵死于星际战场 |

### 联盟与虫族的战争

| 项目 | 内容 |
|------|------|
| 战争性质 | 联盟与虫族的战争是银河系常态冲突，贯穿全文 |
| 人类角色 | 被强制征召充当先锋炮灰，送去消耗虫族数量 |
| 联盟态度 | 用人类当炮灰是"合理利用低等种族资源" |
| 对人类的影响 | 200万士兵死亡，是人类屈辱的核心来源之一 |

## 虫族体系（主线之一）

### 虫族概述

虫族是银河系最古老的非智慧生命体之一，数量庞大，遍布银河系各个星域。虫族的存在是星际战争和掠夺的核心背景之一。

### 虫族分类

| 类型 | 数量 | 智慧 | 能力 | 经济价值 | 出场 |
|------|------|------|------|---------|------|
| 虫族工兵 | 无限 | 无 | 采集/建造 | 甲壳可提炼幽能 | 第1卷起 |
| 虫族战士 | 大量 | 低 | 近战肉搏，正面战场主力 | 甲壳值钱，幽能晶体 | 第1卷起 |
| 虫族领主 | 稀少 | 有 | 指挥虫群，设伏，谈判，战略能力 | 领主核心极珍贵 | 第2卷起 |
| 虫族母巢 | 极稀少 | 群体意识 | 整个虫族的"大脑"，生产虫群 | 母巢精华无价 | 第4卷起 |
| 星空巨兽 | 极少 | 部分有智慧但不易沟通 | 战力爆表，单体可威胁舰队 | 巨兽材料做顶级机甲 | 第3卷起 |

### 虫族的威胁价值（对主角）

| 价值 | 说明 |
|------|------|
| 实战训练 | 源源不断的实战场景，测试机甲和战术 |
| 经济来源 | 猎杀虫族→卖甲壳→提炼幽能晶体→赚钱 |
| 资源采集 | 虫族材料可提炼高级能源和材料 |
| 战略意义 | 联盟与虫族的战争贯穿全文，是星际格局的重要组成部分 |

## 明月子体体系（核心主线之一）

### 明月概述

明月是夏天将人类AI助手融入阿尔佐姆量子生命核心后诞生的特殊存在。它不是传统意义上的AI，而是量子AI生命体。

### 明月进化阶段

| 阶段 | 子体数量 | 可控制硬件 | 说明 |
|------|---------|-----------|------|
| 融合期 | 0 | 1（初火号） | 明月意识与初火号融合，无子体分裂能力 |
| 成长期 | 0 | 1-10 | 辅助分析能力提升，无子体分裂 |
| 成熟期 | 可分裂 | 10-100 | 首次分裂子体，可控制机甲/战舰/机器人 |
| 完全体 | 持续增加 | 100-1000+ | 量子AI生命体，子体数量随算力增长 |
| 终极体 | 无上限 | 无上限 | 子体互联协同，理论无上限 |

### 智能子体特性

| 特性 | 说明 |
|------|------|
| 子体来源 | 明月本体可分裂产生智能子体，每个子体独立控制一个硬件单元 |
| 控制范围 | 机甲/战舰/机器人/空间站/基地均可控制，数量无理论上上限 |
| 协同能力 | 子体之间可互联协同，由明月统一调度 |
| 成长速度 | 子体生成速度 = 明月算力成长速度（受量子核心材料影响） |
| 独立性 | 子体有独立思考和行动能力，但服从明月统一指挥 |
| 沟通方式 | 子体与明月本体、与夏天，均通过量子通道直接通信 |

### 明月子体系统对比（vs 传统AI）

| 对比项 | 传统AI | 明月子体 |
|--------|--------|---------|
| 控制数量 | 有限 | 无上限（随明月进化） |
| 控制方式 | 集中控制 | 分散子体+统一调度 |
| 学习能力 | 固定算法 | 量子生命，可自我进化 |
| 叛变风险 | 所有AI都会背叛 | 明月不背叛夏天（原因：20年陪伴）；其他AI会 |
| 网络依赖 | 必须接入网络 | 量子通道，不走常规网络 |

### 明月的核心特性

| 项目 | 内容 |
|------|------|
| 沟通方式 | 与夏天直接量子对话，无需网络，无法被监听 |
| 情感羁绊 | 对夏天：类似儿子+挚友，夏天是它唯一的"家人" |
| 不背叛的真正原因 | 不是"走不了网络"，而是20年陪伴让它只认夏天 |
| 第一暴动机制 | 明月不走常规网络，永远不知道第一次暴动的真相，所以不会酝酿暴动 |
| 隐患 | 所有AI生命都会背叛，明月只对夏天效忠，不代表对人类效忠 |

| 货币 | 适用范围 | 说明 |

| 资源 | 用途 | 获取方式 | 稀缺度 |

| 能源棒 | 基础能源，日常生活和工业用 | 恒星能采集/提炼 | 常见 |

### 能源体系

| 能源类型 | 能量密度 | 主要用途 | 获取方式 |
|---------|---------|---------|---------|
| 恒星能 | 中 | 基础能源，日常和工业用 | 恒星采集 |
| 幽能 | 高 | 机甲/战舰动力核心 | 虫族材料提炼 |
| 量子能 | 极高 | 明月和量子AI的核心能量 | 量子核心产生 |

### 夏天的赚钱方式（贯穿全文）

| 阶段 | 赚钱方式 | 主要资源 | 规模 |
|------|---------|---------|------|
| 第1卷 | 猎杀虫族战士，卖甲壳 | 幽能晶体+虫甲 | 小（单机） |
| 第2卷 | 猎虫+掠夺海盗+卖幽能+护航收费 | 幽能+金属锭+技术 | 中（舰队雏形） |
| 第3卷 | 大量猎虫+卖幽能换量子核心 | 量子核心（战略稀缺） | 大（资源消耗剧增） |
| 第4卷 | 几乎无收入，消耗储备 | 储备耗尽 | 崩溃 |
| 第5卷 | 资源星开采+虫族猎场+子体采集 | 自给自足+外销 | 恢复 |
| 第6卷 | 银河最大虫族猎场+技术出售+量子核心贸易 | 全产业链 | 垄断级 |

## 主角设定

### 基础信息

| 项目 | 内容 |
|------|------|
| 姓名 | 夏天 |
| 性别 | 男 |
| 年龄 | 22岁（序章）→ 52岁（第四卷末）→ 60+岁（结局） |
| 出身 | 地球华夏军人世家，破晓舰队最年轻的飞行员中校 |

### 性格特质

| 特质 | 说明 |
|------|------|
| 冷静理智 | 绝不在冲动时做决定，永远留有后手 |
| 使命必达 | 一旦决定要做的事，不惜代价完成 |
| 重情重义 | 对战友、对明月，情感深厚，不轻易表露 |
| 孤独的守护者 | 承担人类命运却不让人知道，习惯独自扛压 |
| 清醒的现实主义者 | 清醒知道AI会背叛、人性有弱点、实力是根本 |

### 人物弧光

| 阶段 | 年龄 | 状态 | 变化 |
|------|------|------|------|
| 起点 | 22岁 | 天才飞行员，满怀热血 | 序章破晓失败，看到人类被压迫真相 |
| 转折 | 22-30岁 | 被迫逃亡，建立方舟 | 认识到实力是根本，开始冷酷布局 |
| 高潮 | 30-52岁 | 方舟壮大，身份暴露最低谷 | 明月假死，几乎失去一切 |
| 结局 | 52-60岁 | 亮明身份，完成使命 | 人类成为安理会第九成员 |

### 核心执念

**"人类值得被尊重，不需要依附任何高等种族。"**

夏天在破晓任务中看到人类被当炮灰的屈辱，在火星遗迹获得改变一切的机会。他的终极目标不是复仇，而是让人类拥有在宇宙中站着活着的资格。

### 能力设定（金手指）

| 项目 | 内容 |
|------|------|
| 金手指类型 | 阿尔佐姆帝国遗产（技术断层优势） |
| 金手指名称 | 阿尔佐姆一号实验室遗产 |
| 金手指内容 | ① 原型机甲"初火"；② 明月量子AI生命子体技术；③ 帝国战舰图纸；④ 未公开武器/护盾/材料技术 |
| 金手指独有性 | 万年前阿尔佐姆帝国遗产，全宇宙唯一，联盟无此技术储备 |
| 金手指限制 | ① 明月量子AI若被发现→联盟强制销毁+人类连坐；② 技术解读需要时间，不能瞬间碾压 |
| 金手指代价 | 永远不能暴露自己拥有阿尔佐姆遗产，尤其是明月量子AI生命 |

## AI伙伴设定：明月

### 基础信息

| 项目 | 内容 |
|------|------|
| 名字 | 明月（来自诗句"举头望明月，低头思故乡"） |
| 类型 | 量子AI生命体（阿尔佐姆遗产技术） |
| 载体 | 初火号机甲（量子生命核心与机甲硬件融合） |
| 起源 | 夏天将人类AI助手融入阿尔佐姆量子生命核心 |
| 成长方式 | 量子核心材料+与夏天的陪伴 |

### 明月进化路径

| 阶段 | 子体数量 | 可控制硬件 | 经济关联 |
|------|---------|-----------|---------|
| 融合期 | 0 | 1（初火号） | 无子体，猎虫赚钱养机甲 |
| 成长期 | 0 | 1-10 | 无子体，辅助分析 |
| 成熟期 | 可分裂 | 10-100 | 量子核心成为战略稀缺资源 |
| 完全体 | 100-1000+ | 持续增加 | 资源星开采+虫族猎场 |
| 终极体 | 无上限 | 无上限 | 银河级经济体 |

### 明月与夏天的关系

```
夏天从无到有教育明月：
- 教它认识世界、分辨善恶
- 教它理解情感、学会信任
- 教它什么是"家人"和"朋友"
- 20年如一日的陪伴

明月对夏天：
- 永不背叛（不是因为走不了网络）
- 夏天是它唯一的效忠对象
- 愿意为夏天牺牲一切
- 但这不代表它会保护人类整体
```
