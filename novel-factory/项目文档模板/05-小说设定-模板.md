## 世界规则设定

| 字段 | 内容 |
|------|------|
| 世界构造 | 一重宇宙，多维度空间（各维度不互通），银河系约100亿光年已知半径 |
| 世界层级 | 宇宙联盟（一级）→ 安理会常任理事国（二级）→ 普通成员国（三级）→ 低等种族（四级）→ 原始文明（五级） |
| 时间法则 | 统一时间流，曲速航行可缩短实际旅行时间，时间不可逆 |
| 空间法则 | 空间稳定，亚光速航行+曲速航行，虫洞可实现空间跳跃 |
| 力量上限 | 阿尔佐姆帝国时期的技术极限（量子AI生命+曲速引擎+维度武器），现代最高为仲裁者级文明 |
| 其他规则 | 宇宙联盟三大禁忌：禁止智能AI生命、禁止克隆人、低等种族可被压榨 |

## 世界矛盾设定

| 字段 | 内容 |
|------|------|
| 世界主要矛盾 | 人类作为低等种族被压迫、被当炮灰、被奴隶走私；夏天拥有禁忌AI技术却无法公开拯救人类 |
| 矛盾起源 | 万年前阿尔佐姆帝国AI暴动导致宇宙联盟建立，确立高等种族统治秩序，人类作为新兴文明被归为"低等种族" |
| 矛盾现状 | 人类是联盟正式成员但无实质话语权，被强制征召当炮灰，被其他种族鄙视，夏天手握阿尔佐姆遗产却必须隐藏 |
| 矛盾发展趋势 | 夏天从暗中积蓄力量到最终亮明身份，人类从炮灰到安理会第九常任理事国，建立银河新秩序 |

## 力量体系设定

| 字段 | 内容 |
|------|------|
| 力量类型 | 机甲战力（个人级）、舰队战力（集体级）、明月子体（无人上限）、科技武器（星系级）、阿尔佐姆遗产（超越现代科技的禁忌技术） |
| 力量来源 | 机甲→阿尔佐姆遗产技术；舰队→工业制造；明月子体→量子AI生命；科技→联盟主流技术+阿尔佐姆遗产 |
| 修炼方式 | 机甲驾驶技术（操控能力）、精神力同步（与明月/机甲的契合度）、科技知识（技术解读能力） |
| 核心力量 | 阿尔佐姆遗产（量子AI生命+原型机甲+帝国战舰图纸）——凌驾于一切现有力量体系的断层优势 |
| 高层战力特征 | 仲裁者级文明可碾压恒星；单体战机甲可对抗整支舰队；明月终极体子体无上限可控制整个舰队；阿尔佐姆遗产领先现代科技200-500年 |

#### 机甲制造流程（完整工艺链）

```
【第一阶段：材料开采】
1. 矿物勘探 → 卫星扫描定位矿脉 → 地质分析评估价值
2. 无人采矿机/有人采矿站 → 岩石破碎 → 矿物分离 → 粗矿运输

【第二阶段：材料提炼】
3. 粗矿 → 熔炼厂（去除杂质） → 化学处理（提纯） → 精炼厂（成品）
   示例：铁矿石 → 熔炼生铁 → 精炼低碳钢/高碳钢/合金钢
4. 特殊材料合成：
   - 纳米合金：多种金属精确配比 → 高温熔炼 → 纳米分散 → 热等静压
   - 振金合金：振金矿+精密合金 → 特殊熔炼工艺 → 振金合金锭
   - 幽能合金：幽能晶体+金属 → 能量嵌入工艺 → 幽能合金

【第三阶段：零件制造】
5. 零件分类：
   - 结构件（骨架、外壳）：加工难度低，精度中
   - 动力件（引擎、能量核心）：加工难度高，精度高
   - 传动件（关节马达）：加工难度高，精度高
   - 电子件（量子芯片、传感器）：加工难度极高，精度极高
   - 武器件（光束炮、导弹发射架）：加工难度高，精度高
6. 精密零件制造流程：
   硅晶圆切割 → 量子蚀刻线路 → 量子点植入 → 幽能通道刻蚀 → 多层堆叠 → 封装测试 → 品质分级（A/B/C/D级）

【第四阶段：机甲组装】
7. 骨架制造：碳纤维骨架管 → 切割成型 → 关节连接件安装 → 骨架组装 → 强度测试
8. 动力系统安装：能量核心安装 → 动力管线铺设 → 冷却系统安装 → 动力分配器连接 → 动力系统测试
9. 装甲安装：内部线路铺设 → 传感器安装 → 装甲板安装 → 密封处理 → 装甲强度测试
10. 武器系统集成：武器挂载点安装 → 武器系统连接 → 射击控制校准 → 武器系统测试
11. 系统整合：所有系统连接 → AI核心（明月）安装 → 整体系统调试 → 同步率测试 → 实战模拟测试
12. 交付验收：环境适应测试 → 极限测试 → 交付文档 → 正式列装

【特殊材料说明】
- 振金：崩塌恒星残骸中提取，极耐高温/抗打击，是王牌机甲装甲最优材料
- 幽能晶体：虫族体内提取，高能量密度，用于机甲动力核心和武器充能
- 量子晶体：晶石矿+幽能合成，用于AI核心、量子计算、通讯
```
