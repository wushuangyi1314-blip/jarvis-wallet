# 第一部分：文明等级体系

## 宇宙文明六级分级

宇宙联盟对所有智慧文明按综合实力进行分级，是话语权分配、贸易规则、军事义务的核心依据。

| 等级 | 名称 | 标准 | 典型代表 | 话语权 |
|------|------|------|---------|--------|
| 一级 | 霸主级 | 掌控银河系核心区域，拥有星系级武器，可单独对抗整个联盟 | 仲裁者文明 | 安理会常任，拥有否决权 |
| 二级 | 强权级 | 跨星系领地，舰队规模超过十万艘，科技领先多数成员 | 维拉图斯帝国 | 安理会常任 |
| 三级 | 领主级 | 拥有多个恒星系，舰队规模万艘以上，在区域有主导权 | 图尔萨联邦、银狮族 | 安理会非常任 |
| 四级 | 独立级 | 拥有单个恒星系或大型殖民地，舰队千艘以上 | 艾尔文明、人类（现状） | 联盟普通成员 |
| 五级 | 附属级 | 殖民地或小型领地，舰队百艘以下，受高等文明保护 | 部分蛮荒星域小文明 | 附庸地位 |
| 六级 | 原始级 | 未掌握星际航行技术，仅限本星球 | 部分边缘星球土著 | 不被联盟承认 |

**人类文明现状：四级（独立级）→ 目标：三级（领主级）+ 安理会常任**

| 形状 | 六面体晶体，大小约为人类拇指指甲盖 |
| 颜色 | 根据能量密度从无色到深蓝色不等 |
| 等级 | D级（无色）→ C级（淡蓝）→ B级（蓝色）→ A级（深蓝）→ S级（紫蓝） |

- 宇宙晶的能量可直接转化为任何形式的能源
- 所有高等文明的能源核心都依赖宇宙晶驱动
- 宇宙晶的开采量有限，形成天然货币属性
- 联盟央行（仲裁者文明控制）持有最大宇宙晶储备，通过买卖调节经济

# 第四部分：技术工艺体系

## 4.1 矿产资源分类与开采

### 4.1.1 主要矿产资源

| 矿产 | 分布 | 开采难度 | 主要用途 |
|------|------|---------|---------|
| 铁镍矿 | 小行星带、岩石行星 | 低 | 金属锭原料 |
| 稀土矿 | 行星地壳深层 | 中 | 精密元件材料 |
| 冰矿 | 冰行星、彗星 | 低 | 氢燃料+水资源 |
| 晶石矿 | 小行星带、高重力行星 | 中 | 能量存储元件 |
| 振金矿 | 崩塌恒星残骸 | 极高 | 高级装甲材料 |
| 幽能矿 | 虫族巢穴、特定星域 | 高 | 幽能晶体提炼 |

### 4.1.2 开采流程

**第一阶段：勘探**
```
矿物探测卫星 → 扫描星球表面 → 定位矿脉 → 地质分析 → 评估开采价值
```

**第二阶段：采矿**
```
无人采矿机/有人采矿站 → 岩石破碎 → 矿物分离 → 粗矿运输
```

**第三阶段：提炼**
```
粗矿 → 熔炼炉（初步分离） → 化学处理（提纯） → 精炼厂（最终成品）
```

**典型案例：金属锭生产流程**
```
铁矿石开采 → 熔炼厂（去除杂质，得到生铁） → 精炼厂（加入合金元素，得到精炼金属锭） → 质量检测 → 入库
```

|------|------|------|------|

### 4.2.3 材料加工流程

**机甲装甲加工流程（以纳米合金为例）：**
```
1. 材料配比：多种金属按精确比例混合
2. 高温熔炼：在真空炉中加热至3000°C以上
3. 纳米分散：将合金冷却后进行纳米级粉碎
4. 热等静压：用高压将纳米粉末压实
5. 表面处理：涂层、硬化、回火
6. 质量检测：强度测试、成分分析
7. 成型加工：切割、弯曲、焊接成装甲板
```

## 4.4 装备组装与制造

### 4.4.1 机甲制造流程

```

第二阶段：动力系统安装

第三阶段：装甲安装
内部线路铺设 → 传感器安装 → 装甲板安装 → 密封处理 → 装甲强度测试

第四阶段：武器系统集成
武器挂载点安装 → 武器系统连接 → 射击控制校准 → 武器系统测试

第五阶段：系统整合
所有系统连接 → AI核心安装 → 整体系统调试 → 同步率测试 → 实战模拟测试

第六阶段：交付验收
环境适应测试 → 极限测试 → 交付文档 → 正式列装
```

### 4.4.2 战舰制造流程

```

# 第五部分：资源与能源体系

## 5.1 能源分类

| 能源类型 | 能量密度 | 获取难度 | 主要用途 | 代表技术 |
|---------|---------|---------|---------|---------|

| 幽能 | 高 | 高 | 机甲动力、高级武器 | 幽能晶体 |
| 恒星能 | 中 | 低 | 太空基地、工业供能 | 太阳能板 |
| 量子能 | 极高 | 极高 | AI核心、高级装备 | 量子核心 |

## 5.2 幽能体系详解

### 5.2.1 幽能晶体形成原理

幽能是虫族生命活动的副产物，在虫族体内以晶体形式储存。虫族通过某种未知机制将恒星能量转化为幽能，储存在体内的幽能腺中。

```
恒星能量 → 虫族吸收 → 幽能腺转化 → 幽能晶体储存 → 虫族死亡后晶体残留
```

### 5.2.2 幽能应用

| 应用领域 | 具体用途 | 技术要求 |
|---------|---------|---------|
| 机甲动力 | 为机甲提供高密度能源 | 幽能晶体+能量转换器 |
| 战舰引擎 | 战舰曲速/亚光速航行 | 大量幽能+引擎适配 |
| 武器充能 | 高能武器的能量来源 | 幽能直接注入 |
| 护盾供能 | 护盾发生器的能量来源 | 稳定能量供应 |

### 5.2.3 幽能市场

- **主要产地**：虫族活跃星域，尤其是虫族母巢附近
- **主要买家**：图尔萨联邦、维拉图斯帝国、各大舰队
- **价格波动**：与虫族战争烈度正相关，战争期间价格飙升
- **夏天的商业模式**：猎杀虫族→提取幽能晶体→出售给中间商或直接用户

## 5.3 量子核心体系

### 5.3.1 量子核心用途

量子核心是明月子体生成的关键材料，每个子体需要一个量子核心作为算力载体。

### 5.3.2 量子核心规格

| 规格 | 算力 | 价格 | 适用场景 |
|------|------|------|---------|
| 民用级 | 低 | 1000 B级宇宙晶 | 商业AI、普通舰载AI |
| 军用级 | 中 | 50000 B级宇宙晶 | 机甲AI、战术系统 |
| 战略级 | 高 | 500000 A级宇宙晶 | 旗舰AI、指挥系统 |
| 禁忌级 | 极高 | 非卖品 | 量子AI生命（明月级别） |

**夏天获取量子核心的方式：**
1. 艾伦博士留下储备（早期主要来源）
2. 缴获联邦战舰时获取
3. 攻破敌方据点时获取
4. 蛮荒星域黑市购买（价格极高）
5. 第5卷后：建立自己的量子核心生产线

## 6.1 联盟权力结构

```
银河议会（最高权力机构）
    ├── 安理会（日常最高决策）

## 6.3 联盟经济体系

### 6.3.1 关税同盟

联盟成员间贸易实行零关税，但向非成员国征收高额关税。人类作为成员享受零关税待遇（理论上）。

### 6.3.2 资源配额

联盟对稀缺资源实行配额管理，主要由常任理事国控制分配权。

### 6.3.3 货币发行

宇宙晶开采由联盟央行（仲裁者文明控制）垄断，发行量由安理会决定。

## 6.4 联盟军事体系

### 6.4.1 清道夫舰队

| 项目 | 内容 |
|------|------|
| 性质 | 联盟执法舰队，执行联盟决议 |
| 指挥权 | 安理会直辖，常任理事国共同管理 |
| 主力构成 | 图尔萨联邦提供40%，维拉图斯帝国提供30%，其他成员提供30% |
| 主要任务 | 镇压违反者、维持秩序、打击星际海盗、处理虫族威胁 |

### 6.4.2 联合作战机制

当重大威胁出现时，安理会可授权组建联合舰队，各成员国按比例提供兵力。
