## 1.1 联盟第二代标准机甲——骑士II型（Knight-II）

**基本信息**

| 项目 | 内容 |
|------|------|
| 代际 | 第二代（标准型） |
| 研发时间 | 约公元2180年代 |
| 研发国家 | 图尔萨联邦 |
| 设计用途 | 全域通用型，主力制式装备 |
| 驾驶方式 | 神经链接（手动备份） |
| 装甲 | 复合装甲（内层合金钢+外层陶瓷复合材料），总厚度80-120mm |
| 自重 | 75吨 |
| 高度 | 7.2米 |
| 动力系统 | 小型核聚变反应堆，输出功率约2500千马力 |
| 航速 | 公路模式120km/h，越野模式75km/h，可短暂悬浮（悬停高度5米，持续30秒） |
| 主武器 | 重联装30mm光束炮×1（备弹500发或能量模块×3） |
| 副武器 | 四联装反载具导弹×1（备弹12发，可选装燃烧弹/穿甲弹/延时弹头） |
| 防御武器 | 速射近防炮×2（拦截来袭导弹/无人机）、战术烟雾发射器×6 |
| 典型作战半径 | 300km（内置燃料）/外挂副油箱可达800km |
| 乘员 | 1人（驾驶员，神经链接同步率通常30-60%） |
| 产量 | 超过50万台 |
| 改进型号 | Knight-IIA（装甲增强型）/ Knight-IIL（轻型化改造）/ Knight-IIM（海军型，深潜加强） |
| 评价 | 联盟产量最大的机甲，综合性能均衡 |

**外观描述**

骑士II是联盟工业美学的标杆产物，整体身形修长匀称，高宽比约为3:1，给人一种干练冷峻的印象。其高度7.2米在第二代机甲中属于中等水平，不会显得笨重，也不会过于单薄。头部采用经典的"单眼"设计，一枚大型多光谱传感器居中嵌入黑色面罩式头盔，面罩两侧留有流线型散热槽道，传感器可在夜间或复杂电磁环境下提供1200米内的精确目标识别。躯干线条硬朗平直，胸腔部位凸起一块棱角分明的装甲块，内置核聚变反应堆核心，呼吸灯在激活时会从蓝绿渐变为明黄。肩甲外侧刻有图尔萨联邦的鹰徽纹章，左肩为生产批次编号，右肩为驾驶员所属部队编号。肢体造型遵循功能主义，关节处外露的双层液压杆清晰可见，膝盖与手肘的护甲呈斜切式装甲板，兼顾防护与运动灵活性。配色以深灰与铁黑为主色调，联邦标志性的钴蓝色作为细节点缀（传感器环、武器挂点、腿部推力口）；膝部与肩甲边缘贴有反光橙色警示条，在友军识别系统中显示为标准联盟色。辨识特征：背部两块矩形散热板呈"八"字形展开，是骑士II最容易辨认的剪影；驾驶舱舱门位于左胸，紧急时会弹射红色照明弹。

**科技等级**：TL-2（第二代成熟技术） | **材料等级**：ML-1（成熟合金钢+陶瓷复合材料）

**改装适配性**：

- **推荐改装模块**：装甲强化模块（+20mm复合装甲）、悬挂系统升级（提升越野速度15%）、电子战干扰吊舱（对抗雷达制导武器）、副油箱扩容组件
- **改装限制**：动力舱改造需TL-2以上认证，自行改装核聚变反应堆可能导致输出不稳定；主武器接口标准化，任何30mm光束炮均可兼容，但更换更大口径武器需改装动力匹配
- **改装后性能变化**：装甲强化型（Knight-IIA）正面防护提升25%，但机动性下降8%；轻型改造（Knight-IIL）机动性提升20%，防护下降15%；海军型（Knight-IIM）增加深潜密封涂层，可在水下200米作战，但全重增加2吨

|------|------|

| 研发时间 | 约公元2220年代 |

| 副武器 | 六联装智能导弹×1（可编程弹道，绕障碍物攻击，备弹36发） |
| 特殊武器 | "裁决者"超重型穿甲剑×1（近战用，振金合金刃，可切割第三代机甲装甲） |
| 防御武器 | 强化型近防系统×4、全谱干扰器×1（光学/雷达/红外全频段干扰）、主动拦截系统×1 |
| 典型作战半径 | 400km（内置）/外挂副油箱可达1000km |
| 乘员 | 1人（需经过特种部队选拔，同步率潜力测试必须达到60%以上） |
| 产量 | 约1.5万台 |
| 评价 | 图尔萨联邦精锐部队标志性装备 |

**外观描述**

仲裁者III是图尔萨联邦精锐部队的象征，其外观设计处处透露着"猎杀者"的压迫感。机甲整体身形修长而紧致，比骑士II高30厘米却只重4吨，大量采用倾斜装甲和内嵌式设计来削减视觉体积。头部造型最为独特——采用双传感器"双眼"布局，左右各一枚长条形红外/紫外复合传感器镶嵌在收窄的面甲中段，面甲上半部分覆盖一块弧形防弹玻璃保护驾驶员视野，两块传感器之间有一条细细的蓝色光源线，激活时会亮起表示锁定状态。躯干设计强调棱角与层次，胸腔部位向前突出形成楔形结构，内嵌的主武器重型光束炮炮管从胸前中央直接延伸出来，炮口位于驾驶舱下方，这一设计使主武器无需旋转即可攻击正前方大角度范围的目标，代价是牺牲了部分正面防护。肩甲宽大外张，呈倒梯形，左肩印有特种部队的黑色闪电徽章，右肩为仲裁者系列的银色天平徽章。肢体线条流畅，腿部装甲板沿纵向分割为三段式结构，大腿根部关节处有明显的液压加粗结构以承载22吨重量。配色以哑光黑为底色，关键装甲边缘和武器挂点使用暗金色勾边，传感器光源和推力口为深红色（与骑士II的橙色形成对比，表示精锐级身份），背部中央一块金色纹章板刻有驾驶员专属的斩首行动编号。辨识特征：胸前延伸的重型光束炮炮管和双眼前置布局是仲裁者III最显著的特征；"裁决者"穿甲剑平时折叠挂于右腰后侧。

**科技等级**：TL-3（第三代尖端技术） | **材料等级**：ML-2（纳米合金复合材料）

**改装适配性**：

- **推荐改装模块**：幽能辅助模块扩容（提升悬浮持续时间至90秒）、AI决策系统强化（缩短目标锁定时间30%）、装甲纳米自愈涂层（轻微损伤可在战斗中自动修复）、热能干扰弹（对抗红外制导）
- **改装限制**：振金武器接口需要TL-3以上认证，且仅限图尔萨联邦授权厂商；驾驶舱改造需重新校准神经链接接口；幽能模块改装后需进行为期一周的能量稳定测试
- **改装后性能变化**：幽能扩容型悬浮时间提升50%，但反应堆负载增加导致航速下降5%；AI强化型锁定时间缩短30%，但AI辅助决策系统在强电磁干扰下可能延迟；装甲自愈涂层可将20mm以下穿甲伤害在60秒内修复，但无法修复被贯穿的核心损伤

使徒IV是图尔萨联邦与艾尔文明技术结晶的巅峰之作，其外观设计融合了人类的对称美学与艾尔人的流线型偏好，整体呈现出一种庄严而危险的美感。机甲身形修长挺拔，7.8米的高度在第四代机甲中属于高挑类型，宽肩窄腰的造型让人联想到古典神话中的天使轮廓。头部设计最为惊艳——采用全息投影头盔，头盔外壳为半透明的淡金色材质，内部嵌入的传感器阵列可在透明与完全遮蔽两种模式间切换，激活时头盔表面会浮现艾尔文明特有的能量纹路，呈放射状从额头向脑后延伸，如同神圣的光环。躯干采用经典的人形躯干设计，但线条比前三代更加圆润流畅，几乎看不到直角过渡；胸腔部位微微鼓起，内置的幽能核心在装甲缝隙间透出幽蓝色脉动光芒，如同拥有心跳。肩甲呈羽翼状展开，左右各三片翼型装甲板由振金合金打造，从肩部向后斜上方延伸，静止时如同收起的天使翅膀，展开时可为背部提供额外防护。肢体修长优美，关节处覆盖着流线型的活动护甲，小腿与前臂的装甲板边缘镶嵌着细密的能量导流纹路，在全功率输出时会泛起淡蓝色光芒。配色以纯白为主色（区别于前三代的黑灰色调），翼状肩甲边缘与膝关节护甲为淡金色，幽能核心光源与武器系统为幽蓝色，背部翼甲连接处刻有驾驶员授予的专属徽章纹样。辨识特征：羽翼状肩甲与全息投影头盔是使徒IV最标志性的外观特征；激活"天使"光剑时，头盔的能量纹路会从淡金色变为明亮的幽蓝色。

**科技等级**：TL-4（第四代王牌技术） | **材料等级**：ML-3（振金合金+纳米内衬）

**改装适配性**：

- **推荐改装模块**：幽能核心超频模块（输出功率提升20%，但护盾冷却时间延长50%）、"天使"光剑刃长延伸模块（最长可延至3.5米）、副武器导弹巢扩容（+32发备弹）、指挥系统强化（增加通讯距离与抗干扰能力）
- **改装限制**：振金武器接口标准化但改装需联邦最高技术认证；幽能核心任何未经授权的改装均可能触发安全锁导致整机停机；"天使"光剑变形机构涉及空间折叠原理，仅限艾尔文明原厂技术人员操作
- **改装后性能变化**：幽能超频型持续输出提升20%，但单次悬浮时间从5分钟降至3分钟，且连续高功率输出后需要15分钟冷却；光剑刃长延伸后近战攻击范围增加40%，但挥剑速度下降15%；指挥型改装增加战场信息中继功能，通讯距离提升300%，但全重增加1.5吨，机动性略有下降

| 研发时间 | 约公元2220年代 |
| 研发国家 | 维拉图斯帝国 |
| 设计用途 | 要塞防御、重型突击 |
| 驾驶方式 | 神经链接+维拉图斯特有外骨骼增强系统 |
| 装甲 | 维拉图斯特有液态金属装甲（自愈功能），总厚度200-250mm |
| 自重 | 185吨 |
| 高度 | 9.2米 |
| 动力系统 | 大型核聚变反应堆+双辅助引擎，输出功率约12000千马力 |
| 航速 | 公路模式60km/h，越野模式35km/h |

**外观描述**

铁壁III-H是维拉图斯帝国工程哲学的极端体现——它用吨位碾压一切美学考量。机甲身形低矮宽厚，9.2米的高度却拥有55吨的惊人重量，宽高比接近1:1，活像一座移动的钢铁堡垒。整体造型极度棱角化，几乎看不到曲线或弧度，所有装甲板块均采用斜面设计以最大化弹开攻击的角度。头部设计简洁到近乎原始——一块扁平的楔形装甲覆盖在驾驶舱上方，装甲中央只开了一道横向细缝作为广角视觉传感器，传感器两侧是维拉图斯特有的生物式复眼结构，可在低光环境下提供热成像视觉；头顶四角各有一根粗壮的液压支柱，用于支撑正面冲击时的巨大惯性。躯干是铁壁III-H最壮观的部分——胸腔正面覆盖着一整块厚度达250mm的倾斜式液态金属装甲板，装甲板上布满了密密麻麻的微型注料孔，这些注料孔会在受损时释放液态金属填补缺口；躯干两侧各有三个外挂式护盾模块挂点，挂载的六边形护盾模块可根据战场情况更换（反光束型/反导弹型/反物理型）。肩甲极其宽大，横向宽度超过躯干，几乎与腿部平齐，形成一个巨大的"工"字形剪影，为内部驾驶舱和武器系统提供遮蔽。肢体粗壮敦实，大腿直径超过1.5米，膝关节采用双枢轴结构以支撑55吨自重，每条小腿侧面都嵌有三块可拆卸的追加装甲板。配色以深褐色和铁灰色为主色调，装甲接缝处填充暗红色密封胶（维拉图斯帝国标志色），液态金属在自愈状态下会泛出金属银色的光泽；外挂护盾模块边框为深紫色，表示其为能量防御装备。辨识特征：超宽肩甲与外挂式六边形护盾模块形成的"工"字形剪影是铁壁III-H最显著的特征；从正面看，两门双联装粒子炮的炮口如同两只凝视的眼睛，配合扁平的头部设计，极具压迫感。

**科技等级**：TL-3（第三代重装防御技术） | **材料等级**：ML-2（液态金属复合材料）

**外观描述**

初火号是阿尔佐姆帝国万年前技术遗产的结晶，也是夏天最重要的战斗伙伴。其外观设计完美诠释了阿尔佐姆帝国"以小搏大"的设计哲学——6.2米的身形是所有核心机甲中最矮的，15吨的自重甚至比某些第二代机甲还轻，但内部蕴含的技术密度远超任何现代机甲。整体身形紧凑浑圆，轮廓线条极为流畅，几乎找不到任何棱角或平面过渡，宛如一块被精心打磨的宇宙宝石。头部设计最具辨识度——采用半球形全息头盔结构，头盔外壳为淡金色半透明材质，内部传感器光源透过外壳形成柔和的暖黄色光芒，在完全激活状态下光芒会变为炽烈的橙红色，如同头盔内部燃烧着火焰；头盔两侧向后延伸出两片新月形的能量散热翼，散热翼表面刻有阿尔佐姆帝国特有的火焰纹图腾，与夏天方舟的火焰标志形成呼应。躯干圆润饱满，胸腔中央镶嵌着一颗六边形的蓝紫色宝石——这是阿尔佐姆量子能核心的外显形态，宝石内部可见缓缓流动的能量光纹，在夏天与明月同步率提升后会逐渐与驾驶舱的光效同步脉动；躯干两侧没有任何突出的武器挂点，所有武器接口均通过磁场吸附隐藏在装甲下方。肢体设计高度集成化——传统机甲的外露关节、液压杆和散热片在初火号上几乎看不到，四肢表面覆盖着完整的流线型振金装甲板，仅在关节内侧留有细密的能量循环通道；小腿部位向内收敛形成优雅的锥形结构，脚部则为类人形的标准足形但尺寸略小。配色以暖白色为底色（区别于人类机甲的黑灰色系），全身覆盖着淡淡的香槟金色渐变，关节和能量通道处为明亮的橙红色；在量子跃迁激活前夕，整台机甲会泛起一层流动的蓝紫色光晕，如同被星光笼罩。辨识特征：半球形全息头盔与六边形蓝紫色量子核心宝石的组合是初火号独一无二的视觉签名；在完全
