## 1.1 骑士II型 Knight-II（联盟第二代标准机）

| 项目 | 内容 |
|------|------|
| **代际** | TL-2（科技1级） |
| **研发时间** | 约公元2180年代 |
| **研发国家** | 图尔萨联邦 |
| **设计用途** | 全域通用型，主力制式装备 |
| **功能定位** | 平衡型 |
| **驾驶方式** | 神经链接（手动备份） |
| **装甲** | 复合装甲（内层合金钢+外层陶瓷复合材料），总厚度80-120mm |
| **自重** | 75吨 |
| **高度** | 7.2米 |
| **动力系统** | 小型核聚变反应堆，输出功率约2500千马力 |
| **制式武器** | 30mm光束炮×1（功率500kW，射程8km）；四联装导弹×1（射程10km，多目标攻击） |
| **航速** | 120km/h（公路）/ 75km/h（越野） |
| **产量** | 50万+台 |

**设计背景：**
骑士II是图尔萨联邦统一后的第一种标准化机甲，旨在替换各国原有的杂乱机甲体系。设计理念是"够用就好"——不在任何单一性能上追求极致，而是追求整体平衡和可维护性。联邦采用模块化设计，允许各盟国根据自身需求进行轻度改装，因此骑士II的实际装备型号多达数十种变型。

**外星设计元素：**
无（纯人类设计）

**改装潜力综合评价：**
| 项目 | 评价 | 说明 |
|------|------|------|
| 改装上限 | ML-2 | 动力舱改造需TL-2认证 |
| 推荐改装 | 装甲强化、悬挂升级、电子战干扰吊舱 | 联邦制式接口，改装件通用性强 |
| 改装后变体 | Knight-IIA（防护+25%，机动-8%）；Knight-IIL（机动+20%，防护-15%） | 联邦授权改装型号 |
| 改装限制 | 核聚变反应堆不可私自更换，更大口径武器需动力匹配 | 安全限制 |

**外观描述：**
骑士II是联盟工业美学标杆，整体身形修长匀称，高宽比约3:1。头部采用"单眼"设计，大型多光谱传感器居中嵌入黑色面罩式头盔；躯干线条硬朗，胸腔部位内置核聚变反应堆，呼吸灯在激活时由蓝绿渐变为明黄；肩甲外侧刻有图尔萨联邦鹰徽纹章；膝部与手肘护甲呈斜切式装甲板；配色以深灰与铁黑为主，联邦钴蓝色作为细节点缀。辨识特征：背部两块矩形散热板呈"八"字形展开。

| **驾驶方式** | 神经链接+AI辅助瞄准 |

**设计背景：**
仲裁者III是联邦为应对虫族威胁而研发的王牌机甲，研发初期参考了部分艾尔文明的轻量化设计理念。命名"仲裁者"象征其作为人类与虫族战争中的裁决者角色。联邦高层最初将其定位为限量生产的高端机，但在虫族战争后期因损失惨重而被迫扩大生产，目前产量已达最初的15倍。

**外星设计元素：**
艾尔文明轻量化理念（整体框架结构），但核心系统为联邦自主研发

**改装潜力综合评价：**
| 项目 | 评价 | 说明 |
|------|------|------|
| 改装上限 | ML-3 | 需校官级驾驶资格 |
| 推荐改装 | 幽能辅助模块扩容、AI决策系统强化、纳米自愈涂层 | 联邦授权改装件 |
| 改装后变体 | 仲裁者IIIA（悬浮时间+50%，航速-5%）；仲裁者III-SE（special operations，突击型） | 特种部队型号 |
| 改装限制 | 振金武器接口需TL-3认证；驾驶舱改造需重新校准神经链接 | 技术壁垒 |

**外观描述：**
仲裁者III是"猎杀者"的代名词，整体造型充满压迫感。头部采用双传感器"双眼"布局，面甲正中有一道蓝色光源线，激活时表示锁定状态；躯干呈楔形结构前倾，主武器炮管从胸前中央延伸；肩甲宽大外张呈倒梯形，左肩黑色闪电徽章，右肩银色天平徽章；配色以哑光黑为底，暗金色勾边，传感器光源为深红色。辨识特征：胸前延伸的重型光束炮炮管。

| **功能定位** | 防御型/重装型 |
| **驾驶方式** | 神经链接+强化外骨骼（驾驶员为半机械改造人） |
| **装甲** | 液态金属自愈装甲，总厚度200-250mm |
| **自重** | 185吨 |
| **高度** | 9.2米 |

**外星设计元素：**
维拉图斯特有的液态金属自愈装甲技术（来源：虫族材料逆向研究）
