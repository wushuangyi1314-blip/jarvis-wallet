# 第一部分：高频核心种族（详细展开）

## 1.1 图尔萨联邦（Tursa Federation）

| 项目 | 内容 |
|------|------|
| 联盟地位 | 三级领主级，安理会非常任理事国 |
| 政治体制 | 联邦制，七大城邦联合 |
| 人口规模 | 约800亿 |
| 领土范围 | 银河系中环，约50个恒星系 |
| 军事定位 | 联盟主要战力之一，清道夫舰队最大贡献者 |
| 机甲偏好 | 重装型、重火力型，讲究正面对抗 |

### 社会结构

图尔萨联邦由七大城邦组成，每个城邦是一个独立的政治实体，拥有自己的议会，但对外统一服从联邦议会的决议。联邦的权力核心是"执政官"，由七大城邦轮流担任，每届任期10年。

### 文化特征

- **秩序至上**：图尔萨人认为秩序是文明的基石，没有秩序就没有进步
- **效率崇拜**：图尔萨联邦是联盟最高效的战争机器工厂
- **等级分明**：社会分为四个等级——执政层、贵族层、平民层、契约层
- **四眼崇拜**：四个眼睛代表四个视角，象征全面看待事物的智慧

### 图尔萨联邦机甲体系

| 机甲名称 | 代际 | 类型 | 定位 |
|---------|------|------|------|
| 守护者II | 第二代 | 通用型 | 制式装备，产量最大 |
| 重锤II-H | 第二代 | 重装型 | 攻坚突击 |
| 仲裁者III | 第三代 | 精锐型 | 斩首行动 |
| 使徒IV | 第四代 | 王牌型 | 机甲单挑 |

### 图尔萨联邦军事编制

- 标准机甲小队：4台（1仲裁者III+2守护者II+1猎手II-S）
- 标准机甲大队：64台
- 图尔萨军团：通常由500-1000台机甲组成，配属相应舰队

### 与人类的关系

图尔萨联邦是人类最大的压迫者之一。联邦的法律规定人类只能从事"辅助性劳动"，被强制征召为炮灰兵。图尔萨人普遍看不起人类，认为人类是"有潜力但尚未进化的低等种族"。

## 1.2 维拉图斯帝国（Velatus Empire）

| 项目 | 内容 |
|------|------|
| 联盟地位 | 二级强权级，安理会常任理事国 |
| 政治体制 | 帝国制，皇帝拥有最高决策权 |
| 人口规模 | 约500亿（含机械改造公民） |
| 领土范围 | 银河系核心区，约80个恒星系 |
| 军事定位 | 联盟最强军事力量，舰队规模最大 |
| 机甲偏好 | 机械改造型、高防护型、无人协作型 |

### 社会结构

维拉图斯帝国是一个高度机械化的社会。帝国公民可以根据等级接受不同程度的机械改造，从简单的工具增强到近乎完全机械化的身体。皇帝的权力被称为"绝对逻辑权"，但实际上受到机械议会的制约。

### 文化特征

- **进化至上**：维拉图斯人认为生命的目标是不断进化，机械是进化的方向
- **逻辑优先**：所有决策都要经过逻辑推演，情感被视为"低效"
- **机械崇拜**：身体改造程度越高，社会地位越高
- **效率至上**：维拉图斯帝国的工厂生产效率是联盟最高的

### 维拉图斯帝国特色技术

| 技术 | 说明 | 联盟地位 |
|------|------|---------|
| 机械改造 | 神经接口、外骨骼、强化义肢 | 顶尖 |
| 液态金属装甲 | 自愈型装甲技术 | 独家 |
| 粒子炮 | 维拉图斯特有能量武器 | 顶尖 |
| 无人僚机 | AI控制的无人机群协调 | 顶尖 |

### 维拉图斯帝国机甲体系

| 机甲名称 | 代际 | 类型 | 定位 |
|---------|------|------|------|
| 铁壁III-H | 第三代 | 重装防御型 | 要塞防御 |
| 终结者IV-T | 第四代 | 战术轰炸型 | 对地轰炸 |
| 铁壁III-E | 第三代 | 重装型（外销） | 对外销售 |

### 与人类的关系

维拉图斯帝国对人类持"忽略"态度。在维拉图斯人眼中，人类与蚂蚁无异，不值得花费精力去压迫或关注。但帝国通过向图尔萨联邦出售军事装备，间接参与了人类被压迫的体系。

## 1.3 银狮族（Silver Leon）

| 项目 | 内容 |
|------|------|
| 联盟地位 | 三级领主级，安理会非常任理事国 |
| 政治体制 | 部落联盟制 |
| 人口规模 | 约200亿 |
| 领土范围 | 银河系边缘，多个小型星域 |
| 军事定位 | 联盟最强地面部队，个体战斗力最强 |
| 机甲偏好 | 近战型、高机动型 |

### 社会结构

银狮族由多个部落组成，重大决策由部落长老投票决定。部落之间有深厚的历史恩怨，但面对外部威胁时会暂时团结。银狮族最重视的是"荣耀"和"血统"。

### 文化特征

- **荣耀高于生命**：银狮族战士宁可战死也不愿被俘
- **血统论**：出身高贵的银狮人天生被认为更强
- **猎手文化**：银狮族传统上以狩猎为生，战斗是本能
- **独立精神**：银狮族是联盟内部最不稳定的力量，经常反对联盟决议

### 银狮族特色能力

| 能力 | 说明 |
|------|------|
| 夜视能力 | 双眼可在黑暗中看清物体 |
| 强化爪击 | 赤手空拳可撕裂金属 |
| 生物强化 | 通过基因改造增强体质 |
| 隐身潜行 | 银狮族特有的环境伪装 |

### 银狮族机甲体系

| 机甲名称 | 代际 | 类型 | 定位 |
|---------|------|------|------|
| 狮心II | 第二代 | 近战型 | 银狮族特有 |
| 银爪III | 第三代 | 高机动型 | 精锐部队 |
| 王狮IV | 第四代 | 指挥型 | 部落首领专用 |

### 与人类的关系

银狮族对人类的态度复杂。一方面，银狮族尊重强者，人类展现价值后会逐渐尊重；另一方面，银狮族鄙视"依附他人"的行为，而人类被迫依附联盟的结构让他们看不起。

## 1.4 艾尔文明（Aer Civilization）

| 项目 | 内容 |
|------|------|
| 联盟地位 | 三级领主级，安理会非常任理事国 |
| 政治体制 | 议会制 |
| 人口规模 | 约50亿（能量形态生命） |
| 领土范围 | 银河系中环，数个独立星域 |
| 军事定位 | 几乎无军事力量，主要提供技术和情报 |
| 机甲偏好 | 轻量化型、远程型 |

### 社会结构

艾尔文明是由能量形态生命建立的文明，社会结构松散，由五人核心议会管理重大事务。艾尔人寿命极长，决策效率极低，但技术积累深厚。

### 文化特征

- **知识至上**：艾尔人穷其一生追求知识和真理
- **中立主义**：艾尔人极少参与战争，提供中立调解
- **能量崇拜**：艾尔人认为纯能量形态是生命的最高形式
- **探索精神**：艾尔人是银河系最出色的探索者和科学家

### 艾尔文明特色技术

| 技术 | 说明 | 联盟地位 |
|------|------|---------|
| 量子通讯 | 无法被窃听的通讯技术 | 独家 |
| 能量存储 | 高效的能量储存媒介 | 顶尖 |
| 远程感知 | 大范围侦察系统 | 顶尖 |
| 轻量化材料 | 纳米级材料科学 | 顶尖 |

### 艾尔文明机甲体系

| 机甲名称 | 代际 | 类型 | 定位 |
|---------|------|------|------|
| 艾尔之息II | 第二代 | 远程支援型 | 艾尔文明特有 |
| 光辉III | 第三代 | 精准定位型 | 狙击专用 |

### 与人类的关系

艾尔文明是人类在联盟中最"友善"的种族之一。历史上艾尔人曾向人类提供基础量子技术，但后来在图尔萨联邦的压力下停止了技术输出。艾尔人对人类的处境表示同情，但不愿公开对抗联盟。

## 1.5 仲裁者文明（Arbiters Civilization）

| 项目 | 内容 |
|------|------|
| 联盟地位 | 一级霸主级，安理会常任理事国+否决权 |
| 政治体制 | 神权制 |
| 人口规模 | 约10亿 |
| 领土范围 | 银河系核心区，最富饶的区域 |
| 军事定位 | 拥有银河系最强私人舰队，从不轻易动用 |
| 机甲偏好 | 实验型、特殊型 |

### 社会结构

仲裁者文明由"仲裁者家族"统治，这个家族声称拥有"神授权力"。仲裁者家族成员寿命可达800年，外貌威严，皮肤呈淡金色。普通仲裁者公民没有政治权力，只能服从家族决议。

### 文化特征

- **权威崇拜**：仲裁者认为自己的权威来自神，不可挑战
- **秩序执念**：仲裁者将维持联盟秩序视为神圣使命
- **傲慢自大**：仲裁者几乎不与其他种族平等交往
- **秘密主义**：仲裁者文明的大部分技术和历史都是秘密

### 仲裁者文明特色技术

| 技术 | 说明 | 联盟地位 |
|------|------|---------|
| 空间稳定合金 | 原理不明的超坚固材料 | 独家（禁忌） |
| 曲速核心 | 小型空间跃迁技术 | 独家（禁忌） |
| 星系级武器 | 可操控恒星能量的武器 | 独家（禁忌） |

### 仲裁者文明机甲体系

| 机甲名称 | 代际 | 类型 | 定位 |
|---------|------|------|------|
| 深渊之眼EX | 实验型 | 情报收集 | 仲裁者专用 |
| 仲裁者级 | 第四代 | 王牌型 | 极少出动 |

### 与人类的关系

在仲裁者眼中，人类几乎不存在。仲裁者家族从未正式讨论过人类问题，对他们来说，人类只是联盟底层统计数字中的一个数字。夏天的人类身份曝光后，仲裁者家族才开始关注这个"意外"。

## 1.6 人类文明（Humanity）

| 项目 | 内容 |
|------|------|
| 联盟地位 | 四级独立级，普通成员（实际底层） |
| 政治体制 | 联合政府制 |
| 人口规模 | 约300亿 |
| 领土范围 | 太阳系及周边，实际被联盟控制 |
| 军事定位 | 炮灰，提供大量步兵单位 |
| 机甲偏好 | 学习吸收型、改进型 |

### 社会结构

地球联合政府由各国政府组成联合议会，但联合政府的权力受到联盟严格限制。人类内部有亲联盟派和反抗派的对立。亲联盟派认为应该服从联盟换取生存，反抗派认为应该争取独立地位。

### 文化特征

- **生存渴望**：人类的最高文化主题是"活下去"
- **学习能力**：人类学习能力强，能快速吸收其他种族的技术
- **韧性**：人类在绝境中展现出惊人的适应能力
- **尊严追求**：人类渴望获得尊重，而非被当作低等种族

### 人类特色技术

| 技术 | 说明 | 联盟地位 |
|------|------|---------|
| 应用科技转化 | 将现有技术快速转化为应用 | 中上 |
| 量子计算 | 阿尔佐姆遗产加成 | 顶尖（遗产） |
| 技术逆向工程 | 从缴获装备中提取技术 | 中 |

### 人类机甲体系

| 机甲名称 | 代际 | 来源 | 定位 |
|---------|------|------|------|
| 初火号 | 第五代 | 阿尔佐姆遗产 | 夏天专用 |
| 方舟-I | 第二代改进 | 逆向工程+遗产 | 方舟制式 |
| 方舟-II | 第三代改进 | 逆向工程+遗产 | 精锐部队 |
| 明月子体机甲 | 第四代无人 | 明月AI控制 | 无人集群 |

### 人类内部派系

| 派系 | 立场 | 代表人物 | 目标 |
|------|------|--------|------|
| 联合政府主流派 | 亲联盟 | 政府官员 | 在联盟框架内改善处境 |
| 反抗军 | 独立派 | 陈远航、秦羽 | 争取人类独立地位 |
| 方舟 | 暗中力量 | 夏天 | 利用阿尔佐姆遗产暗中提升人类实力 |
