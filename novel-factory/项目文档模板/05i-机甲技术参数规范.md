# 一、速度系统

## 1.1 速度单位定义

| 单位 | 定义 | 换算 |
|------|------|------|
| km/h | 公里/小时 | 大气层内常规速度单位 |
| 马赫 | 音速倍数 | 1马赫 ≈ 1224 km/h（海平面） |
| km/s | 公里/秒 | 星际航行常用单位 |
| 光速比 | 与光速比值 | 1光速 = 299792 km/s |

## 1.2 各等级机甲速度参考

| 机甲等级 | 大气层航速 | 近地轨道速度 | 星际巡航速度 | 曲速/跃迁 |
|---------|-----------|-------------|-------------|-----------|
| TL-1（0级） | 120-200 km/h | 7.8 km/s（第一宇宙速度） | 100-300 km/s | 无 |
| TL-2（1级） | 200-400 km/h | 7.8-11.2 km/s | 300-1000 km/s | 无 |
| TL-3（2级） | 400-800 km/h（2-4马赫） | 11.2 km/s | 1000-5000 km/s | 短程跃迁（<10光年） |
| TL-4（3级） | 800-2000 km/h（3-8马赫） | 15-20 km/s | 5000-20000 km/s | 中程跃迁（<100光年） |
| TL-5（4级） | 2000-5000 km/h（5-20马赫） | 20-50 km/s | 20000-100000 km/s | 远程跃迁（无限制） |

## 1.3 初火号速度参数（TL-5）

| 状态 | 速度 | 说明 |
|------|------|------|
| 大气层常规飞行 | 3000 km/h（约2.5马赫） | 经济模式 |
| 大气层紧急飞行 | 8000 km/h（约6.5马赫） | 最大推力 |
| 近地轨道机动 | 30 km/s | 第一宇宙速度级 |
| 星际巡航速度 | 50000 km/s（约0.00017光速） | 常规航行 |
| 星际极速航行 | 150000 km/s（约0.0005光速） | 紧急情况 |
| 量子跃迁 | 1000光年/次 | 跃迁后需15分钟稳定期 |

**重要说明**：初火号速度设定为TL-5等级，但**星际巡航速度50000-150000 km/s**远高于联邦普通战舰。这是因为阿尔佐姆遗产技术领先人类文明约10000年。

# 六、写作调用指南

## 6.1 描写机甲速度的标准写法

```
✅ 正确示例：
"初火号的引擎轰鸣，在大气层中划出一道蓝紫色的光轨，
航速瞬间攀升至6000公里/小时。"

❌ 错误示例：
"初火号以极高的速度飞行"（太模糊）
"初火号以每秒100公里的速度飞行"（大气层内太慢）
```

## 6.2 描写战斗防御的标准写法

```
✅ 正确示例：
"仲裁者III的光束炮击中初火号的护盾，
阿尔佐姆护盾泛起涟漪——攻击被完全抵挡。"
"使徒IV的振金光剑砍在初火号的装甲上，
火花四溅，却连一道划痕都没留下。"

❌ 错误示例：
"初火号轻松挡下了攻击"（缺乏细节）
```

## 6.3 描写武器攻击的标准写法

```
✅ 正确示例：
"初火号的光束炮切换至粒子模式，
一道蓝白色的光柱瞬间击穿了铁壁III-H的护盾，
在它的装甲上留下一个直径三米的焦黑窟窿。"

❌ 错误示例：
"初火号开火了，敌人被击败"（缺乏细节）
```

## 6.4 能源消耗描写参考

| 战斗场景 | 能源消耗 | 描写方式 |
|---------|---------|---------|
| 常规战斗1小时 | 15-20% | "能源消耗了五分之一" |
| 高烈度战斗10分钟 | 30-50% | "能源告急，只剩一半" |
| 全力输出 | 50-80% | "量子能核心咆哮，能源急剧下降" |
| 跃迁 | 30% | "蓝紫色光芒包裹机甲，30%能源瞬间消失" |
