# 第一部分：核心矿产

## 1.1 铁镍矿（Iron-Nickel Ore）

| 项目 | 内容 |
|------|------|
| 基础信息 | 最常见的金属矿藏，宇宙中最丰富的矿产之一 |
| 分布 | 小行星带、岩石行星地壳、废弃殖民星球 |
| 开采难度 | 低，技术成熟 |
| 主要用途 | 金属锭（低碳钢/高碳钢/合金钢）的原料 |
| 价格 | 10 D级宇宙晶/吨（未经加工的粗矿） |
| 提炼产物 | 低碳钢（10 D级宇宙晶/吨）、高碳钢（30 D级宇宙晶/吨）、合金钢（50 D级宇宙晶/吨） |

**提炼工艺流程**：
```
铁矿石 → 熔炼炉（1600°C，去除杂质） → 生铁 → 精炼厂（加入合金元素） → 低碳钢/高碳钢/合金钢 → 质量检测 → 入库
```

| 提炼产物 | 稀土金属锭、荧光粉、精密合金添加剂 |

**提炼工艺流程**：
```
稀土粗矿 → 化学处理（酸浸取） → 溶剂萃取 → 沉淀 → 焙烧 → 稀土氧化物 → 熔炼 → 稀土合金 → 精密元件
```

## 1.3 振金矿（Vibranium Ore）

| 项目 | 内容 |
|------|------|
| 基础信息 | 宇宙中最珍贵的装甲材料，来源是崩塌恒星残骸 |
| 分布 | 超新星爆发遗迹、中子星附近，极难开采 |
| 开采难度 | 极高，需要特殊采矿设备，且产地稀少 |
| 主要用途 | 王牌机甲装甲、第四代及以上机甲核心装甲材料 |
| 价格 | 5000 B级宇宙晶/吨（原始矿），精炼后10000 B级宇宙晶/吨 |
| 提炼产物 | 振金合金（振金+其他金属的合金） |

**提炼工艺流程**：
```
振金粗矿 → 特殊熔炼（需3000°C以上高温） → 提纯 → 振金锭 → 振金合金（加入其他金属） → 装甲板成型
```

**特性说明**：
- 振金是目前已知最轻且最坚硬的金属材料
- 极耐高温，可承受光束炮直射
- 有能量传导性，可用于能量武器散热
- 目前联盟只有仲裁者文明和维拉图斯帝国掌握精炼技术

```

## 2.3 机甲装甲加工流程（详细）

### 纳米合金装甲加工流程

```
第一步：材料配比
- 铁：40%
- 镍：20%
- 铬：15%
- 钼：10%
- 其他稀有金属：15%
精确比例混合，确保最终材料的均匀性

第二步：高温熔炼
- 在真空炉中加热至3000°C以上
- 熔炼过程需要持续搅拌确保均匀
- 去除杂质和气体

第三步：纳米分散
- 将熔融金属冷却后进行纳米级粉碎
- 粒度控制在50-100纳米范围
- 粉碎过程需在惰性气体保护下进行

第四步：热等静压
- 将纳米粉末装入高压模具
- 在高温高压下压实（温度1500°C，压力200MPa）
- 持续时间约2小时

第五步：表面处理
- 涂层：防腐蚀涂层
- 硬化：表面淬火处理
- 回火：消除内应力

第六步：质量检测
- 强度测试（拉伸、压缩、冲击）
- 成分分析（确保材料均匀性）
- 缺陷检测（超声波或X光）

第七步：成型加工
- 切割：激光切割或水刀切割
- 弯曲：压力机成型
- 焊接：氩弧焊或激光焊接
- 装甲板成型后送装备车间
```

### 振金合金装甲加工流程

```
第一步：振金矿熔炼
- 振金矿在真空感应炉中加热至3500°C
- 使用特殊溶剂去除杂质
- 提纯得到纯振金

第二步：合金配制
- 振金：60%
- 钛：20%
- 铬：10%
- 其他稀有金属：10%
- 在惰性气体保护下熔炼

第三步：特殊锻造
- 振金合金在固态下进行等温锻造
- 锻造温度控制在800-1000°C
- 多方向锻造确保组织均匀

第四步：精密加工
- 使用激光切割
- 金刚石刀具精密车削
- 确保装甲板厚度精确

第五步：质量检测
- 振金材料稀缺，检测更为严格
- 每块装甲板都有独立编号
- 记录生产批次和质检结果

特别说明：
- 振金装甲产能极低，联盟年产量不足100吨
- 主要用于第四代及以上机甲
- 夏天通过阿尔佐姆遗产获得部分振金合金
```

## 3.1 零件分类与精度要求

| 零件类型 | 精度要求 | 制造难度 | 典型部件 | 制造工艺 |
|---------|---------|---------|---------|---------|
| 结构件 | 中 | 低 | 骨架、外壳框架 | 切割、焊接、铆接 |
| 动力件 | 高 | 高 | 引擎、能量核心 | 精密铸造、精密加工 |
| 传动件 | 高 | 高 | 关节马达、液压系统 | 精密加工、热处理 |
| 电子件 | 极高 | 极高 | 量子芯片、传感器 | 纳米级加工、量子蚀刻 |
| 武器件 | 高 | 高 | 光束炮、导弹发射架 | 精密加工、特殊材料处理 |

## 3.2 精密零件制造流程

### 量子芯片制造流程（以明月核心材料为例）

```
第一步：硅晶圆准备
- 使用高纯度硅晶体
- 切割成指定厚度的晶圆
- 晶圆直径通常为200-300mm

第二步：量子蚀刻
- 在芯片上蚀刻量子线路
- 使用极紫外光刻技术
- 线宽控制在10纳米级别

第三步：量子点植入
- 精确植入量子点
- 量子点是量子计算的核心
- 需要在低温环境下进行

第四步：幽能通道刻蚀
- 刻蚀幽能传导通道
- 连接量子点和外部电路
- 通道宽度控制在纳米级别

第五步：多层堆叠
- 将多个量子处理层堆叠
- 每层之间需要精密对准
- 典型芯片有32-64层

第六步：封装测试
- 将芯片封装在保护壳中
- 进行功能测试
- 测试量子计算能力和稳定性

第七步：品质分级
- A级：量子计算能力达到设计指标的100%
- B级：达到设计指标的90%
- C级：达到设计指标的80%
- D级：80%以下，用于低端产品
```

## 3.3 机甲关节制造

机甲关节是高精度零件的代表，因为关节需要灵活转动同时承受巨大压力。

```
关节制造流程：
1. 轴承外环毛坯铸造（高强度合金钢）
2. 轴承内环毛坯铸造
3. 滚柱（滚珠）制造
4. 保持架加工
5. 热处理（渗碳、淬火、回火）
6. 磨削加工（达到微米级精度）
7. 表面处理（防腐蚀、耐磨）
8. 装配
9. 转动测试
10. 润滑系统安装
```

```

第四阶段：武器系统集成
1. 武器挂载点安装

第六阶段：交付验收
