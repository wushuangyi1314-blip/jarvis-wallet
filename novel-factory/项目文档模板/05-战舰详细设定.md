# 第一部分：战舰总论

## 1.1 战舰定位

战舰是星际战争的核心作战平台，相比机甲拥有更大的体型、更强的火力、更远的航程和更强的续航能力。战舰承担着舰队决战、轨道轰炸、兵力投送、后勤补给等核心任务，是星际文明的战略级资产。

**战舰与机甲的分工：**
- 机甲：灵活机动作战、地面支援、斩首行动、单挑对决
- 战舰：舰队决战、轨道支援、远程火力投送、战略运输

> **核心设定原则：** 高等级战舰拥有压倒性火力，但同时伴随巨大的操控负担、能源消耗和维护成本。数量优势在很多情况下可以弥补单舰质量差距。

---

## 1.2 战舰分级体系（对应科技树）

战舰级别由科技综合水平决定，按排水量和作战能力划分：

| 舰级 | 科技要求 | 排水量 | 航行能力 | 代表舰级 | 文明归属 |
|------|---------|--------|---------|---------|---------|
| 战舰V0 | 综合1级 | 5,000-50,000吨 | 太阳系内航行 | "熔火"级 | 人类联盟 |
| 战舰V1 | 综合2级 | 50,000-500,000吨 | 恒星际航行/曲速 | "星陨"级 | 图尔萨联邦 |
| 战舰V2 | 综合3级 | 500,000-5,000,000吨 | 星系内航行/空间折叠 | "深渊"级 | 艾尔文明/仲裁者遗产 |
| 战舰V3 | 综合4级 | 5,000,000-50,000,000吨 | 宇宙航行/时间场 | "终焉"级 | 虚空帝国（虚构） |
| 战舰V4 | 综合5级 | 无上限 | 超宇宙航行/因果跨越 | "本源"级 | 超宇宙文明（虚构） |

**科技限制说明：**
- 战舰V0需要物理1+能源1以上，否则舰体结构强度不足以支持航行
- 战舰V1需要物理2+能源2以上，否则无法实现曲速航行
- 战舰V2需要物理3+材料3以上，否则空间折叠将撕裂舰体
- 战舰V3需要物理4+能源4以上，时间场操控需要极高能量密度
- 战舰V4为超宇宙级文明专属，超越物理法则层面

## 2.2 护卫舰（Frigate）

**定位：** 最低级别作战舰艇，承担护航、巡逻、反潜等低烈度任务。

| 项目 | 内容 |
|------|------|
| 舰级 | 战舰V0（综合1级） |
| 排水量 | 8,000吨 |
| 全长 | 120米 |
| 舰宽 | 15米 |
| 吃水 | 8米 |
| 乘员 | 40人（标准值）+20人（航空组） |
| 动力系统 | 核聚变反应堆×2，输出功率500MW |
| 航速 | 最大航速30节，加速至最高速需15分钟 |
| 续航力 | 太阳系内无限制航行，典型作战半径5000万公里 |
| 装甲 | 复合装甲，厚度50-100mm |
| 主武器 | 200mm单联装舰炮×1（射程100km，射速5发/分钟，穿深可击穿驱逐舰装甲） |
| 副武器 | 近防激光炮×4（拦截导弹）、四联装防空导弹×2（射程50km） |
| 舰载机 | 小型无人机×4（侦察用） |

**外观描述：**
"熔火"级护卫舰外观朴素实用，舰体呈修长的纺锤形，舰首有一门200mm舰炮呈半埋入式安装，舰体中段两侧各有四个导弹发射口，舰尾有明显的核聚变反应堆散热口，运行时泛着暗红色光芒，这也是"熔火"得名的原因。舰桥位于舰体前部上方，视野开阔但缺乏防护，是明显的弱点。

## 4.2 动力系统详细说明

### 核聚变反应堆（V0-V1级）

- **原理：** 氢核聚变产生能量，技术成熟可靠
- **优点：** 技术成熟，维护成本低，可快速更换
- **缺点：** 能量密度有限，大型战舰需要多个反应堆并联
- **使用限制：** 需要定期更换燃料棒；反应堆过载可能引发爆炸

### 暗物质引擎（V1-V2级）

- **原理：** 暗物质与暗物质湮灭产生能量，可操控空间形成曲速场
- **优点：** 能量密度是核聚变的100倍，可实现曲速航行
- **缺点：** 暗物质燃料稀缺且储存困难；曲速核心维护成本极高
- **使用限制：** 曲速航行对舰体有压力，舰体寿命会缩短；每次曲速后需检修；曲速7级每季度使用不超过3次

### 零点能引擎（V2-V3级）

- **原理：** 从量子真空中提取零点能，能量近乎无限
- **优点：** 能源近乎无限，可支持空间折叠；无需燃料补给
- **缺点：** 建造和维护需要阿尔佐姆遗产级材料；启动需要大量稀有元素
- **使用限制：** 空间折叠后需要24-48小时冷却；需要专业团队操作

### 弦能核心（V3-V4级）

- **原理：** 超弦振动能量利用，可操控空间和时间
- **优点：** 能量输出超越物理法则，可实现超远距空间折叠和时间场操控
- **缺点：** 需要超维材料建造，每年使用限制10次以内
- **使用限制：** 弦能核心失控可能导致空间折叠事故；操控需要500人以上专业团队
