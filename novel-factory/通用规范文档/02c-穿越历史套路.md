# 1、科举仕途流

> 代表作：《上品寒士》《逍遥小书生》《寒门崛起》

## 1.1 一句话公式

**科举仕途流 = "寒门书生" + "科举之路" + "庙堂权谋"**

---

## 1.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 寒门书生/穿越者 + 才华横溢 + 科举入仕 |
| 女主 | 才女/官家小姐 + 红颜知己 |
| 配角 | 书院同窗/恩师/同僚 |
| 反派 | 权贵子弟/贪官污吏 |

**主角特质：**
- 穿越有现代知识
- 诗词文章碾压
- 科举一路开挂
- 最终庙堂权谋

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 穿越到古代+发现身份+第一次科举 |
| 困境（10-30章） | 科考压力/权贵打压 |
| 机遇（每20章） | 诗词文章出名+步步高升 |
| 官场（30-60章） | 进入官场+权谋斗争 |
| 巅峰（60-80章） | 位极人臣/改变历史 |
| HE（80-100章） | 功成名就+美人相伴 |

### 【套路公式】

**核心循环：**
```
科举→入仕→权谋→更高官位→更大舞台
```

**爽点设计：**
```
1. 诗词文章碾压全场
2. 科考一路过关
3. 庙堂权谋胜利
4. 改变历史
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 位极人臣 | 科举成功+官场得意 |
| 感情线 | 才女相伴+红颜知己 |
| 代价 | 官场风险/政治斗争 |
| 核心 | 知识改变命运 |

---

## 1.3 写科举仕途流必有的要素

✅ 穿越+现代知识
✅ 科举主线
✅ 诗词文章（碾压爽点）
✅ 庙堂权谋
✅ 改变历史的格局

❌ 不要太顺（金手指太大会无趣）

---

## 1.4 避开的坑

❌ 历史知识不要有bug
❌ 科举描写要符合史实
❌ 权谋不要太幼稚
❌ 不要写太多后宫（影响主线）
