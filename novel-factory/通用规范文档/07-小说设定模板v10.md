## 概述

小说设定模板是步骤6（初版卷纲→小说设定）的输出规范文档。根据品类不同，设定分为通用部分（所有品类都要填）和分类特殊部分（按品类选择填写）。

---

# 通用部分（必填）

## 世界规则设定

**定义：** 描述小说世界的根本规则，包括世界构造、层级结构、时间法则、空间法则等。

**格式：**

```
世界构造：（一重世界/多重世界/位面/位格等）
世界层级：（凡界→灵界→仙界等）
时间法则：（正常时间流/时间可逆/时间加速等）
空间法则：（空间稳定/可撕裂空间/瞬移等）
力量上限：（当前世界最高力量层级）
其他规则：（因果律/天道/命运/气运等）
```

## 世界矛盾设定

**定义：** 描述世界层面的核心矛盾，即整个世界面临的主要问题或冲突。

**格式：**

```
世界主要矛盾：（核心冲突描述）
矛盾起源：（矛盾的起源是什么）
矛盾现状：（当前矛盾的局势）
矛盾发展趋势：（矛盾会如何演变）
```

## 力量体系设定

**定义：** 描述世界中存在的各种力量类型、力量的来源、力量的修炼方式。

**格式：**

```

修炼方式：（修炼功法/战斗积累/顿悟等）
核心力量：（最重要的力量体系是什么）
高层战力特征：（如移山填海/寿元万年等）
```
