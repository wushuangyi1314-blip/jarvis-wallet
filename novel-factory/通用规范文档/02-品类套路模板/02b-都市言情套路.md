# 1、总裁文

> 代表作：《甜诱！京圈太子爷》《闪婚总裁契约妻》

## 1.1 一句话公式

**总裁文 = "霸道总裁" + "豪门恩怨" + "甜撩追宠"**

---

## 1.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 女主 | 普通出身 + 性格坚韧 + 隐藏身份/能力 |
| 男主 | 京圈豪门/霸道总裁 + 外冷内热 + 占有欲强 |
| 配角 | 闺蜜/助攻 + 恶毒女配/绿茶 |
| 反派 | 豪门内斗/家族阴谋/情敌 |

**标配人设组合：**
- 坚韧小白花 × 京圈太子爷
- 契约新娘 × 傲娇总裁
- 双重身份女主（马甲掉落爽点）

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 意外相遇/契约结婚/被迫嫁入豪门 |
| 相知（10-30章） | 误解→了解→心动/甜蜜互动 |
| 危机（30-50章） | 家族阻挠/绿茶作妖/身份误会 |
| 分离（50-70章） | 出国/假死/消失/被迫分离 |
| 重逢（70-90章） | 追妻火葬场/再次相遇 |
| HE（90-100章） | 误会解开/甜蜜生活/马甲掉落 |

---

### 【套路公式】

**核心循环：**
```
相遇→误解→心动→危机→分离→重逢→HE
```

**马甲设定：**
```
女主隐藏身份：千金/设计师/神秘身份
马甲掉落场景 = 重大爽点
```

**经典场景：**
- 女主身份揭露
- 男主追妻（虐男主）
- 绿茶被打脸
- 甜蜜互动/壁咚/强吻

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 大团圆 | 功成名就 + 甜蜜生活 |
| 感情线 | 双洁/从头到尾只爱一人 |
| 代价 | 豪门恩怨的牺牲 |
| 核心 | 真爱能战胜一切 |

---

## 1.3 写总裁文必有的要素

✅ 霸道总裁人设（外冷内热/占有欲）
✅ 豪门背景（家族恩怨）
✅ 甜撩互动（壁咚/强吻/宠溺）
✅ 女主要有能力（不是傻白甜）
✅ 追妻火葬场（虐男主）
✅ 马甲掉落爽点

❌ 女主不要太傻白甜（读者会弃书）
❌ 男主不要太渣（有底线）

---

## 1.4 避开的坑

❌ 绿茶女配不要太多（审美疲劳）
❌ 误会不要拖太久（超过30章会弃书）
❌ 豪门阴谋不要太狗血
❌ 不要写太多三角恋（影响甜度）

# 2、重生文

> 代表作：《嫡女重生悔婚》《重生之将门毒后》《似锦》

## 2.1 一句话公式

**重生文 = "预知前世" + "逆天改命" + "今生复仇/补偿"**

---

## 2.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 女主 | 重生回来 + 预知前世 + 复仇/改命 |
| 男主 | 前世错过/渣男/今生追妻 |
| 配角 | 恶毒姐妹/继母继父/白莲花 |
| 反派 | 前世仇人/今生继续作妖 |

**主角特质：**
- 预知未来（知道谁会害她）
- 智商在线（不再傻白甜）
- 有仇必报

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 重生回来/发现回到前世节点 |
| 相知（10-30章） | 避开前世坑/手撕绿茶 |
| 危机（30-50章） | 前世仇人反击/家族阴谋 |
| 分离（50-70章） | 男主/女主消失/被迫离开 |
| 重逢（70-90章） | 前世错过的人再次相遇 |
| HE（90-100章） | 复仇成功/甜蜜结局 |

---

### 【套路公式】

**核心循环：**
```
重生→避开前世坑→打脸绿茶→发现前世真相→今生复仇→收获幸福
```

**爽点设计：**
```
1. 预知未来，避开所有陷阱
2. 前世仇人今生继续作妖，但女主已看穿
3. 手撕恶毒女配
4. 前世错过的人今生再相遇
5. 复仇成功
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 复仇成功 | 收拾所有仇人 |
| 感情线 | 前世错过/今生补上 |
| 代价 | 重生的代价/前世的遗憾 |
| 核心 | 今生不再重蹈覆辙 |

---

## 2.3 写重生文必有的要素

✅ 重生节点明确
✅ 前世记忆/预知能力
✅ 智商在线（不再傻白甜）
✅ 有仇必报/有恩必还
✅ 前世仇人今生继续作妖

❌ 重生后不要太顺（金手指太大会无趣）

---

## 2.4 避开的坑

❌ 女主不要突然变圣母（人设要统一）
❌ 复仇不要拖太久（要干脆）
❌ 前世记忆不要一下全想起来（节奏要控制）
❌ 不要写太多重生（一部作品就够了）

# 3、年代文

> 代表作：《婚途升职》《重回七零》《重生八零》

## 3.1 一句话公式

**年代文 = "70-90年代背景" + "技术流/创业" + "奋斗成长"**

---

## 3.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 女主 | 大女主/技术流/创业达人 |
| 男主 | 兵哥哥/干部/有责任感的男人 |
| 配角 | 家人/亲戚/邻居 |
| 反派 | 极品亲戚/情敌 |

**时代特色：**
- 背景：70-90年代
- 技术流：手工艺/烹饪/服装/养殖
- 奋斗感：从无到有

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 穿越/重生到年代+金手指 |
| 困境（10-30章） | 家庭困境/物质匮乏/极品亲戚 |
| 机遇（每10章） | 发现商机/技术进步 |
| 创业（30-60章） | 事业起步/成功 |
| 感情（贯穿） | 遇到真爱/互相扶持 |
| HE（80-100章） | 事业爱情双丰收 |

---

### 【套路公式】

**核心循环：**
```
困境→找到商机→技术致富→带动周围人→收获爱情
```

**年代设定：**
```
1. 时代特色：物质匮乏/计划经济/铁饭碗
2. 金手指：超前技术/商业头脑/人脉
3. 爽点：发家致富+打脸极品亲戚
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 发家致富 | 从无到有/带动周围人 |
| 感情线 | 相互扶持/细水长流 |
| 时代感 | 见证时代变迁 |
| 核心 | 奋斗改变命运 |

---

## 3.3 写年代文必有的要素

✅ 时代背景真实（70-90年代特色）
✅ 金手指要有时代感（技术/商业）
✅ 奋斗过程（不是一下就成功）
✅ 极品亲戚（增加冲突）
✅ 发家致富线（爽点）

❌ 不要太夸张（年代感要真实）

---

## 3.4 避开的坑

❌ 时代设定不要出错（会出戏）
❌ 不要太现代思维（要符合年代）
❌ 发家不要太容易（要有过程）
❌ 极品亲戚不要太多（会狗血）

---
