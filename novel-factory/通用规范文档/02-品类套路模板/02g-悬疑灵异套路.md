# 1、推理破案流

> 代表作：《十日终焉》《死亡轮回》《无人生还》
## 1.1 一句话公式

**推理破案流 = "高智商推理" + "破案解谜" + "逻辑严谨"**

---

## 1.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 侦探/警察 + 高智商 + 破案 |
| 女主 | 搭档/助手 |
| 配角 | 嫌疑人/证人 |
| 反派 | 高智商罪犯 |

**主角特质：**
- 高智商
- 观察细节
- 逻辑严谨

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 命案发生+侦探介入 |
| 困境（10-20章） | 线索繁杂/嫌疑人 |
| 推理（每10章） | 发现关键线索 |
| 高潮（30-40章） | 推理成功+真相大白 |
| 结局（40-50章） | 案件解决 |

### 【套路公式】

**核心循环：**
```
命案→线索→推理→真凶→案件解决
```

**爽点设计：**
```
1. 细节观察
2. 逻辑推理
3. 真相揭露
4. 高智商对决
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 案件解决 | 真相大白 |
| 感情线 | 可有可无 |
| 代价 | 推理过程费脑 |
| 核心 | 逻辑严谨的胜利 |

---

## 1.3 写推理破案流必有的要素

✅ 案件设计精妙
✅ 线索合理埋藏
✅ 逻辑自洽
✅ 高智商对决
✅ 推理过程详细

❌ 结局不要太明显

---

## 1.4 避开的坑

❌ 线索不要太明显
❌ 不要有逻辑漏洞
❌ 凶手不要太随机
❌ 结局不要太牵强
