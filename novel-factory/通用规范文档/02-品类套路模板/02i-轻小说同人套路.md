# 1、轻小说文风

> 代表作：《在下坂本》《齐木楠雄》《银魂》
## 1.1 一句话公式

**轻小说文风 = "轻松吐槽" + "日常向" + "角色魅力"**

---

## 1.2 四维度完整公式

### 【人物公式】
| 角色 | 公式 |
|------|------|
| 主角 | 吐槽役 + 日常无敌 |
| 女主 | 吐槽搭档 |
| 配角 | 喜剧担当 |
| 反派 | 基本没有/日常小矛盾 |

**文风特色：**
- 吐槽役
- 轻松日常
- 角色魅力

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 日常开始+角色登场 |
| 日常（10-30章） | 日常搞笑 |
| 小事件（每10章） | 日常小危机 |
| 温馨（30-60章） | 温情时刻 |
| 巅峰（60-80章） | 日常大事件 |
| HE（80-100章） | 日常继续 |

### 【套路公式】

**核心循环：**
```
日常→吐槽→小事件→温情→日常继续
```

**爽点设计：**
```
1. 吐槽役
2. 角色魅力
3. 日常搞笑
4. 温情时刻
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 日常继续 | 没有结局 |
| 感情线 | 可有可无 |
| 代价 | 无 |
| 核心 | 角色魅力 |

---

## 1.3 写轻小说文风必有的要素

✅ 吐槽役
✅ 日常氛围
✅ 角色魅力
✅ 轻松搞笑

❌ 不要太沉重

---

## 1.4 避开的坑

❌ 不要写太多剧情
❌ 不要有太多冲突
❌ 角色不要太崩
❌ 不要有太多严肃内容

---
# 2、同人创作流

> 代表作：《火影同人》《海贼同人》《综漫同人》
## 2.1 一句话公式

**同人创作流 = "原作角色" + "同人设定" + "新故事"**

---

## 2.2 四维度完整公式

### 【人物公式】
| 角色 | 公式 |
|------|------|
| 主角 | 穿越者/原作角色 + 同人设定 |
| 女主 | 原作女主/原创女主 |
| 配角 | 原作角色 |
| 反派 | 原作BOSS/新BOSS |

**设定特色：**
- 原创作设定
- 同人新设定
- 角色不崩

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 进入同人世界+了解设定 |
| 困境（10-20章） | 原作事件 |
| 机遇（每10章） | 改变原作 |
| 崛起（20-50章） | 创造新故事 |
| 巅峰（50-70章） | 最终事件 |
| HE（70-100章） | 完结 |

### 【套路公式】

**核心循环：**
```
原作事件→改变命运→新故事→同人结局
```

**爽点设计：**
```
1. 改变原作悲剧
2. 角色不崩
3. 新故事发展
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 改变原作 | 同人结局 |
| 感情线 | 原创/原作女主 |
| 代价 | 改变命运 |
| 核心 | 同人精神 |

---

## 2.3 写同人创作流必有的要素

✅ 原作角色不崩
✅ 尊重原作设定
✅ 改变原作遗憾
✅ 新故事发展

❌ 角色OOC

---

## 2.4 避开的坑

❌ 原型太OOC
❌ 改变原作太多
❌ 写违禁内容
❌ 忽视原作粉

---
