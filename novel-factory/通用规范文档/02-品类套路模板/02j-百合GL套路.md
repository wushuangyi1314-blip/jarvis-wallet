# 1、校园百合

> 代表作：《轻声说爱你》《女校日常》《同级生》
## 1.1 一句话公式

**校园百合 = "校园背景" + "girl meets girl" + "青春成长"**

---

## 1.2 四维度完整公式

### 【人物公式】
| 角色 | 公式 |
|------|------|
| 女主1 | 开朗/内敛 + 遇到另一个她 |
| 女主2 | 性格互补 + 产生感情 |
| 配角 | 闺蜜/助攻 |
| 反派 | 误会/情敌 |

**主角特质：**
- 校园背景
- 青春成长
- 性格互补

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 相遇/同班 |
| 相知（10-20章） | 产生好感 |
| 困境（20-30章） | 自我认知/外界压力 |
| 确认（30-50章） | 确认感情 |
| 甜蜜（50-70章） | 甜蜜互动 |
| HE（70-100章） | 公开/毕业 |

### 【套路公式】

**核心循环：**
```
相遇→好感→确认感情→甜蜜→外界压力→HE
```

**甜点设计：**
```
1. 眼神对视
2. 小动作互动
3. 互相吃醋
4. 确认感情
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| HE | 甜蜜在一起 |
| 感情线 | 双向奔赴 |
| 代价 | 社会压力 |
| 核心 | girl meets girl |

---

## 1.3 写校园百合必有的要素

✅ 校园氛围
✅ 性格互补
✅ 青春成长
✅ 甜蜜互动

❌ 不要太狗血

---

## 1.4 避开的坑

❌ 三角恋太多
❌ 误会拖太久
❌ 太多外部压力
❌ 角色太工具人
