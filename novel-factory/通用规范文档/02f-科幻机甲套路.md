# 1、机甲觉醒流

> 代表作：《吞噬星空》《全球高武》《机甲时代的崛起》

## 1.1 一句话公式

**机甲觉醒流 = "机甲/武道" + "觉醒进化" + "人类崛起"**

---

## 1.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 普通人 + 机甲觉醒 + 人类希望 |
| 女主 | 机甲战士/女主 |
| 配角 | 机甲队友/教官 |
| 反派 | 怪兽/外星生物 |

**主角特质：**
- 机甲/武道修炼
- 觉醒进化
- 人类最强

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 机甲觉醒+发现天赋 |
| 困境（10-20章） | 怪兽围困/训练 |
| 机遇（每10章） | 机甲升级/觉醒 |
| 崛起（20-50章） | 成为强者 |
| 巅峰（50-70章） | 对抗最终BOSS |
| HE（70-100章） | 人类崛起 |

### 【套路公式】

**核心循环：**
```
觉醒→战斗→升级→更强敌人→更大挑战
```

**爽点设计：**
```
1. 机甲觉醒
2. 以弱胜强
3. 突破极限
4. 人类最强
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 人类崛起 | 打败怪兽 |
| 感情线 | 机甲伴侣 |
| 代价 | 战斗牺牲 |
| 核心 | 人类不屈的意志 |

---

## 1.3 写机甲觉醒流必有的要素

✅ 机甲/武道设定
✅ 觉醒进化
✅ 战斗描写
✅ 人类最强

❌ 不要写太多感情

---

## 1.4 避开的坑

❌ 战斗不要太重复
