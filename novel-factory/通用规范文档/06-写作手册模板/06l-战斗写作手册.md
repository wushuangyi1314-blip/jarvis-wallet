# 战斗写作手册

## 第七章：打斗通用技巧

### 7.1 动作描写的技巧

**具体胜于模糊**

❌ 模糊：他一拳打了过去，对方倒下了
✅ 具体：一拳轰在对方小腹，对手虾一样弓起身子

**五感描写**

除了视觉，还有听觉、触觉等：

> 拳头入肉的闷响。
>
> 骨骼断裂的声音。
>
> 血溅在脸上的温热。

**动作连贯**

动作要有连贯性，一个动作接一个动作：

> 他先是用剑格挡住了迎面劈来的一刀，顺势下压，剑尖划过对方手臂，
>
> 紧接着一个转身，剑柄撞在对方胸口，
>
> 最后一脚将对方踹飞。

### 7.2 打斗节奏的把握

**张弛有度**

不能全程高能，要有节奏变化：

| 阶段 | 节奏 | 描写力度 |
|------|------|---------|
| 开场 | 紧张 | 快速交代双方状态 |
| 试探 | 稍缓 | 互相试探，有来有往 |
| 僵持 | 紧张 | 双方全力以赴 |
| 转折 | 极紧张 | 关键一击 |
| 收尾 | 舒缓 | 结果确定 |

**留白的艺术**

不是所有动作都要写出来：

> 两剑相交。
>
> 电光火石间，两人已经换了十几招。
>
> 等萧炎回过神来，对手已经倒在地上。

### 7.3 打斗中的对话

**战斗中能说话吗？**

可以，但要注意合理性：
- 修仙战斗：可以用灵力震动空气发声
- 机甲战斗：通讯频道
- 网游战斗：语音/打字
- 都市战斗：短句，一边打一边说
- 末世战斗：尽量不说话

**战斗中的对话类型：**

| 类型 | 示例 |
|------|------|
| 嘲讽 | "就这？" |
| 心理战 | "你的右肩有伤吧？" |
| 警告 | "小心后面！" |
| 宣告 | "这一招，送你上路。" |
| 惨叫 | "啊——" |

### 7.4 环境描写

**环境作为变量**

环境会影响战斗：

| 环境 | 影响 |
|------|------|
| 夜间 | 视野受限 |
| 雨中 | 视线模糊，地面湿滑 |
| 室内 | 空间有限 |
| 废墟 | 掩体多，地形复杂 |
| 高空 | 坠落危险 |

**破坏的描写**

战斗会改变环境：

> 两人从屋内打到屋外，沿途的墙壁被撞出一个人形窟窿。
>
> 院中的石桌在余波中碎裂。
>
> 最后，两人落在村口的枯井旁，四周一片狼藉。

## 第八章：战斗心理描写

### 8.1 战斗中的心理状态

**冷静型**

> 萧辰的心跳没有丝毫加速。他的大脑在高速运转，分析着对手的每一个动作。
>
> 出拳的角度、脚步的移动、呼吸的节奏……
>
> 破绽！

特点：
- 内心独白多
- 分析性思维
- 在危机中寻找机会

**热血型**

> 萧辰的血液在燃烧。
>
> 这种感觉，让他沉醉。
>
> 战斗，战斗，直到一方倒下！

特点：
- 情绪外露
- 行动先于思考
- 越战越勇

**紧张型**

> 萧辰的手心全是汗。
>
> 他知道，一旦失误就是死亡。
>
> 但他没有退路。

特点：
- 心理压力大
- 通过身体反应展示（手抖、呼吸急促）
- 紧张中做出决策

### 8.2 不同类型主角的战斗心理

| 类型 | 战斗心态 | 描写方式 |
|------|---------|---------|
| 废柴逆袭 | 压抑后的爆发 | 长期被压制的愤怒释放 |
| 天才型 | 从容不迫 | 游刃有余 |
| 老司机型 | 冷静计算 | 分析+执行 |
| 成长型 | 边战边学 | 在战斗中领悟 |

### 8.3 战斗中的内心转变

**从恐惧到勇敢**

> 第一刀砍来的时候，萧辰本能地想躲。
>
> 但他没有退路了。
>
> 这一次，他选择迎面而上。

**从轻敌到重视**

> 一开始，他以为这只是一场轻松胜利。
>
> 但对手的第三招让他意识到——这是生死之战。

**从慌乱到专注**

> 前三招他完全是被动挨打。
>
> 但渐渐地，他的世界安静下来。
>
> 眼中只剩下对手的动作。

## 质量检查点

调用本手册后，创作战斗场景时需检查：

- [ ] 战斗类型是否符合小说世界观（修仙/机甲/网游等）
- [ ] 力量体系/技能系统是否自洽
- [ ] 战斗逻辑是否通顺（能量消耗、冷却、代价）
- [ ] 动作描写是否具体
- [ ] 战斗节奏是否有张有弛
- [ ] 主角实力是否在设定范围内
- [ ] 战斗结果是否有铺垫（不是突然就赢了）
- [ ] 是否有越级战斗的理由
- [ ] 环境是否影响了战斗
- [ ] 战斗后的影响是否交代（伤势、资源消耗等）
