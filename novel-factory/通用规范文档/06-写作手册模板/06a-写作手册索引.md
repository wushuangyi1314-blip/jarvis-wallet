# 写作手册索引

## 概述

写作手册是小说家Agent在创作时的工具参考书，包含人物塑造、打斗描写、反派设计、爽点设计四大领域。每个手册设计为按需查阅的碎片化知识，不是通读型文档。

**本索引是小说家Agent的"查表手册"——拿到任务后，先查这个文件，找到对应的手册和章节，再去调用。**

## 三、场景类型→手册调用对照表（细化表）

**所有战斗场景通用调用顺序：**

```
第一步：06b打斗基础理论（必读，战斗框架）
第二步：品类对应战斗手册（修仙→06g，网游→06i，科幻→06h，都市→06j，末日→06k）
第三步：06l战斗写作手册（战斗通用技巧+心理）
第四步：若涉及修仙/玄幻元素，加读06g
```

| 场景类型 | 必读手册 | 说明 |
|---------|---------|------|
| **战斗-修仙/玄幻** | 13a打斗基础 + 06g修仙玄幻 + 06l | 六大门派覆盖：肉身/法术/阵法/法宝/精神/功法 |
| **战斗-虚拟网游** | 13a打斗基础 + 06i网游 + 06l | 含完整数值系统（八步伤害计算） |
| **战斗-科幻机甲** | 13a打斗基础 + 06h科幻机甲 + 06l | 含机甲系统/星舰战/太空战 |
| **战斗-都市/异能** | 13a打斗基础 + 06j都市 + 06l | 含异能战斗（三大分类/代价/克制） |
| **战斗-末日废土** | 13a打斗基础 + 06k末日 + 06l | 含丧尸/资源管理/道德困境 |
| **战斗-通用（无分类）** | 13a打斗基础 + 06l | 通用拳脚/冷兵器/枪械战斗 |
| **人物塑造** | 06c人物塑造 | 所有有人物出场场景 |
| **反派设计** | 06h反派设计 | 反派首次出场/对决/结局前 |
| **爽点设计** | 06e爽点 | 升级/打脸/情感高潮前 |
| **打斗通用技巧** | 06l战斗写作 | 动作描写/节奏/对话/环境描写 |

## 四、状态文件中 required_docs 的填写规则

**这是确保正确调用的核心机制——每次spawn时，状态文件必须内嵌当前任务需要的文档列表。**

```json
{
  "agent": "rd-agent",
  "task": "创作第3批第1-10章",
  "phase": "创作执行",
  "batch": "第3批",
  "novel_category": "玄幻修仙",
  "required_docs": {
    "must_read": [
      "06c-人物塑造手册",
      "06g-修仙玄幻打斗",
      "06b-打斗基础理论",
      "06h-反派设计手册v2",
      "06l-战斗写作手册"
    ],
    "by_chapter": {
      "ch1": "13a人物-新人出场",
      "ch3": "06g修仙-六大门派",
      "ch5": "06h反派-第一种反派类型",
      "ch7": "06e爽点-升级爽点设计",
      "ch10": "06l战斗写作-战斗节奏"
    }
  },
  "context_summary": "主角刚突破筑基，发现宗门被灭，第3章埋下神秘玉佩伏笔",
  "active_foreshadowing": [
    "神秘玉佩-埋第3章-预计第15章回收",
    "仇人势力-埋第1章-预计第20章登场"
  ]
}
```

**填写规则（强制）：**
- 必须包含当前品类的必读手册
- 必须包含当前批次涉及战斗类型对应的战斗手册
- 如有新角色出场，必须包含 06c人物塑造
- 如有反派出场/对决，必须包含 06h反派设计
- 如有升级/高潮，必须包含 06e爽点
- `by_chapter` 字段：每章列出该章最需要参考的手册章节

## 五、各手册快速调用路径

### 06c-人物塑造手册
- 新角色首次出场 → "角色类型"+"对白描写"
- 角色关键转折 → "角色成长弧光"+"心理描写"
- 角色关系变化 → "人物关系"+"情感互动"

### 06b-打斗基础理论
- 任何战斗开始前先读 → "战斗基础框架"+"力量体系"
- 不确定用哪个战斗手册 → 先读这本，确认后再选品类手册

### 06g-修仙玄幻打斗
- 修仙战斗 → "六大攻击流派"（肉身/法术/阵法/法宝/精神/功法）
- 法宝对决 → "法宝攻击"+"精神攻击"
- 渡劫场景 → "特有元素-天劫"

### 06h-反派设计手册v2
- 新反派设计 → "反派类型"+"动机设计"
- 反派高光 → "反派高光设计"+"行为逻辑"
- 反派结局 → "反派死亡/退场设计"

### 06h-科幻机甲打斗
- 机甲战斗 → "机甲系统"+"驾驶模式"
- 战舰/星战 → "星舰战"+"太空环境"
- 机甲近身战 → "机甲VS机甲"+"机甲VS生物"

### 06i-网游打斗
- 数值系统 → "八步伤害计算"（完整框架）
- PVP战斗 → "PVP单挑"+"技能组合"
- 副本/Boss → "副本Boss"+"Raid战"

### 06j-都市现代战斗
- 都市战斗（无异能）→ "街头搏斗"+"冷兵器"+"枪械"
- 都市异能战斗 → "异能战斗-三大规则"+"异能vs枪械"
- 网络/情报战 → "网络战与情报战"

### 06k-末日生存战斗
-丧尸战斗 → "丧尸分类表"+"标准遭遇战"
- 末日资源管理 → "弹药经济"+"十大生存法则"
- 末日道德困境 → "道德困境描写"

### 06l-战斗写作手册
- 动作描写 → "通用技巧-具体胜于模糊"
- 战斗节奏 → "张弛有度"+"留白艺术"
- 战斗心理 → "恐惧三层次"+"心理转变"

### 06e-爽点设计手册
- 升级快感 → "升级体系"+"突破描写"
- 打脸爽点 → "打脸技巧"+"蓄力与释放"
- 情感共鸣 → "情感高潮"+"感动点"

### 06f-品类打斗手册调用指南
- **每次任务必先查这本**，确认品类→手册映射
- 填写 required_docs 前 → 先查"品类×手册总表"和"调用顺序"
- 不确定状态文件怎么填 → 参考"状态文件 required_docs 填写规范"
