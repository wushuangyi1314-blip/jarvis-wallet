## 概述

小说大纲模板是步骤8（终版卷纲→小说大纲）的输出规范文档。小说大纲是全书的详细剧情展开，是章节大纲的前置文档。

---

## 必须包含的内容

| 模块 | 说明 |
|------|------|
| 一句话核心卖点 | ≤30字 |
| 主线剧情 | 按时间线展开，5000-10000字 |
| 每卷核心事件列表 | 列出所有卷的核心事件 |
| 主角成长弧光 | 起点→转折→发展→高潮→结局 |
| 重要配角的独立故事线 | 女主、主要反派、主要配角的独立线 |
| 伏笔分布总图 | 标注埋入卷/回收卷 |
| 爽点/高潮点排布 | 全书≥10个高光节点，高潮间隔≤15章 |

---

## 主角成长弧光格式

### 起点

（主角初始状态：身份、处境、能力、性格）

### 转折

（主角遇到的转折点：什么事件改变了主角的命运）

### 发展

（主角的成长过程：经历哪些重要事件，如何成长）

### 高潮

（主角的巅峰时刻：解决了什么问题，达到了什么高度）

### 结局

（主角的最终状态：最终解决了什么，实现了什么）
