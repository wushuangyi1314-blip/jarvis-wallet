## 概述

小说设定模板是步骤6（初版卷纲→小说设定）的输出规范文档。根据品类不同，设定分为通用部分（所有品类都要填）和分类特殊部分（按品类选择填写）。

每个设定项均包含：
- **定义：** 该设定具体描述什么内容
- **格式：** 该设定的输出格式规范

---

# 通用部分（必填）

## 世界规则设定

**定义：** 描述小说世界的根本规则，包括世界构造、层级结构、时间法则、空间法则等。世界规则是设定的基础，决定了故事能发生什么、不能发生什么。

**格式：**

```
世界构造：（一重世界/多重世界/位面/位格等）
世界层级：（凡界→灵界→仙界等，或其他层级结构）
时间法则：（正常时间流/时间可逆/时间加速等）
空间法则：（空间稳定/可撕裂空间/瞬移等）
力量上限：（当前世界最高力量层级）
其他规则：（因果律/天道/命运/气运等特殊规则）
```

---

## 世界矛盾设定

**定义：** 描述世界层面的核心矛盾，即整个世界面临的主要问题或冲突。这是贯穿全书的大背景，影响所有势力的兴衰。

**格式：**

```
世界主要矛盾：（描述世界层面的核心冲突，如正邪对立、种族之争、天道轮回等）
矛盾起源：（矛盾的起源是什么）
矛盾现状：（当前矛盾的局势）
矛盾发展趋势：（矛盾会如何演变）
```

---

## 力量体系设定

**定义：** 描述世界中存在的各种力量类型、力量的来源、力量的修炼方式。力量体系是升级流小说的核心骨架。

**格式：**

```
力量类型：（列出世界上存在的主要力量类型，如灵力、妖力、魔力、斗气等）
力量来源：（力量从何而来，如吸收天地灵气、吞噬万物、感悟天道等）
修炼方式：（力量如何提升，如修炼功法、战斗积累、顿悟等）
核心力量：（最重要的力量体系是什么，其特征是什么）
高层战力特征：（顶级强者的共同特征，如移山填海、寿元万年等）
力量与境界关系：（力量大小与境界的关系）
```

---

## 社会体系设定

**定义：** 描述世界的社会结构、阶层划分、组织形式。社会体系影响角色的出身、身份、社交关系。

**格式：**

```
社会结构：（如宗门林立/王朝统治/城邦割据/联邦制等）
阶层划分：（如贵族/平民/奴隶，或修士/凡人等）
主要组织：（如门派/家族/王朝/商会/佣兵团等）
法律规则：（世界的法律和规则，由谁制定，谁来执行）
社会矛盾：（阶层之间的矛盾和冲突）
```

---

## 等级体系

**定义：** 描述力量/实力的等级划分。等级体系是升级流小说的核心线索，让读者对主角的成长有清晰的感知。

**格式：**

| 等级层次 | 等级名称 | 能力描述 | 数量/稀缺度 |
|---------|---------|---------|------------|
| 1 | （最低级） | （能达到什么能力） | （该等级人数/稀缺程度） |
| 2 | （次低级） | （能达到什么能力） | （该等级人数/稀缺程度） |
| 3 | （中级） | （能达到什么能力） | （该等级人数/稀缺程度） |
| ... | ... | ... | ... |

**说明：** 等级层次建议5-9个层次，等级名称要简洁有力，便于读者记忆。

## 势力设定

**定义：** 描述书中的主要势力，包括其性质（正派/反派/中立）、实力、目的、代表人物。势力是故事的推动者，也是主角成长道路上的障碍或助力。

**格式：**

| 势力名称 | 势力性质 | 势力实力 | 势力目的 | 代表人物 |
|---------|---------|---------|---------|---------|
| （势力1） | （正派/反派/中立） | （强/中/弱） | （势力的目的） | （代表人物） |
| （势力2） | （正派/反派/中立） | （强/中/弱） | （势力的目的） | （代表人物） |
| ... | ... | ... | ... | ... |

## 种族设定

**定义：** 描述世界中的主要种族及其特征。不同种族有不同的天赋、能力、文化、地位，是丰富世界观的重要元素。

**格式：**

| 种族名称 | 种族特征 | 天赋能力 | 种族地位 | 种族文化 |
|---------|---------|---------|---------|---------|
| （种族1） | （外貌/体质特征） | （天赋能力） | （在世界的地位） | （文化特点） |
| （种族2） | （外貌/体质特征） | （天赋能力） | （在世界的地位） | （文化特点） |
| ... | ... | ... | ... | ... |

## 地图设定

**定义：** 描述故事发生的主要地理区域，包括大陆、海洋、山川河流、城市等。地图设定帮助读者建立空间感，也是剧情开展的舞台。

**格式：**

```
主要区域：
- 区域名称：（如中州/东海/北原等）
  地理位置：（描述该区域的地理位置）
  势力分布：（该区域有哪些势力）
  特色描写：（该区域的特色，如灵气充沛/资源丰富/危险等）

城市/据点：
- 城市名：（如青云城/天机阁等）
  城市等级：（如一级城/二级城等）
  势力掌控：（谁掌控该城市）
  特色：（该城市的特色，如拍卖行/情报中心等）
```

## 人物设定

**定义：** 描述书中的主要角色，包括主角、配角、反派。人物设定是人设一致性的锚点，确保角色行为符合其背景和性格。

### 主角设定

**格式：**

| 项目 | 内容 |
|------|------|
| 姓名 | （主角姓名） |
| 性别 | （男/女） |
| 年龄（开篇） | （开篇年龄） |
| 出身 | （出身背景，如偏远山村/没落宗门/大家族等） |
| 性格特质 | （核心性格，如谨慎内敛/热血冲动/腹黑等） |
| 人物弧光 | （成长变化轨迹：起点→转折→高潮→结局） |
| 核心执念 | （最核心的信念/追求，如活下去/复仇/守护等） |
| 能力设定（金手指） | （金手指名称+能力+为何独有+使用代价） |

### 主要反派设定

**格式：**

| 项目 | 内容 |
|------|------|
| 姓名 | （反派姓名） |
| 身份/定位 | （如宗门长老/帝国太子/魔道巨擘等） |

| 与主角关系 | （如宿敌/仇人/利益冲突等） |
| 人物小传 | （200字以内，描述反派的来历、性格、动机） |

### 主要配角设定

**格式：**

| 项目 | 内容 |

| 与主角关系 | （与主角的关系，如师徒/兄弟/恋人/对手等） |
| 人物小传 | （100字以内，描述配角来历和特点） |

### 次要配角设定

**格式：**

| 项目 | 内容 |
|------|------|

## 关系网设定

**定义：** 描述角色之间的关系网络，包括主角与其他角色的关系、各势力之间的关系。关系网是情感线和冲突线的支撑。

**格式：**

```
主角关系网：
- 主角 ↔ （角色名）：（关系类型，如师徒/兄弟/恋人/仇敌）
  关系建立：（如何建立的关系）
  关系发展：（关系会如何发展）
  关系冲突点：（关系中潜在的冲突）

势力关系：
- （势力A） ↔ （势力B）：（关系，如友好/敌对/中立）
  关系原因：（为什么是这种关系）
```

## 主线/支线任务设定

**定义：** 描述主角面临的主要任务和支线任务。主线任务是贯穿全文的核心目标，支线任务是丰富剧情的单元剧。

**格式：**

```
主线任务：
- 任务名称：（如复仇/崛起/守护等）
  任务目标：（具体要达到什么）
  任务障碍：（面临的障碍）
  任务奖励：（完成后获得什么）

支线任务：
- 任务名称：（支线任务名）
  任务类型：（如奇遇/挑战/探索等）
  任务内容：（具体描述）
  任务结果：（任务完成后的结果）
```

---

## 升级路线/奇遇设定

**定义：** 描述主角的成长路线，包括升级路径、关键奇遇、重要突破点。升级路线要有层次感，奇遇要有独特性。

**格式：**

```
升级路线：
- 起点：（主角初始状态）
  第一个成长阶段：（达到什么境界，获得什么）
  第二个成长阶段：（达到什么境界，获得什么）
  ...
  终极目标：（最终要达到什么境界）

关键奇遇：
- 奇遇名称：（如崖底洞府/神秘传承等）
  奇遇时间：（在什么阶段获得的）
  奇遇内容：（具体获得了什么）
  奇遇独特性：（为什么这个奇遇是独特的）
  奇遇代价：（获得奇遇需要付出什么代价）
```
