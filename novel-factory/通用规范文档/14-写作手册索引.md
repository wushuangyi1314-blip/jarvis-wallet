# 写作手册索引

## 概述

写作手册是小说家Agent在创作时的工具参考书，包含人物塑造、打斗描写、反派设计、爽点设计四个领域。每个手册设计为按需查阅的碎片化知识，不是通读型文档。

---

## 手册清单

| 序号 | 手册名 | 用途 | 调用时机 |
|-----|-------|------|---------|
| 13a | 人物塑造手册 | 人物性格/心理/成长/对白描写技巧 | 创作人物戏份前 |
| 13b | 打斗描写手册 | 战斗场面/技能组合/力量对轰描写技巧 | 创作打斗场景前 |
| 13c | 反派设计手册 | 反派类型/动机/行为逻辑/高光设计 | 设计反派或反派出场前 |
| 13d | 爽点设计手册 | 升级快感/打脸爽点/情感共鸣点设计 | 铺垫爽点/高潮前 |

---

## 各手册详细说明

---

### 人物塑造手册（13a）

**用途：** 人物性格、心理、成长弧光、对白描写的技巧参考

**调用时机：**
- 新角色首次出场前
- 重要角色的关键戏份（转折点/高潮戏）前
- 角色关系发生重大变化时
- 发现角色行为不符合人设时

**调用方式：**
```
参考《人物塑造手册》第X章"角色类型"小节
参考《人物塑造手册》第X章"对白技巧"小节
```

**不调用可能出现的质量问题：**
- 角色性格前后矛盾
- 对白千人一面
- 角色成长突兀
- 角色行为缺乏动机

---

### 打斗描写手册（13b）

**用途：** 战斗场面、技能组合、力量对轰、环境互动的描写技巧

**调用时机：**
- 创作任何战斗场景前
- 设计技能对决前
- 主角升级后首次展示新能力前
- 大战场面/群战场景前

**调用方式：**
```
参考《打斗描写手册》第X章"战斗节奏"小节
参考《打斗描写手册》第X章"技能组合"小节
```

**不调用可能出现的质量问题：**
- 战斗描写流水账
- 战斗节奏混乱（过快/过慢）
- 技能展示缺乏层次感
- 战斗结果缺乏铺垫

---

### 反派设计手册（13c）

**用途：** 反派类型、动机、行为逻辑、高光设计技巧

**调用时机：**
- 设计新反派前
- 反派首次出场前
- 反派与主角对决前
- 反派领盒饭前（设计反派的结局）

**调用方式：**
```
参考《反派设计手册》第X章"反派类型"小节
参考《反派设计手册》第X章"反派高光"小节
```

**不调用可能出现的质量问题：**
- 反派工具人化（纯功能性反派）
- 反派动机不合理
- 反派实力忽高忽低
- 反派缺乏高光时刻

---

### 爽点设计手册（13d）

**用途：** 升级快感、打脸爽点、情感共鸣点、读者情绪调动技巧

**调用时机：**
- 铺垫升级/奇遇前
- 设计打脸情节前
- 情感线推进前
- 高潮点/大高潮场景前

**调用方式：**
```
参考《爽点设计手册》第X章"升级爽点"小节
参考《爽点设计手册》第X章"打脸技巧"小节
```

**不调用可能出现的质量问题：**
- 爽点不够爽（铺垫不足）
- 爽点排布过于密集或稀疏
- 读者情绪无法调动
- 高潮点缺乏层层递进

---

## 状态文件中required_docs的填写规则

在状态文件（rd-agent-state.json）中，`required_docs`字段应包含当前批次需要的写作手册章节：

```json
{
  "required_docs": [
    "章节大纲-第3批",
    "伏笔台账",
    "人物塑造手册-第2章-对话技巧",
    "打斗描写手册-第3章-战斗节奏"
  ]
}
```

**填写规则：**
- 必须包含当前批次的章节大纲和伏笔台账
- 必须包含当前批次涉及场景类型对应的写作手册章节
- 如涉及新角色出场，必须包含人物塑造手册相关章节
- 如涉及战斗场景，必须包含打斗描写手册相关章节
- 如涉及反派，必须包含反派设计手册相关章节
- 如涉及升级/高潮，必须包含爽点设计手册相关章节
