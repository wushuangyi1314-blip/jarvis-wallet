# 1、散人逆袭流

> 代表作：《全职高手》《散人玩家》《网游之天下第一》

## 1.1 一句话公式

**散人逆袭流 = "散人玩家" + "技术流" + "以弱胜强"**

---

## 1.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 散人玩家 + 技术流 + 不加入公会 |
| 女主 | 网友/主播 + 后期奔现 |
| 配角 | 公会成员/副本队友 |
| 反派 | 霸榜公会/人民币战士 |

**主角特质：**
- 纯技术流（不花钱）
- 不加入大公会
- 以操作意识碾压

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 散人玩家+副本记录+被关注 |
| 困境（10-30章） | 公会打压/副本受阻 |
| 机遇（每20章） | 技术展示+碾压土豪 |
| 崛起（30-60章） | 成为散人传说 |
| 巅峰（60-80章） | 打败所有公会 |
| HE（80-100章） | 巅峰神话+抱得美人 |

### 【套路公式】
**核心循环：**
```
副本→记录被破→技术碾压→散人崛起
```
**爽点设计：**
```
1. 副本记录被主角超越
2. 土豪打不过操作
3. 散人抱团取暖
4. 以弱胜强
```

---

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 散人崛起 | 成为传说 |
| 感情线 | 网友奔现 |
| 代价 | 技术流辛苦 |
| 核心 | 操作意识碾压人民币 |

---

## 1.3 写散人逆袭流必有的要素

✅ 技术操作描写详细
✅ 散人不加入公会
✅ 以弱胜强的爽感
✅ 土豪打不过操作的快感
✅ 副本记录争夺

❌ 不要写太多感情（影响游戏线）

---

## 1.4 避开的坑

❌ 游戏设定要有逻辑
❌ 不要写太多公会内斗
❌ 土豪不要一下就被打败（要有过程）
❌ 不要写太多外挂/脚本（违规）

---

✅ 提前布局（核心）
✅ 复仇线

❌ 仇人不要太弱智
