## 概述

小说设定模板是步骤6（初版卷纲→小说设定）的输出规范文档。根据品类不同，设定分为通用部分（所有品类都要填）和分类特殊部分（按品类选择填写）。

---

## 通用部分（必填）

| 设定项 | 说明 |
|-------|------|
| 世界规则设定 | 世界的规则、构造、层级 |
| 世界矛盾设定 | 世界层面的核心矛盾 |
| 力量体系设定 | 世界存在的力量体系 |
| 社会体系设定 | 社会结构、阶层、组织 |
| 等级体系 | 力量/实力等级划分 |
| 势力设定 | 主要势力及其性质、目的 |
| 种族设定 | 主要种族及其特征 |
| 地图设定 | 主要地图/区域 |
| 人物设定 | 主角+主要反派+主要配角 |
| 关系网设定 | 角色之间的关系网络 |
| 主线/支线任务设定 | 主要任务和支线任务 |
| 升级路线/奇遇设定 | 主角的升级路线和奇遇安排 |

---

## 分类特殊部分（按品类选填）

| 品类 | 特殊设定项 |
|------|-----------|
| 网游 | 技能设定、装备设定、道具设定、公会/团队设定、副本设定 |
| 玄幻/仙侠 | 功法设定、灵根/资质设定、炼丹/炼器设定、宗门设定 |
| 都市 | 职业设定、商业设定、身份设定（隐藏身份/双重身份） |
| 末日/废土 | 丧尸/怪物设定、基地设定、资源设定、阵营设定 |
| 科幻 | 科技设定、星际设定、舰船/武器设定 |
| 历史 | 时代背景设定、势力/门派设定、武学设定 |
