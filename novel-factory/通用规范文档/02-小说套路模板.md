# 一、凡人流

> 代表作：《凡人修仙传》
> 数据来源：对话记录-2026-04-16/17

## 1.1 一句话公式

**凡人流 = "谨慎者生存"而非"天才者胜利"**

---

## 1.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 普通人 + 谨慎到极致 + 有限温情 + 小绿瓶外挂 |
| 女主 | 出场惊艳但戏份极少 + 聚少离多 |
| 配角 | 每一阶段有一个核心NPC + 师徒/兄弟/情人三线 |
| 反派 | 比主角强一点 + 有自己的故事 + 打脸后变踏脚石 |

**主角特质：**
- 出身：山村少年 / 凡人 / 资质平庸
- 性格：谨慎到极致，不浪，每步皆计算
- 金手指：小绿瓶 / 残魂指导 / 家族传承（有限度）
- 温情：极少，极克制
- 结局：永远在飞升，永远在路上

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 山村少年→发现小绿瓶→门派记名弟子 |
| 困境（10-50章） | 资源稀缺 + 敌人压迫 + 资质平庸 |
| 机遇（每10章） | 意外获得灵药/功法/宝物 |
| 脱险（每20章） | 用智谋/阵法/宝物反杀或逃跑 |
| 升级（每100章） | 境界突破 + 实力飞跃 + 新敌人出现 |
| 大高潮（每400章） | 飞升新世界 |

### 【套路公式】

**核心循环：**
```
遇险→计算→脱身→反杀→升级→更强敌人→遇险
```

**资源获取优先级：**
```
小绿瓶催化（主要）+ 捡漏（辅助）+ 交易（补充）
```

**战斗模式：**
```
能不打就不打 → 打就必有后手 → 打不过就跑 → 跑不掉就用命换
```

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 没有结局 | 永远是"修炼-升级-飞升"的循环 |
| 感情线 | 克制到几乎没有 |
| 代价 | 每一次进步都要付出（时间/鲜血/寿命/感情） |
| 核心 | 韩立最大的运气不是小绿瓶，是"总能积累剩余资源化解人情债" |

---

## 1.3 写凡人流必有的要素

✅ 小绿瓶类金手指（催化植物，加速修炼）
✅ 谨慎型主角（不浪，每步都计算）
✅ 资源稀缺感（永远缺灵石/缺功法/缺灵药）
✅ 越级挑战用智谋（阵法/傀儡/毒药/逃跑）
✅ 每一百章一次大境界突破
✅ 每一千章一次飞升新世界

❌ 不要写太多感情戏（读者反馈感情戏是毒点）
❌ 不要让主角太霸气（苟道流核心是低调）

---

## 1.4 避开的坑

❌ 前100章不要让主角太出风头（需要铺垫）
❌ 不要让女主戏份超过剧情的20%（读者会吐槽）
❌ 不要在同一个套路重复超过3次（审美疲劳）
❌ 800章后要有实质性的剧情升级（否则读者弃书）

# 二、退婚打脸流

> 代表作：《斗破苍穹》
> 数据来源：对话记录整理

## 2.1 一句话公式

**退婚打脸流 = "三年之辱，莫欺少年穷"**

---

## 2.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 天才陨落 + 坚韧不屈 + 逆袭打脸 + 不断提升 |
| 女主 | 大家闺秀 + 悔婚 + 后期追夫 |
| 配角 | 师父/药老 + 兄弟/战友 + 红颜知己 |
| 反派 | 嚣张跋扈 + 看不起主角 + 打脸变踏脚石 |

**主角特质：**
- 出身：曾经的天才/被退婚/被嘲笑
- 性格：坚韧、隐忍、有仇必报
- 金手指：药老/戒指/传承/特殊体质
- 成长弧光：废物→天才→强者

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 天才陨落→被退婚→获得金手指（药老） |
| 困境（10-30章） | 资质恢复+修炼→被人看不起 |
| 机遇（每10章） | 获得功法/丹药/技能 |
| 打脸（每20章） | 当年嘲笑我的人→狠狠打脸 |
| 升级（每100章） | 境界突破 + 更大舞台 |
| 大高潮（每300章） | 打败未婚妻/超越当年强者 |

### 【套路公式】

**核心循环：**
```
被打脸→隐忍→实力提升→当众打脸→更强敌人→再被打脸
```

**爽点设计：**
```
打脸场景 = 众人面前 + 意想不到 + 彻底碾压
```

**经典打脸结构：**
```
1. 对方嘲讽主角
2. 众人围观
3. 主角展示实力
4. 对方震惊/打脸
5. 收获好处（丹药/功法/声望）
```

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 大团圆 | 功成名就 + 美人相伴 |
| 感情线 | 有明确女主，感情线清晰 |
| 代价 | 修炼路上的艰辛和牺牲 |
| 核心 | 三年之辱，今日雪耻 |

---

## 2.3 写退婚打脸流必有的要素

✅ 退婚/被嘲笑的屈辱开局
✅ 隐忍蓄势阶段
✅ 金手指（药老/系统/传承）
✅ 不断的打脸场景（每隔20-30章一次大打脸）
✅ 每一百章一次大境界突破
✅ 当年女神后悔的爽感

❌ 不要让主角一直被动挨打不反击
❌ 打脸要干脆，不能拖泥带水

---

## 2.4 避开的坑

❌ 开局不要拖太久，50章内要有第一次大爽点
❌ 打脸场景不能太雷同，要换花样
❌ 感情线不要写太多三角恋（读者会弃书）
❌ 后期不要让主角变成无脑打脸机器，要有智谋

---

# 三、系统流

> 代表作：《我这么天才》《最强反套路系统》
> 数据来源：对话记录整理

## 3.1 一句话公式

**系统流 = "任务+奖励+代价" = 反套路+轻松幽默**

---

## 3.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 轻松幽默 + 反套路 + 系统任务驱动 |
| 女主 | 可有可无/轻松搞笑/不纠缠 |
| 配角 | 系统/宠物/搞笑同伴 |
| 反派 | 工具人反派 + 送经验包 |

**主角特质：**
- 性格：轻松幽默、不走寻常路
- 金手指：最强系统/反套路系统
- 与传统区别：带玩家视角，有第四面墙感
- 核心：完成任务→变强→再任务

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 获得系统→第一个任务→奖励到手 |
| 困境（10-30章） | 任务难度提升/特殊任务 |
| 机遇（任务奖励） | 完成系统任务获得奖励 |
| 反转（每20章） | 反套路操作让敌人吃亏 |
| 升级（任务驱动） | 任务奖励→实力提升→更难任务 |
| 大高潮 | 完成系统主线任务/终极任务 |

### 【套路公式】

**核心循环：**
```
接任务→完成任务→获得奖励→变强→新任务
```

**系统设计要素：**
```
任务类型：主线/支线/日常/隐藏
奖励类型：技能/道具/属性点/称号
代价：完成任务需要付出代价/限制
```

**反套路核心：**
```
1. 系统是吐槽役
2. 主角操作反套路
3. 任务完成方式出人意料
4. 敌人一脸懵逼
```

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 大圆满 | 系统成神/开创纪元 |
| 感情线 | 可有可无，轻松向 |
| 代价 | 完成任务消耗/系统副作用 |
| 核心 | 系统是工具，不是主角 |

---

## 3.3 写系统流必有的要素

✅ 明确的任务系统（主线/支线/日常）
✅ 有趣的反套路操作
✅ 系统吐槽/系统面板展示
✅ 任务奖励→实力提升的正循环
✅ 每一50章一次大任务奖励

❌ 不要让系统太强（压制主角）
❌ 不要只有任务没有日常（太紧绷）

---

## 3.4 避开的坑

❌ 不要开局就给出终极任务（节奏会乱）
❌ 任务不能太重复（审美疲劳）
❌ 系统不要太智能（有自主意识会抢戏）
❌ 代价不要太轻（没紧张感）

---

# 五、家族修仙流

> 代表作：《玄鉴仙族》
> 数据来源：对话记录-2026-04-16/17

## 5.1 一句话公式

**家族修仙流 = "家族百年兴衰" = "个人英雄融入家族命运"**

---

## 5.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 家族一员 + 能力有限 + 为家族牺牲 + 群像中突出 |
| 女主 | 家族联姻/利益结合 + 感情慢慢培养 |
| 配角 | 家族成员群像 + 各有性格 + 多代传承 |
| 反派 | 敌对家族 + 生存竞争 + 残酷法则 |

**主角特质：**
- 出身：家族子弟（非天才但努力）
- 性格：责任感强、为家族牺牲
- 金手指：家族传承/特殊血脉
- 群像：每个家族成员都有自己故事

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 家族困境→主角出场→第一次危机 |
| 困境（10-50章） | 家族内斗/外敌入侵/资源匮乏 |
| 机遇（每10章） | 主角获得机遇+带动家族发展 |
| 牺牲（每50章） | 家族成员牺牲→感动读者 |
| 升级（每100章） | 家族变强→更大舞台 |
| 大高潮 | 家族传承延续/大兴 |

### 【套路公式】

**核心循环：**
```
家族危机→主角应对→家族发展→更强敌人→更大危机
```

**家族核心矛盾：**
```
- 资源有限，必须争夺
- 家族利益vs个人利益
- 传承与牺牲
```

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 家族延续 | 不是个人结局，是家族百年传承 |
| 感情线 | 次要，服务于家族线 |
| 代价 | 牺牲（小人物也有高光） |
| 核心 | 个人融入家族命运，家族兴则我兴 |

---

## 5.3 写家族修仙流必有的要素

✅ 家族群像（每个成员有血有肉）
✅ 生存压力（吃人修仙残酷法则）
✅ 传承感（一代代人的努力）
✅ 牺牲场景（动情点）
✅ 家族荣誉感

❌ 不要没有主角感（群像也要有核心人物）
❌ 不要牺牲太频繁（会麻木）

---

## 5.4 避开的坑

❌ 不要写太多人物（记不住）
❌ 不要牺牲太随意（要写的有意义）
❌ 节奏太慢会弃书
❌ 不要没有爽点（一直憋屈）

---

# 六、序列流

> 数据来源：对话记录-2026-04-16

## 6.1 一句话公式

**序列流 = "扮演好序列角色，才能晋升更高序列"**

---

## 6.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 理智冷静 + 扮演法 + 序列晋升 |
| 女主 | 可有可无/剧情重要NPC |
| 配角 | 同一序列的竞争者/盟友 |
| 反派 | 更高序列存在/敌对势力 |

**主角特质：**
- 性格：理智冷静、善于伪装
- 金手指：序列系统/扮演法
- 核心：扮演角色→获得能力→晋升

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 获得序列/穿越→了解世界规则 |
| 困境（10-30章） | 序列竞争/扮演困难/敌人追杀 |
| 机遇（每10章） | 扮演成功→获得新技能 |
| 战斗（每20章） | 序列战+情报战 |
| 升级（每100章） | 序列晋升+扮演更高级角色 |
| 大高潮 | 达到序列1/成为旧日 |

### 【套路公式】

**核心循环：**
```
扮演→获得能力→遇到敌人→用扮演能力战斗→晋升
```

**序列设计要素：**
```
1. 序列9→序列1（9个等级）
2. 每个序列有独特能力
3. 序列之间有相克关系
4. 扮演法：扮演越像，实力越强
```

**战斗模式：**
```
不是一力降十会，是用情报和策略
```

### 【结局公式】

| 要素 | 说明 |
|------|------|
| 晋升终点 | 成为旧日/支柱/真神 |
| 感情线 | 可有可无 |
