## 概述

小说套路模板提供各品类的套路公式，是步骤1（创意→主线故事）和步骤4（主线故事→核心设定）的重要参考文档。

---

## 玄幻/仙侠类套路

### 凡人流

**核心特征：**
- 主角资质平庸/低下
- 资源获取困难，每一步都艰难
- 升级路线清晰，等级分明
- 强调机缘、毅力、战斗智慧

**套路公式：**
```
起点（资质低下）→ 奇遇（获得机缘）→ 成长（艰难升级）→ 越级挑战（以弱胜强）→ 劫难（生死危机）→ 突破（更高境界）
```

### 退婚流

**核心特征：**
- 开篇被轻视/退婚
- 主角隐藏实力或获得逆袭
- 打脸环节明确
- 后期对方后悔

**套路公式：**
```
开篇（被轻视/退婚）→ 隐藏（展示部分实力）→ 打脸（让对方后悔）→ 后续（持续碾压）
```

### 系统流

**核心特征：**
- 主角获得系统
- 系统提供任务/奖励
- 升级有明确指引
- 轻松幽默风格

**套路公式：**
```
获得系统 → 完成任务 → 获得奖励 → 变强 → 更多任务
```

### 掌门流

**核心特征：**
- 主角建立或成为掌门
- 发展门派/宗门
- 培养弟子
- 门派战争

**套路公式：**
```
创立门派 → 招揽人才 → 发展壮大 → 门派纷争 → 成为顶级门派
```

### 洪荒流

**核心特征：**
- 设定宏大，多重空间
- 力量体系复杂
- 涉及天道、因果
- 圣人大战

**套路公式：**
```
转世/重生 → 修炼 → 证道 → 圣人博弈 → 合道
```
