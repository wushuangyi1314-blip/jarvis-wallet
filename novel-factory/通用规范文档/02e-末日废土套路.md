# 1、丧尸围城流

> 代表作：《全球进化》《末日流放者》《丧尸之都会》

## 1.1 一句话公式

**丧尸围城流 = "丧尸爆发" + "生存挣扎" + "人类存续"**

---

## 1.2 四维度完整公式

### 【人物公式】

| 角色 | 公式 |
|------|------|
| 主角 | 普通人 + 觉醒异能 + 带领团队 |
| 女主 | 队友 + 生死与共 |
| 配角 | 幸存者/队友/兄弟 |
| 反派 | 丧尸潮/变异兽/恶势力 |

**主角特质：**
- 普通人觉醒
- 带领团队生存
- 寻找安全区

---

### 【情节公式】

| 阶段 | 内容 |
|------|------|
| 开局（五章） | 丧尸爆发+发现异能 |
| 困境（10-20章） | 丧尸围城/资源匮乏 |
| 机遇（每10章） | 发现安全区/异能提升 |
| 崛起（20-50章） | 建立基地/队伍壮大 |
| 巅峰（50-70章） | 对抗丧尸潮 |
| HE（70-100章） | 人类存续/重建希望 |

### 【套路公式】

**核心循环：**
```
丧尸围城→异能觉醒→团队协作→建立基地→更大威胁
```

**爽点设计：**
```
1. 觉醒异能（进化）
2. 丧尸围城逃出生天
3. 建立幸存者基地
4. 对抗变异兽
```

---

### 【结局公式】


| 要素 | 说明 |
|------|------|
| 人类存续 | 重建希望 |
| 感情线 | 生死与共 |
| 代价 | 队友牺牲 |
| 核心 | 人类不屈的意志 |

---

## 1.3 写丧尸围城流必有的要素

✅ 丧尸围城（核心压力）
✅ 异能觉醒（进化）
✅ 资源匮乏
✅ 团队协作
✅ 幸存者基地

❌ 丧尸不要太弱（失去紧张感）

---

## 1.4 避开的坑

❌ 不要写太多恋爱（末日氛围要紧张）
❌ 不要忽视生存压力
❌ 配角不要太多（记不住）
❌ 不要突然无敌
