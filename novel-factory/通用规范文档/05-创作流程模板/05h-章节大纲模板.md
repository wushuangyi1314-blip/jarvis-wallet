## 概述

章节大纲模板是步骤9（小说大纲→章节大纲）的输出规范文档。章节大纲是小说创作的直接指导文档，每章都有明确的创作指引。

---

## 必须包含的内容

### 单章格式

每章必须包含：

| 项目 | 说明 |
|------|------|
| 章号 | 第X章 |
| 章名 | （章名） |
| 本章核心事件 | **50-100字**，清晰描述本章核心人物+核心事件，包含"谁"做了什么/导致了"什么结果" |
| 本章出场人物 | 列出出场人物（主要人物用**加粗**） |
| 信息量指标 | 推进⬆️ / 埋伏笔📌 / 回收伏笔🔄 / 高潮🔥 |
| 与上章衔接 | 与上一章的衔接点，首章填"首章" |
| 与下章衔接 | 与下一章的衔接点 |

**核心事件字数标准：**
- 最少：50字（核心人物+核心动作+结果）
- 最多：100字
- 标准：60-80字

**不合格示例（过于简单）：**
- "韩立离开山村"（仅13字，缺少结果）
- "破晓舰队进入虫族区域"（仅14字）

**合格示例：**
- "韩立决定离开山村，跟随三叔外出学艺，但父母不舍，三叔承诺照顾"（52字）
- "破晓舰队进入虫族活跃区域，联盟却下达不许还击命令，舰队陷入被动挨打局面"（56字）

## 信息量指标说明

| 符号 | 含义 | 说明 |
|------|------|------|
| 推进⬆️ | 本章有实质性剧情推进 | 主角有明确行动+结果 |
| 埋伏笔📌 | 本章埋下了伏笔 | 需要记录到伏笔台账 |
| 回收伏笔🔄 | 本章回收了伏笔 | 验证伏笔是否自洽 |
| 高潮🔥 | 本章有高潮点 | 情绪爆发/重大转折 |

每章至少要有1个信息量指标，最好有2个以上。
