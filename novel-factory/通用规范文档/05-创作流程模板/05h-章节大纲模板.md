## 概述

章节大纲模板是步骤9（小说大纲→章节大纲）的输出规范文档。章节大纲是小说创作的直接指导文档，每章都有明确的创作指引。

---

## 必须包含的内容

### 单章格式

每章必须包含：

| 项目 | 说明 |
|------|------|
| 章号 | 第X章 |
| 章名 | （章名） |
| 本章核心事件 | **120-200字**，清晰描述本章核心人物+核心动作+结果，包含"谁"在"什么场景"做了"什么事"导致了"什么结果" |
| 本章出场人物 | 列出出场人物（主要人物用**加粗**） |
| 信息量指标 | 推进⬆️ / 埋伏笔📌 / 回收伏笔🔄 / 高潮🔥 |
| 与上章衔接 | 与上一章的衔接点，首章填"首章" |
| 与下章衔接 | 与下一章的衔接点 |

**核心事件字数标准：**
- 最少：120字
- 最多：200字
- 标准：140-180字

**不合格示例（过于简单）：**
- "韩立离开山村"（9字，缺少人物和结果）
- "破晓舰队进入虫族区域"（14字，缺少结果）

**合格示例（120字以上）：**
- "韩立不愿在山村度过一生，决定跟随三叔外出学艺；临行前父母万般不舍，韩立含泪告别，三叔承诺会将韩立安全带回。七玄门位于七玄山深处，三叔告诉韩立，那里可以让他出人头地，韩立对山外的世界充满期待"（120字，勉强达标）

## 信息量指标说明

| 符号 | 含义 | 说明 |
|------|------|------|
| 推进⬆️ | 本章有实质性剧情推进 | 主角有明确行动+结果 |
| 埋伏笔📌 | 本章埋下了伏笔 | 需要记录到伏笔台账 |
| 回收伏笔🔄 | 本章回收了伏笔 | 验证伏笔是否自洽 |
| 高潮🔥 | 本章有高潮点 | 情绪爆发/重大转折 |

每章至少要有1个信息量指标，最好有2个以上。

## 模板样例（以玄幻小说第1-3章为例）

### 单章格式样例

**第1章：山村少年**

| 项目 | 内容 |
|------|------|
| 章号 | 第1章 |
| 章名 | 山村少年 |
| 本章核心事件 | 韩立不愿在山村度过一生，决定跟随三叔外出学艺；临行前父母万般不舍，韩立含泪告别，三叔承诺会将韩立安全带回。七玄门位于七玄山深处，三叔告诉韩立，那里可以让他出人头地，韩立对山外的世界充满期待（120字） |
| 本章出场人物 | **韩立**、韩立父母、**三叔** |
| 信息量指标 | 推进⬆️ |
| 与上章衔接 | 首章 |
| 与下章衔接 | 韩立跟随三叔离开山村，踏上前往七玄门的旅途 |

**第2章：路上遇险**

| 项目 | 内容 |
|------|------|
| 章号 | 第2章 |
| 章名 | 路上遇险 |
| 本章核心事件 | 韩立与三叔途经山林，遭遇劫匪埋伏，三叔腿部中刀身受重伤；韩立利用地形机智引开劫匪，将劫匪引入悬崖绝路，成功脱身。三叔因失血过多昏迷，韩立背起三叔连夜赶路，终于找到一处猎户人家借宿疗伤（126字） |
| 本章出场人物 | **韩立**、**三叔**、劫匪、猎户 |
| 信息量指标 | 推进⬆️、埋伏笔📌 |
| 与上章衔接 | 韩立跟随三叔离开山村，踏上前往七玄门的旅途 |
| 与下章衔接 | 三叔带韩立抵达七玄门，伤势已有好转，韩立开始七玄门新生活 |
