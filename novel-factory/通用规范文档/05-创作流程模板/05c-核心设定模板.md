---

## 概述

核心设定模板是步骤4（主线故事→核心设定）的输出规范文档。用于指导小说核心设定的输出格式，确保核心设定包含所有必要信息。

---

## 必须包含的模块

| 模块 | 必填 | 说明 |
|------|-----|------|
| 世界规则 | ✅ | 几重世界/位面，主世界特征，力量上限 |
| 力量体系 | ✅ | 存在哪几种力量，核心力量是什么，高层战力特征 |
| 核心矛盾 | ✅ | 主角要解决什么问题，最大障碍是什么，对手势力是谁 |
| 主角设定 | ✅ | 基础信息+性格特质+人物弧光+核心执念+能力设定（金手指） |
| 女主设定 | ✅ | 单女主/多女主/无女主+女主类型+基础信息+与主角关系定位 |
| 其他重要角色 | ⚠️ | 1-2个核心配角简要设定（后续随故事补充） |

---

## 各模块详细说明

### 世界规则

```
几重世界/位面：（描述世界层级结构）
主世界特征：（描述主要世界的核心特征）
力量上限：（当前世界的最高力量层级是什么）
```

### 力量体系

```
存在哪几种力量：（列出世界上存在的主要力量类型）
核心力量是什么：（最重要的力量体系是什么）
高层战力特征：（描述顶级强者的共同特征）
```

### 核心矛盾

```
主角要解决什么问题：（主角的终极目标）
最大障碍是什么：（主角面临的最大困难/困境）
对手势力是谁：（主要对手或反派势力）
```

### 主角设定

| 子模块 | 内容要求 |
|-------|---------|
| 基础信息 | 姓名/性别/年龄/出身 |
| 性格特质 | 性格特点描述，要具体不要空洞 |
| 人物弧光 | 成长变化轨迹：起点→转折→高潮→结局 |
| 核心执念 | 主角最核心的信念/执念是什么 |
| 能力设定（金手指） | 必须包含一项独特金手指，说明为何独有 |

### 女主设定

| 子模块 | 内容要求 |
|-------|---------|
| 女主角数 | 单女主/多女主/无女主 |
| 女主类型 | 清冷型/活泼型/霸道型/温柔型/其他 |
| 基础信息 | 女主姓名、年龄、身份 |
| 与主角关系定位 | 如何与主角相遇，关系发展定位 |

### 其他重要角色（1-2个核心配角）

| 子模块 | 内容要求 |
|-------|---------|
| 基础信息 | 姓名、性别、身份/定位 |
| 与主角关系 | 与主角的关系 |
| 核心特质 | 1-2句话描述核心特点 |

---

## 输出格式示例

```markdown
## 世界规则

| 项目 | 内容 |
|------|------|
| 世界数量 | 一重世界（凡人界+修仙界） |
| 主世界特征 | 凡人界：世俗王朝；修仙界：宗门林立 |
| 力量上限 | 渡劫期（飞升后进入仙界） |

## 力量体系

| 项目 | 内容 |
|------|------|
| 存在哪几种力量 | 灵力、妖力、魔力 |
| 核心力量是什么 | 灵力（修仙主流） |
| 高层战力特征 | 大能者移山填海，寿元万年 |

## 主角设定

| 项目 | 内容 |
|------|------|
| 姓名 | 韩立 |
| 性别 | 男 |
| 年龄 | 18岁（开篇） |
| 出身 | 偏远山村普通人 |
| 性格特质 | 谨慎内敛，不爱出头，低调行事 |
| 人物弧光 | 普通人→修士→宗门弟子→修仙界大能 |
| 核心执念 | 活着，活得更久 |
| 能力设定 | 掌天瓶（神秘小绿瓶，可加速灵药成熟），唯一性：来历不明，无法抢夺 |
```
