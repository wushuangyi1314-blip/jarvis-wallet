```
xl:  32px  → 区块间距
2xl: 48px  → 大区块间距
3xl: 64px  → 页面留白
```

## 🎬 动效速查

### 时长
```
微交互:  100-150ms → 悬停、焦点
组件:    200-300ms → 展开、折叠
页面:    300-400ms → 路由切换
复杂:    400-600ms → 加载动画
```

### 缓动曲线
```css
标准:   cubic-bezier(0.4, 0, 0.2, 1)
进入:   cubic-bezier(0, 0, 0.2, 1)
退出:   cubic-bezier(0.4, 0, 1, 1)
弹性:   cubic-bezier(0.175, 0.885, 0.32, 1.275)
```

### 常用动效
```
淡入:   opacity 0→1, 200ms, ease-out
淡出:   opacity 1→0, 200ms, ease-in
上浮:   translateY(8px)→0, 200ms
下压:   scale 1→0.98, 100ms
展开:   height auto, 300ms
旋转:   rotate 0→90deg, 200ms
```

## 📐 布局速查

### 断点
```
Mobile:  < 640px   → 4列
Tablet:  640-1024px → 8列
Desktop: 1024-1280px → 12列
Wide:    > 1280px  → 12列
```

### 页面结构
```
顶部导航: 56-64px
内容区最大宽度: 1280px
桌面内边距: 24px
移动端内边距: 16px
```

### 栅格
```
列数: 12列
槽宽: 24px (桌面) / 32px (宽屏)
外边距: 自动居中
```

## ♿ 无障碍速查

### 对比度
```
AA级 (最低): 正文 4.5:1 / 大文字 3:1
AAA级 (推荐): 正文 7:1 / 大文字 4.5:1
```

### 交互
```
焦点可见: 所有可交互元素有:focus样式
键盘支持: Tab导航 / Enter确认 / Esc取消
ARIA: 图标按钮需aria-label
```

## ✅ 设计检查清单

### 新页面/组件
- [ ] 符合8px网格系统
- [ ] 色彩对比度 ≥ 4.5:1
- [ ] 有悬停/聚焦/禁用状态
- [ ] 动效时长 100-300ms
- [ ] 移动端适配
- [ ] 键盘可访问

### 设计评审
- [ ] 一致性 - 与现有设计模式一致?
- [ ] 层次 - 视觉权重正确传达优先级?
- [ ] 空白 - 有足够的呼吸空间?
- [ ] 聚焦 - 用户视线被正确引导?
- [ ] 克制 - 没有不必要的元素?
