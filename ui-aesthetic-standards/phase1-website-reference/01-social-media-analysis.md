## 📊 设计趋势总结

### 共同模式
1. **极致留白**: 所有顶级网站都大量使用负空间
2. **有限配色**: 1-2个主色 + 中性色背景
3. **粗体标题**: 大字体(48-84px)建立层级
4. **微妙阴影/圆角**: 不用厚重阴影, 用圆角(8-24px)和色差
5. **产品/截图作为英雄图**: 不用库存照片, 用实际产品UI

### 布局趋势
- **Z型/F型模式**: 标准着陆页扫描模式
- **分裂式英雄区**: 40-60%文字/图片分割
- **Bento Box网格**: 不规则卡片网格(如Apple)
- **透明导航**: 与内容融合的导航栏

### 按钮风格
- **药丸形/胶囊按钮**: 完全圆角或4-8px圆角
- **幽灵按钮**: 透明背景+边框作为次要CTA
- **高对比主CTA**: 白底黑字或品牌色
