# Phase 3: AI工具评测网站风格建议

## 背景
基于对11个顶级网站的截图分析和Apple HIG + Material Design规范学习，为AI工具评测网站提供风格定位和设计建议。

---

## 1. 风格定位

### 目标用户画像
- 科技爱好者、AI从业者、开发者
- 追求效率和品质的理性用户
- 需要客观、中立、深度评测的读者

### 推荐风格: "学术现代极简" (Academic Modern Minimal)

#### 风格描述
结合Anthropic的学术感和Linear的开发者美学，创造一个既有权威感又现代的评测平台。

#### 关键词
- **权威**: 专业的视觉表达
- **清晰**: 信息层次分明
- **克制**: 不过度设计
- **现代**: 符合当前设计趋势

#### 参考网站
1. **Anthropic** - 温暖的学术感, serif标题混用
2. **Linear** - 极简黑色美学, 高对比度
3. **Figma** - 清晰的产品展示方式

## 5. 字体方案建议

### 主字体: Inter (Google Fonts)
```
理由:
- 现代无衬线, 专业感
- 广泛使用, 熟悉度高
- 适合数字屏幕
- 支持多语言
```

### 标题字体: 混用Inter + Source Serif (可选)
```
方案A: 纯Inter
- 所有文字使用Inter
- 通过weight和size区分层级

方案B: Serif混用 (Anthropic风格)
- 正文/UI: Inter
- 大标题: Source Serif 4 (学术感)
```

### 字号系统

```
H1 (主标题): 48px / 56px, Bold (700), 行高1.1
H2 (区块标题): 32px, Bold (700), 行高1.2
H3 (卡片标题): 24px, Semibold (600), 行高1.3
H4 (小标题): 18px, Semibold (600), 行高1.4

Body Large: 18px, Regular, 行高1.6
Body: 16px, Regular, 行高1.6
Body Small: 14px, Regular, 行高1.5

Caption: 12px, Regular, 行高1.4
Label: 12px, Medium (500), 行高1.2, 字间距+0.5
```

### 字间距
```
标题: -0.02em (略紧凑)
正文: 0 (正常)
Label/Caption: +0.02em (略宽松)
```

## 6. 组件设计规范

### 工具卡片 (Tool Card)

```
尺寸:
- 宽度: 自适应栅格 (最小240px)
- 高度: 自适应内容
- 圆角: 16px

背景: #FFFFFF (Light) / #141414 (Dark)
边框: 1px solid #E5E7EB (Light) / #374151 (Dark)

阴影:
- 默认: 无阴影
- 悬浮: 0 4px 12px rgba(0,0,0,0.08)

内边距: 20px

内容:
- Logo: 48x48px, 圆角8px
- 名称: H4
- 简介: Body Small, 2行截断
- 标签: 胶囊形, 8px 圆角
- 评分: 5星系统 + 数字
```

### 按钮样式

```

- 背景: #6366F1 (Primary)
- 文字: #FFFFFF
- 高度: 44px
- 圆角: 8px
- Padding: 16px 24px
- 字体: 16px, Medium

Secondary Button:
- 背景: 透明
- 边框: 1px solid #6366F1
- 文字: #6366F1

Ghost Button:
```

```
- Default: 背景#F3F4F6, 文字#374151

```

```
Logo: 左侧
导航链接: 居中, 间距24px
操作按钮: 右侧
```

## 7. 动效建议

### 时长
```
微交互: 150ms
页面元素: 200ms
页面过渡: 300ms
```

### 缓动
```
进入: ease-out (减速)
退出: ease-in (加速)
悬浮: ease-in-out
```

### 动效场景
```
1. 卡片悬浮: translateY(-4px), 阴影加深
2. 按钮点击: scale(0.98), 150ms
3. 页面切换: fade + slide, 300ms
4. 图片加载: fade in, 200ms
5. 下拉展开: height auto, 300ms
```

## 8. 总结

### 设计清单

- [ ] 暖白/深黑双主题
- [ ] Indigo品牌色 #6366F1
- [ ] Inter字体 + Source Serif可选
- [ ] 8pt网格系统
- [ ] 大圆角卡片(16px)
- [ ] 微妙阴影(悬浮时)
- [ ] 清晰评分系统
- [ ] 快速筛选标签
- [ ] 产品截图展示区
- [ ] 响应式Bento网格

### 关键设计原则

1. **内容优先**: UI不与内容竞争
2. **一致体验**: 所有页面遵循同一视觉语言
3. **性能意识**: 动画不过度, 加载要快
4. **无障碍**: 满足WCAG AA标准
5. **移动优先**: 先设计移动端, 再放大到桌面
