## 模型A：追妻火葬场（现代言情）

### 基础参数

| 指标 | 数值 |
|------|------|
| 总字数 | 30,000字 |
| 章节数 | 30章 |
| 每章字数 | 1000字 |
| 节奏周期 | 每3章一个小高潮 |
| 主线 | 虐恋 → 真相 → 追妻 → 甜宠 |

### 用户画像

| 维度 | 描述 |
|------|------|
| 目标读者 | 18-35岁女性 |
| 核心诉求 | 情绪释放、身份代入、甜虐交织 |
| 读点 | 前期虐女主爽（等男主追妻）、后期甜宠 |
| 雷区 | 女主太傻白甜、男主太渣无底线、结局太虐 |

### 章节结构（30章）

#### 【第一阶段：强冲突开场】（第1-5章）

目标：制造"为什么不解释"的悬念，抓住读者

**第1章：开场即高能**

| 项目 | 内容 |
|------|------|
| 字数 | 1000字 |
| 核心事件 | 婚礼现场被当众抛弃 |
| 情绪 | 震惊/愤怒/委屈 |
| 必须包含 | 男主说出台词："她害死了我妹妹，这婚今天必须离！" |

**第1章详细设计：**

【开场画面】
酒店婚礼现场，宾客满座，林念念穿着婚纱站在红毯上。

【核心冲突】
大屏幕上突然播放一段监控录像——"林念念"推倒了顾清婉，她从楼梯滚落，当场昏迷。

【高能台词】
顾霆深当着所有人的面，将离婚协议砸在她脸上：
"林念念，清婉还在ICU，你居然还有心情办婚礼？"
"签字，离婚。"

【结尾钩子】
林念念捡起离婚协议，看到上面写的内容，瞳孔地震——
协议上写的是"净身出户"，但她手里攥着的另一份文件，另有玄机。

---

**第2章：净身出户**

| 项目 | 内容 |
|------|------|
| 字数 | 1000字 |
| 核心事件 | 女主被迫签字，被赶出顾家 |
| 情绪 | 绝望/憋屈/坚强 |
| 必须包含 | 女主发现"真相"另有隐情，但选择沉默 |

**第2章详细设计：**

【承接】
林念念看着顾霆深冷漠的眼神，苦涩地笑了。
她没有解释，只是在离婚协议上签了字。

【转折】
"我只要一样东西。"她开口。
顾霆深冷笑："说。"
"我的母亲留给我的遗物——那条项链。"

【暗示伏笔】
顾霆深不知道的是，那条项链里藏着一个芯片，
里面记录着当晚监控录像的真正来源——
推顾清婉的人，根本不是林念念。

【结尾钩子】
林念念被赶出顾家大门，外面下着大雨。
她站在雨中，看着手机上的消息——
"林小姐，您的母亲生前留给您的遗嘱，需要亲自开启。"

---

**第3章：最低谷**

| 项目 | 内容 |
|------|------|
| 字数 | 1000字 |
| 核心事件 | 女主流落街头，发现怀孕 |
| 情绪 | 绝望/无助/惊恐 |
| 必须包含 | 女主发现自己怀孕了，但男主不知道 |

**第3章详细设计：**

【场景】
林念念站在雨中，无处可去。
手机里是医院的短信："林女士，您确认怀孕6周……"

【闪回暗示】
她想起一个月前的那一夜——顾霆深喝醉了，把她当成了别人。
那是她这辈子唯一一次和他亲密。

【选择】
她没有告诉他。
她知道，这个孩子只会成为她和他之间的筹码。
她不想。

【结尾钩子】
一个神秘男人出现在她身后，撑起一把伞。
"林小姐，我受您母亲之托，在您需要的时候出现。"

### 核心公式总结

**【人物公式】**

| 角色 | 设定 | 作用 |
|------|------|------|
| 女主 | 普通出身+隐藏能力+坚强隐忍 | 读者代入感 |
| 男主 | 霸道总裁+外冷内热+后期追妻 | 甜宠来源 |
| 女配（反派） | 恶毒白莲花+假装柔弱 | 制造冲突 |
| 男配 | 反派弟弟/暗恋女主 | 增加波折 |

**【情节公式】**

第1阶段（1-5章）：开场即虐
- 婚礼被抛弃 → 净身出户 → 雨夜流落街头 → 发现怀孕 → 追诉上门

第2阶段（6-15章）：虐心积累
- 真相线索浮现 → 女主变坚强 → 男主动摇 → 英雄救美 → 小波折

第3阶段（16-25章）：真相+逆袭
- 男主发现真相 → 反派认罪 → 女主反杀 → 追妻火葬场

第4阶段（26-30章）：甜宠HE
- 宠妻日常 → 见家长 → 恶人有恶报 → 生子 → 补办婚礼

**【爽点公式】**

| 阶段 | 爽点类型 | 频率 |
|------|---------|------|
| 第1阶段 | 小爽（女主忍耐） | 每章1-2个 |
| 第2阶段 | 中爽（女主反击） | 每2章1个大爽点 |
| 第3阶段 | 大爽（男主追妻+反派打脸） | 每章都有 |
| 第4阶段 | 甜爽（宠妻） | 持续甜蜜 |

| 主线 | 憋屈开场 → 金手指觉醒 → 打脸逆袭 → 权贵追宠 |

### 用户画像

| 维度 | 描述 |
|------|------|
| 目标读者 | 18-40岁女性 |
| 核心诉求 | 代入逆袭快感、看反派被打脸、身份反转爽 |
| 读点 | 女主从最弱变最强，从被欺负到万人捧 |
| 雷区 | 女主太弱一直被欺负、金手指开太大失去悬念 |

### 章节结构（35章）

#### 【第一阶段：极度憋屈开场】（第1-7章）

目标：让读者憋屈到想弃书，但又放不下（因为知道要逆袭）

**第1章：原身惨死**

| 项目 | 内容 |
|------|------|
| 字数 | 1000字 |
| 核心事件 | 原身被毒死/被推下悬崖/被陷害至死 |
| 情绪 | 愤怒/憋屈 |
| 必须包含 | 原身死前发誓："若有来世，我一定让他们血债血偿" |

**第1章详细设计：**

【开场画面】
相府后院，庶女沈云瑶跪在雪地里，身上只穿着单薄的衣裳。

【迫害】
嫡姐沈如月端着毒酒走过来："妹妹，这是父亲赐给你的鸩酒。"
"你娘是贱婢，你也是贱种。去死吧。"

【惨死】
沈云瑶被强行灌下毒酒，口吐鲜血倒地。
临死前，她看到的是沈如月冷酷的笑容，和远处站着的——她的未婚夫，三皇子。

【结尾钩子】
她的意识渐渐消散，最后的念头是：
"若有来世……我沈云瑶……必定……血债血偿……"

---

**第2章：重生归来**

| 项目 | 内容 |
|------|------|
| 字数 | 1000字 |
| 核心事件 | 女主发现自己重生到三年前 |
| 情绪 | 震惊/狂喜/复仇决心 |
| 必须包含 | 发现有金手指（空间/系统/医术等） |

**第2章详细设计：**

【重生】
沈云瑶猛然睁开眼，发现自己躺在柔软的床榻上。
窗外阳光明媚，丫鬟碧玉正在给她梳头。

【记忆涌来】
她想起来了——这是永安三年，她十五岁那年。
距离她的娘亲被陷害、距离她被毒死、还有三年。

【金手指出现】
突然，她脑海中响起一个声音：
"恭喜宿主绑定神医空间，请问是否激活？"
沈云瑶瞳孔地震——前世她被毒死时，正是因为不懂医术，无法自救。

【决心】
"激活！"她在心中喊道。
从这一刻起，沈云瑶不再是那个任人欺负的废物庶女。
她是——神医传承者。

【结尾钩子】
丫鬟碧玉小声说："小姐，二皇子刚才派人送来请帖，邀请您去参加三日后的赏花宴……"
沈云瑶眼眸微冷——前世，她就是在赏花宴上被沈如月当众羞辱，彻底沦为京城笑柄。
这一次，她绝不会让历史重演。

---

**第3-7章：憋屈积累**

| 第7章 | 赏花宴前夕，被沈如月设计陷害 | ⭐⭐⭐⭐⭐ | 超级憋屈，等待逆袭 |

#### 【第二阶段：金手指觉醒小试牛刀】（第8-18章）

目标：女主开始反击，每2-3章打脸一次，节奏加快

**第8-10章：赏花宴反击（第一个大高潮）**

| 章节 | 核心事件 | 爽点指数 | 说明 |
|------|---------|----------|------|
| 第8章 | 赏花宴当天，沈如月设计让她出丑 | 小虐 | 阴谋开始 |
| 第9章 | 女主用医术自救，化解危机 | 中爽 | 金手指首次使用 |
| 第10章 | 女主反杀，当众揭穿沈如月的阴谋 | 大爽⭐⭐⭐ | 第一次大捷 |

**第10章详细设计（重点章）：**

【场景】
赏花宴上，众人围坐。沈如月"不小心"把茶泼在沈云瑶身上，
然后假装惊慌："哎呀，妹妹对不起，我不是故意的！"

【原轨迹】
前世，沈云瑶只能忍气吞声，穿着湿衣服直到宴会结束，
成为全京城的笑柄。

【新轨迹】
沈云瑶微微一笑："姐姐不必自责。不过我这身衣服是皇后娘娘赏赐的，
弄脏了可不好交代。不如让我展示一下从娘亲那里学的医术，给大家助助兴？"

【打脸开始】
沈云瑶当场展示了一手精湛的针灸术，震惊四座。
连太后都忍不住赞叹："这丫头的手法，比太医院的御医还精湛！"

【反转】
沈如月脸色惨白。她没想到这个废物庶女，居然还有这一手。
而三皇子看向沈云瑶的眼神，第一次有了变化。
