# 一、核心方法论：种子创作法

## 1.1 什么是种子创作法

```
【输入】用户给一个"种子"
  ├── 核心情节（被背叛/被欺负/被冤枉等）
  ├── 人物关系（夫妻/闺蜜/婆媳/师徒等）
  └── 创意点（独特设定/社会矛盾/人物反差等）

【输出】AI发散变体
  └── 同一情节 → 不同年代/职业/身份/金手指的多个版本

【本质】固定故事结构 + 变量自由组合 = 高效批量创作
```

## 1.2 为什么用种子创作法

| 对比 | 固定模板 | 种子创作法 |
|------|---------|------------|
| 用户输入 | 少，但固定 | 中等，但灵活 |
| 创意独特性 | 中等（套公式） | 高（每次发散不同） |
| 批量生产 | 一般 | 强（一个种子出多个变体） |
| 爆款率 | 依赖模板通用性 | 依赖种子质量+发散创意 |
| 适配平台 | 一般 | 强（可针对不同平台调优） |

## 1.3 核心公式（固定结构）

所有短故事都遵循7阶段结构，每个阶段的反转节点和伏笔要求必须标注：

```
阶段1【开场冲突】(1-3章)
  └── 被背叛/被欺负/被冤枉 → 情绪钩子
  └── 【反转节点】此阶段至少1个反转（如：看似坏人其实是好人/看似帮助实际是陷害）
  └── 【伏笔埋入】至少1个伏笔（如：配角的奇怪行为/物品/话语）

阶段2【重生/穿越】(4-5章)
  └── 获得第二次机会 + 发现金手指（可选）
  └── 【反转节点】可选（如：重生后发现事情并非表面那样）
  └── 【伏笔回收/新埋】回收阶段1的伏笔，并埋入新伏笔

阶段3【小试牛刀】(6-10章)
  └── 第一次反击成功 → 建立读者信心
  └── 【反转节点】此阶段至少1个反转（如：以为胜券在握实际落入圈套/看似绝境实际转机）
  └── 【伏笔埋入】至少1个新伏笔（为后续反转铺垫）

阶段4【中段积累】(11-20章)
  └── 复仇准备 + 发现更深阴谋 + 势力积累
  └── 【反转节点】此阶段至少1个反转（如：盟友变敌人/敌人变盟友）
  └── 【伏笔埋入】至少1个新伏笔（核心伏笔在此阶段深化）

阶段5【大高潮前奏】(21-25章)
  └── 最终对决前夕 + 感情线确定
  └── 【反转节点】此阶段至少1个反转（如：反派露出真面目/意外强援出现）
  └── 【伏笔回收】回收至少2个伏笔（为高潮蓄力）

阶段6【大高潮】(26-30章)
  └── 终极打脸 + 反派认罪 + 复仇完成
  └── 【反转节点】此阶段至少1个大反转（如：主角反杀/反派阴谋曝光/隐藏身份揭示）
  └── 【制胜逻辑】主角胜利必须基于智慧或能力，禁止"反派降智送人头"
  └── 【伏笔回收】全部伏笔在此阶段回收完毕

阶段7【HE结局】(31-35章)
  └── 甜蜜收尾 + 身份揭示 + 完美落幕
  └── 【最终反转】可选（如：彩蛋式反转为续作埋伏笔）
```

**伏笔规则：** 全部伏笔必须在阶段6结束前回收完毕，不得有悬空伏笔。

### 2.4.2 主配矛盾变量（单纯矛盾，不涉及社会问题）

**定义：** 主角与配角之间的单纯利益/性格冲突，不依赖家庭背景/阶级差异/社会矛盾。

| 矛盾类型 | 配角身份 | 冲突来源 | 典型场景 |
|---------|---------|---------|----------|
| **同事抢功** | 同事/下属 | 利益冲突 | 主角方案被同事抢了邀功/主角成果被上司冒领 |
| **同事背锅** | 同事/下属 | 甩锅推责 | 主角被同事陷害背黑锅/主角发现同事做假账 |
| **晋升竞争** | 同事/竞争者 | 零和博弈 | 竞争同一个升职名额/竞争同一个项目 |
| **客户争夺** | 同行/销售 | 利益冲突 | 主角客户被同行撬走/主角被同行恶意低价抢单 |

# 二B、反派设计模块（必须遵循）

## 二B.1 反派设计三要素

每个反派必须同时具备以下三要素，缺一不可：

| 要素 | 定义 | 示例 |
|------|------|------|
| **动机** | 反派做坏事的原因，必须合理可理解 | "我也曾被伤害过" / "为了保护家人" / "出身低微受尽欺压" |
| **目标** | 反派想要达成的目的，必须清晰 | "取代主角地位" / "夺取家族权力" / "复仇" |
| **策略** | 反派实现目标的手段，必须有脑子 | "借刀杀人" / "挑拨离间" / "设局陷害" |

**反面案例（禁止）：**
```
❌ 反派就是坏，看主角不顺眼就欺负他（无脑坏）
❌ 反派智商下线，主角轻松获胜（降智反派）
❌ 反派实力忽强忽弱，看情况需要（战力崩塌）
```

**正确案例：**
```
✅ 反派A：动机（曾被主角家族迫害）→ 目标（毁掉主角家族）→ 策略（先拉拢再背叛+利用主角弱点设局）
✅ 反派B：动机（出身底层想翻身）→ 目标（夺取公司控制权）→ 策略（潜伏多年收集把柄+安插内鬼+关键时刻一击致命）
```

## 二B.2 反派等级设计

| 反派等级 | 出现阶段 | 策略复杂度 | 主角应对难度 |
|---------|---------|-----------|-------------|
| **小反派** | 阶段1-3 | 单线策略，单个手段 | 低，主角凭金手指可应对 |
| **中反派** | 阶段3-5 | 多线策略，会利用盟友 | 中，主角需要智取+成长 |
| **大反派** | 阶段5-6 | 复合策略，多层布局 | 高，主角必须用智慧+能力+伏笔联动才能战胜 |

## 二B.3 主角制胜逻辑约束

**核心原则：主角胜利必须基于智慧或能力，禁止"反派降智送人头"。**

| 制胜类型 | 说明 | 示例 |
|---------|------|------|
| **智慧型** | 主角利用信息差/布局/识破反派计谋获胜 | 主角早知反派会背叛，提前收集证据反杀 |
| **能力型** | 主角金手指/专业能力在关键时刻发挥作用 | 神医空间关键时刻救活关键人物 |
| **组合型** | 智慧+能力双重胜利 | 主角识破陷阱+用能力反制（推荐） |

**禁止出现：**
```
❌ 反派突然变蠢，自己犯低级错误
❌ 反派临阵倒戈，毫无铺垫
❌ 主角靠"运气好"获胜
❌ 反派强行降智，为了让主角赢而输
```

## 二B.4 场景功能角色（非强行添加）

**路人/工具人原则：按需出现，服务剧情，不得强行添加。**

| 角色类型 | 出现场景 | 功能 | 示例 |
|---------|---------|------|------|
| **场景过渡** | 主角到新地点 | 交代环境/引出剧情 | 茶馆偶遇/街头目睹 |
| **信息传递** | 需要传递情报 | 透露关键信息 | 丫鬟传话/朋友告知 |
| **冲突推动** | 需要制造矛盾 | 推动情节发展 | 绿茶闺蜜煽风点火 |
| **打脸工具** | 需要爽点场景 | 提供打脸机会 | 势利眼亲戚/拜高踩低同事 |
| **背景板** | 群戏场景 | 营造氛围 | 婚礼宾客/宴会人士 |

**检查标准：** 每增加一个路人角色，必须能回答"这个角色删掉会影响剧情吗？"如果不影响，不加。

# 三、发散变体方法论

## 3.1 变体维度矩阵

```
种子情节 + 年代 + 身份 + 职业 + 关系 + 金手指 + 社会矛盾
     ×         ×        ×       ×        ×          ×
   固定      可变    可变     可变     可变        可变
```

## 3.2 发散步骤

```
Step 1: 确定种子（用户输入）
  └── 核心情节 + 人物 + 关系 + 创意点

Step 2: 选择年代
  └── 现代 / 80年代 / 历史 / 修仙 / 末世

Step 3: 确定身份+职业
  └── 身份决定爽点来源，职业决定金手指方向

Step 4: 选择关系组合
  └── 夫妻/闺蜜/婆媳等，决定核心冲突

Step 5: 确定金手指
  └── 能力型/身份型/辅助型

Step 6: 加入社会矛盾
  └── 增加真实感和讨论度

Step 7: 代入7阶段结构
  └── 输出完整大纲
```

# 五、创作检查清单

## 5.1 开场检查（决定留存率）

- [ ] 前3章有强冲突吗？（离婚/背叛/阴谋）
- [ ] 读者知道主角的"憋屈"吗？
- [ ] 有"为什么不解释"的悬念吗？
- [ ] 第一章出现的主要人物有几个？（≤3个最好）
- [ ] 阶段1是否有至少1个反转？
- [ ] 阶段1是否埋入至少1个伏笔？

## 5.2 反派设计检查（必须通过）

- [ ] 每个反派是否有明确动机？
- [ ] 每个反派是否有清晰目标？
- [ ] 每个反派是否有具体策略？（不是无脑坏）
- [ ] 反派等级是否与主角成长匹配？
- [ ] 主角胜利是否有智慧或能力支撑？（禁止降智获胜）

## 5.3 节奏检查（决定完读率）

- [ ] 每3章有一个小爽点吗？
- [ ] 每10章有一个大高潮吗？
- [ ] 主角在进步/变强吗？
- [ ] 反派有给主角制造麻烦吗？
- [ ] 每个阶段是否有标注的反转节点？

## 5.4 伏笔检查（全部回收）

- [ ] 阶段1埋入的伏笔是否在阶段2有呼应？
- [ ] 阶段3-4埋入的伏笔是否在阶段5-6有回收？
- [ ] 全部伏笔是否在阶段6结束前回收完毕？
- [ ] 是否有悬空伏笔？（禁止）

## 5.5 场景角色检查（非强行添加）

- [ ] 每个路人角色删除后是否影响剧情？
- [ ] 路人角色是否服务于明确的功能？（信息传递/冲突推动/打脸工具）
- [ ] 是否没有为加而加的冗余角色？

## 5.6 爽点密度检查

| 阶段 | 爽点数量 | 类型 |
|------|---------|------|
| 第1-5章 | 1-2个 | 小爽（主角忍耐/伏笔） |
| 第6-15章 | 3-5个 | 中爽（主角反击） |
| 第16-25章 | 5-8个 | 大爽（打脸开始） |
| 第26-35章 | 10+个 | 连续爽（复仇完成） |

## 5.7 结局检查

- [ ] 结局是HE吗？（番茄读者不喜欢Be）
- [ ] 恶人有恶报吗？
- [ ] 主角得到幸福了吗？
- [ ] 有"意犹未尽"的感觉吗？

# 六、快速创作流程

## 6.1 接到种子后的操作步骤

```
Step 1: 分析种子
  └── 提取：核心冲突 + 人物关系 + 创意点

Step 2: 选择变体方向
  └── 用户指定 OR AI推荐 OR 随机抽取

Step 3: 填写变量矩阵
  └── 年代/身份/职业/关系/金手指/社会矛盾

Step 4: 生成7阶段大纲
  └── 按固定结构填充每阶段核心事件
  └── 标注每阶段反转节点（每阶段至少1个）
  └── 设计反派三要素（动机+目标+策略）
  └── 规划伏笔埋入和回收位置

Step 5: 产出前3章试写
  └── 验证节奏和爽点密度
  └── 检查反派设计是否符合三要素
  └── 检查伏笔是否已埋入

Step 6: 用户反馈后继续全稿
  └── 阶段6必须通过"主角制胜逻辑"检查
```

# 六、创意因子模块：破除模板同质化的核心武器

> **背景问题：** 同一种爆款题材模型（如"追妻火葬场"），被大量作者重复使用，
> 会导致"千篇一律"——相似的开头、相似的反转、相似的结局。
> 读者审美疲劳，爆款率下降。
>
> **解决方案：** 在模板骨架之上，叠加"创意因子"——
> 让同一模板生长出完全不同的故事血肉。
>
> **核心原则：** 模板是骨架，创意因子是血肉。骨架相同，血肉不同。

---

## 6.1 创意因子层级矩阵

| 层级 | 作用 | 回答的问题 | 重要程度 |
|------|------|-----------|----------|
| **USP** | 灵魂 | "这个故事独特在哪？" | ⭐⭐⭐⭐⭐ |
| **人设反差** | 血肉 | "角色有什么不一样？" | ⭐⭐⭐⭐ |
| **世界观细化** | 场景 | "故事发生在怎样的世界里？" | ⭐⭐⭐⭐ |
| **情感锚点** | 升华 | "只有这个故事能写的情感是什么？" | ⭐⭐⭐⭐⭐ |

---

## 6.2 USP（独特卖点）—— 最重要

### 6.2.1 什么是USP

USP = Unique Selling Proposition（独特卖点）

**定义：** 读者翻开这个故事，第一眼就能感受到的"不同"。

### 6.2.2 USP设计方向

| 方向 | 说明 | 示例 |
|------|------|------|
| **身份特殊化** | 主角有独特的身份标签 | 聋哑人/自闭症/超忆症 |
| **能力特殊化** | 金手指有独特的能力限制 | 能力有冷却时间/副作用 |
| **职业特殊化** | 用非传统职业做主角 | 仵作/法医/盗墓/捞尸 |
| **世界观特殊化** | 设定不是常规设定 | 女尊世界/赛博朋克/克苏鲁 |
| **关系特殊化** | 人物关系非传统 | 两个穿越者互相博弈 |

### 6.2.3 USP × 题材 案例库

#### "追妻火葬场" + 不同USP

| 版本 | 种子 | USP | 独特性 |
|------|------|-----|--------|
| 常规版 | 被抛弃+怀孕+渣男后悔 | 无 | 套公式 |
| USP-A | 被抛弃+怀孕+渣男后悔 | 女主是聋哑人 | 情感表达受限，更虐心 |
| USP-B | 被抛弃+怀孕+渣男后悔 | 女主有超忆症（无法忘记痛苦） | 记忆成为双刃剑 |
| USP-C | 被抛弃+怀孕+渣男后悔 | 女主是战地记者 | 职业属性带来新矛盾 |
| USP-D | 被抛弃+怀孕+渣男后悔 | 女主能听到别人的谎言 | 真相揭露方式独特 |

#### "穿越逆袭" + 不同USP

| 版本 | 种子 | USP | 独特性 |
|------|------|-----|--------|
| 常规版 | 庶女被欺→逆袭 | 无 | 套公式 |
| USP-A | 庶女被欺→逆袭 | 嫡姐也是穿越者 | 对手有同等金手指 |
| USP-B | 庶女被欺→逆袭 | 女主有读心术 | 敌我识别成为新叙事线 |
| USP-C | 庶女被欺→逆袭 | 架空的"女尊世界" | 世界观反转（女强男弱） |
| USP-D | 庶女被欺→逆袭 | 女主是仵作（验尸+推理） | 职业+推理线 |

#### "空间囤货末世" + 不同USP

| 版本 | 种子 | USP | 独特性 |
|------|------|-----|--------|
| 常规版 | 末世→囤货→生存 | 无 | 套公式 |
| USP-A | 末世→囤货→生存 | 废土赛博朋克背景 | 科技+废土融合 |
| USP-B | 末世→囤货→生存 | 空间是"数字备份"有冷却时间 | 能力有代价 |
| USP-C | 末世→囤货→生存 | 有人能盗取她的空间 | 增加"数据黑客"对抗线 |
| USP-D | 末世→囤货→生存 | 末世是"楚门世界"（模拟） | 真相更震撼 |

## 6.3 人设反差——在公式里加变量

### 6.3.1 角色性格反差

| 公式项 | 常规设定 | 反差设定 |
|--------|---------|---------|
| 女主性格 | 温柔善良 | 表面温柔实则腹黑 |

| 反派动机 | 单纯嫉妒 | 为复仇接近女主（黑白不分） |

### 6.3.2 角色身份反差

| 公式项 | 常规设定 | 反差设定 |
|--------|---------|---------|
| 女主身份 | 普通出身 | 隐藏身份（黑客/神医/千金） |
| 男主身份 | 霸道总裁 | 身份比女主更惨 |
| 反派身份 | 富家子弟 | 看似穷人实则幕后黑手 |
| 助攻身份 | 忠仆 | 反派卧底（反转） |

### 6.3.3 人设反差案例

**"穿越逆袭"模板 + 庶女身份**

```
常规版：
庶女被嫡女欺负 → 逆袭打脸 → 嫁给王爷

反差版本A（性格反差）：
庶女人设：表面唯唯诺诺，实际是"白切黑"
→ 她故意让嫡姐放松警惕，收集证据
→ 嫡姐以为她还是那个软柿子
→ 结果被一击致命
→ 独特性：不是"变强"，是"藏拙"

反差版本B（关系反差）：
嫡女也是穿越者，重生回来
→ 两个人都有前世记忆
→ 互相知道对方底牌
→ 变成"两个穿越者互相博弈"
→ 独特性：对手不是纸老虎

反差版本C（身份反差）：
庶女其实是皇帝亲生女儿（被偷换）
→ 嫡姐争了一辈子的位置，本来就是她的
→ 她从"庶女"变"真公主"
→ 嫡姐直接心态崩溃
→ 独特性：身份反转更彻底
```

## 6.4 世界观细化——让故事"长"在特定世界

### 6.4.1 世界观细化维度

| 维度 | 常规设定 | 细化设定 |
|------|---------|---------|
| 时代背景 | 古代架空 | 具体朝代+真实历史事件 |
| 社会制度 | 嫡庶分明 | 加上"商贾vs士族"阶层矛盾 |
| 职业体系 | 医生 | 仵作（验尸）+ 推理线 |
| 修炼体系 | 修仙 | "星际机甲"科幻背景 |
| 地理设定 | 王朝 | "七国争霸"+每个国家不同设定 |

### 6.4.2 世界观细化案例

**"空间囤货末世"模板**

```
常规版：
末世→囤货→生存→打丧尸→HE

细化版本A（赛博朋克末世）：
→ 背景是"废土赛博朋克"
→ 不是普通丧尸，是"失控AI"
→ 主角能力：入侵AI系统
→ 独特性：科幻+生存融合

细化版本B（克苏鲁末世）：
→ 末世不是天灾，是"古神苏醒"
→ 人类理智值成为资源
→ 主角能力：理智值高于常人
→ 独特性：恐怖氛围+生存

细化版本C（末日重建+商战）：
→ 末世后不是打丧尸，是"资源战争"
→ 主角用商业手段而非武力
→ 建立商会/物流网/情报网
→ 独特性：智斗>打斗
```

## 6.5 情感锚点——只有这个故事能写的情感

### 6.5.1 什么是情感锚点

**情感锚点 =** 故事结尾最打动人的那一瞬间，是这个故事独有的。

### 6.5.2 情感锚点设计公式

```
不是"她原谅他了"，而是"他终于学会用她的方式爱她"

不是"她赢了"，而是"她选择放下"

不是"坏人被惩罚"，而是"她亲手原谅了伤害她的人"
```

### 6.5.3 情感锚点案例库

| 题材 | 常规结局 | 情感锚点 | 独特性 |
|------|---------|---------|--------|
| 追妻火葬场 | 女主原谅，男主追回 | "他学会用她的手语爱她" | 不是原谅，是理解 |
| 穿越逆袭 | 女主复仇成功，当皇后 | "她发现仇人女儿是无辜的，选择放过" | 不是复仇，是放下 |
| 空间囤货末世 | 主角成为首领，拯救人类 | "她拒绝成为领袖，选择做普通人" | 不是权力，是选择 |
| 悬疑反转 | 真相大白，恶人被罚 | "她选择和反派同归于尽式的活着" | 不是胜利，是解脱 |
| 神医总裁 | 女主成为神医，和男主在一起 | "她用医术救活了反派的孩子" | 不是报应，是救赎 |

### 6.5.4 情感锚点设计检查表

```
□ 故事结局的情感，是这个故事独有的吗？
□ 如果换一个人设/背景，这个情感还能成立吗？
□ 读者读到结局，会"哇"的一声吗？
□ 这个情感能让我自己想哭吗？

（如果任何一个问题答"否"，重新设计情感锚点）
```

## 6.6 创意因子组合公式

### 6.6.1 最低组合要求

**每个故事至少包含：**

```
1个USP + 1个人设反差 + 1个世界观细化 = 3个创意因子
```

### 6.6.2 进阶组合

**更高组合（爆款级）：**

```
1个USP + 1个人设反差 + 1个世界观细化 + 1个情感锚点 = 4个创意因子
```

### 6.6.3 创意因子组合检查表

```
□ 故事有USP吗？（如果说不清楚"这个故事独特在哪"，USP缺失）
□ 有人设反差吗？（如果角色和其他故事的角色"差不多"，反差缺失）
□ 世界观有细化吗？（如果世界是"通用古代/通用末世"，细化缺失）
□ 有情感锚点吗？（如果结局是"打脸→HE"，锚点缺失）

创意因子数量 ≥ 3 → 故事有独创性
创意因子数量 < 3 → 故事可能千篇一律
```

## 6.8 创意因子实战流程

### Step 1：确定种子

```
种子 = 核心冲突 + 人物关系 + 基础设定
```

### Step 2：匹配爆款题材模型

```
种子 → 对应爆款题材模型（如"追妻火葬场"）
```

### Step 3：植入USP

```
问："这个故事最独特的一点是什么？"
答："_____（USP）"

如果答不上来 → 重新思考USP
```

### Step 4：叠加人设反差

```
问："角色和其他故事的角色有什么不同？"
答："女主是_____（反差设定）"

如果答"和其他故事差不多" → 加强反差
```

### Step 5：细化世界观

```
问："故事发生在怎样的世界里？"
答："是_____（细化世界观），不是"通用设定""

如果答"普通古代/普通末世" → 重新细化
```

### Step 6：设计情感锚点

```
问："故事结尾，最打动人的一瞬间是什么？"
答："是_____（情感锚点），不是简单的"打脸→HE""

如果答不上来 → 重新设计结局
```

### Step 7：检查清单

```
□ USP：_____
□ 人设反差：_____
□ 世界观细化：_____
□ 情感锚点：_____
□ 创意因子总数：___（≥3个才合格）
```

## 6.9 创意因子案例：完整版

### 案例："追妻火葬场"模板 × 创意因子

**Step 1：种子**

```
核心冲突：女主被渣男抛弃，渣男后悔
人物关系：夫妻→离婚
基础设定：现代都市
```

**Step 2：匹配模型**

```
→ 追妻火葬场（模型A）
```

**Step 3：植入USP**

```
USP：女主是聋哑人（不能说话，只能用手语表达情感）

差异化：这个故事里，"沟通"本身就是最大的障碍
男主追妻时，无法用言语解释，只能用行动证明
```

**Step 4：叠加人设反差**

```
女主：表面温柔，实则是"无声的复仇者"
→ 她不是"不会说话"，是"选择不说话"
→ 她用手语写了一封信给男主，里面是所有的误会真相
→ 但她没有递出去——她选择让他自己发现

人设反差：女主的"哑"不是弱点，是武器
```

**Step 5：细化世界观**

```
世界观：聋哑人社区（女主是手语翻译师）
→ 她在一个聋哑人学校工作
→ 她的同事、学生都是聋哑人
→ 她的"无声世界"是真实的、有温度的
→ 而不是"因为不能说话所以很惨"

细化：这不是"残疾人的苦难"，是"另一个世界的美好"
```

**Step 6：设计情感锚点**

```
情感锚点：
男主最终学会了手语
在故事的最后一幕，他用手语说：
"我花了一年学会手语，只为了告诉你——
我听懂了你三年前没说出口的那句话。"

但女主已经不需要这句话了
因为她早就释然
她用手语回答：
"我等的是你愿意学，不是你学会。"

→ 不是"她原谅他了"
→ 是"她见证了他的改变，比原谅更重要的是成长"
```

**Step 7：检查清单**

```
□ USP：女主是聋哑人 ✓
□ 人设反差：女主的"哑"是武器，不是弱点 ✓
□ 世界观细化：聋哑人社区，有温度的"无声世界" ✓
□ 情感锚点：他学会手语，但她已释然 ✓
□ 创意因子总数：4个 ✓（≥3个）
```
