## 四、虫族生物科技

### 4.1 虫族基础设定

| 属性 | 描述 |
|------|------|
| 种群数量 | 宇宙中有很多虫族种群 |
| 环境适应 | 根据生存环境自适应进化出不同的种群形态 |
| 控制机制 | 母虫主导（量子信息+信息素双重控制） |
| 智慧等级 | 非智慧种族 |
| 行为特征 | 吞噬资源星，与智慧种族对抗 |
| 资源价值 | 虫族身体是优质材料资源 |

### 4.2 母虫控制机制

#### 4.2.1 量子信息控制

| 属性 | 描述 |
|------|------|
| 控制方式 | 量子纠缠通讯 |
| 控制范围 | 超越光年，理论上无限 |
| 特点 | 即时协调，超越距离限制 |
| 弱点 | 需要稳定的量子场环境 |

**技术原理**：
- 母虫与所有虫族个体存在量子纠缠连接
- 母虫的意念直接通过量子场传递
- 即使相隔数光年，指令也能即时传达

#### 4.2.2 信息素控制

| 属性 | 描述 |
|------|------|
| 控制方式 | 化学信息素扩散 |
| 控制范围 | 短距离（数公里到数光年不等） |
| 特点 | 本能行为指令，反应快速 |
| 弱点 | 受环境影响，浓度随距离稀释 |

**技术原理**：
- 信息素分子携带特定指令信息
- 虫族嗅觉器官接收并执行
- 可触发战斗、觅食、繁殖等本能行为
