## 七、流派专属设定

### 凡人流

**核心公式：** 谨慎者生存，而非天才者胜利

**标配元素：**
- 小绿瓶/葫芦类逆天宝物
- 越级战斗靠智谋
- 资源稀缺必须精打细算

### 退婚流

**核心公式：** 三年之辱，莫欺少年穷

**标配元素：**
- 未婚妻退婚
- 主角废柴逆袭
- 打脸环节

### 系统流

**核心公式：** 任务+奖励+代价 = 成长闭环

**标配元素：**
- 任务系统
- 技能/属性面板
- 商城兑换

### 家族修仙流

**核心公式：** 家族百年的求生与牺牲

**标配元素：**
- 群像视角
- 家族兴衰主线
- 牺牲与传承

### 苟道流

**核心公式：** 稳字当头，但在能力范围内承担责任

**标配元素：**
- 纸人分身
- 万全准备
- 稳健翻盘

### 序列流

**核心公式：** 扮演好自己的序列角色，才能晋升更高序列

**标配元素：**
- 序列晋升体系
- 扮演法限制
- 22条途径设定

## 八、开局模板

### 8.1 凡人流开局

```
开头要素：
1. 主角处境艰难（资质低/资源匮乏）
2. 获得逆天机缘（小绿瓶/系统）
3. 谨慎选择，不冒进
4. 建立第一个小目标（筑基/进入某秘境）
```

### 8.2 退婚流开局

```
开头要素：
1. 退婚现场（三年之约）
2. 主角被羞辱
3. 获得逆天传承/系统
4. 留下悬念（三年后打脸）
```
