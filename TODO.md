## 待办文档维护规则（2026-04-18 新增）

### 状态变更规则

| 操作 | 规则 |
|------|------|
| **完成** | 更新状态为 ✅，记录完成时间，不删除 |
| **状态变更** | 在原条目下追加变更记录，保留历史 |
| **新增** | 在对应分类下追加新条目，保留历史记录 |
| **删除** | 禁止删除，只有完成或不处理两种归宿 |

### 模板

```markdown
### N. [任务名]

| 项目 | 内容 |
|------|------|
| 提出时间 | YYYY-MM-DD |
| 当前状态 | 🔴/🟡/✅ |
| 变更记录 | |
| - | YYYY-MM-DD HH:MM → 状态更新说明 |
```

### 示例

```markdown
### 3. 示例任务

| 项目 | 内容 |
|------|------|
| 提出时间 | 2026-04-18 |
| 当前状态 | ✅ 已完成 |
| 变更记录 | |
| - | 2026-04-18 17:50 → 创建任务 |
| - | 2026-04-19 09:00 → ✅ 已完成 |
```
