# 微信公众号爆款文章爬取改写流水线

> 最后更新：2026-05-28 00:26 by 阿呆

## 技能清单

| 技能 | 用途 | 场景 |
|------|------|------|
| `wechat-article-search` | 关键词搜索 + 真实URL + 摘要 + 发布时间 | 场景一 |
| `wechat-search-redfox` | 10w+爆款榜单 + 精确阅读数 + 点赞 + 评论 + 评分 | 场景二 |
| `wechat-article-spider` | 抓取正文Markdown + 图片下载 | 两场景共用 |
| `gzh-10w-hot` | 分类10w+榜单（阅读数仅"10w+"标签）| 场景二辅助 |

## 核心流程

```
场景一：按关键词搜索 → 改写
──────────────────────────────────────
① wechat-article-search -r "关键词" -n 数量
   → 真实URL + 标题 + 摘要 + 作者 + 发布时间

② wechat-article-spider <URL> <输出目录>
   → 正文Markdown + 图片

③ 阿呆AI分析改写


场景二：10w+爆款榜单 → 改写
──────────────────────────────────────
① wechat-search-redfox --keyword "垂类" --start-date <日期>
   → 真实URL + 精确阅读数 + 点赞 + 评论 + 三维评分 + 摘要

   或 gzh-10w-hot --type "分类" --start_date <日期> --end_date <日期>
   → 真实URL + 阅读数标签(10w+) + 摘要（无精确数字）

② wechat-article-spider <URL> <输出目录>
   → 正文Markdown + 图片

③ 阿呆AI分析改写
```
